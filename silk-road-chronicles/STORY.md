## 1. 世界观概述

**时代背景**：西汉末年至东汉初年，丝绸之路刚刚打通，西域36国林立，汉匈争霸的余波未平。玩家扮演一个无名小卒，从一穷二白开始，通过政、商、军三条路线逐步崛起，最终统一西域诸国。

**核心循环**：
```
农业→粮食→人口→劳动力→资源采集→工业→兵器→军事→征服→领土→农业
  ↕                                                                    ↕
商业←←←←←←←←←←←←←←←← 金币 ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
  ↕
外交→联盟→贸易协定→商业加成→金币→城市建设→人口→军事
```

## 2. 资源-人口-发展依赖体系

### 2.1 六大基础资源

| 资源 | 来源 | 用途 | 关联建筑 |
|------|------|------|----------|
| **粮食(food)** | 农田、渔猎、部落贡赋 | 养活人口、喂养军队、奴隶消耗 | 农田→灌溉渠→精耕细作 |
| **金币(gold)** | 税收、贸易、商路、朝贡 | 建设建筑、招募武将、外交赠礼 | 集市→商队驿站→丝路商站 |
| **铁矿(iron)** | 矿场采集、贸易 | 锻造兵器、建设军事建筑 | 铁匠铺→兵器坊 |
| **马匹(horse)** | 马场、贸易、部落贡赋 | 训练骑兵、外交赠礼 | 马场→骆驼厩 |
| **玉石(jade)** | 矿场、贸易 | 高价值贸易品、外交赠礼 | 玉石作坊 |
| **丝绸(silk)** | 纺织工坊、贸易 | 高价值贸易品、外交赠礼 | 织造坊→丝路商站 |

### 2.2 人口体系

```
人口类型金字塔：
┌─────────────┐
│   武将(稀有)  │  ← 任务/事件/培养获得
├─────────────┤
│  文官(稀缺)   │  ← 任务/事件/培养获得
├─────────────┤
│  公民(核心)   │  ← 人口自然增长/奴隶转化
│  男公民/女公民  │  ← 可分配为：商人/农民/工匠/士兵
├─────────────┤
│  奴隶(底层)   │  ← 奴隶市场购买/战俘/部落贡赋
│  劳奴/匠奴/   │  ← 可分配为：农场/矿场/伐木/娱乐
│  舞女/侍妾/   │  ← 可培养转化为：公民/武将
│  战奴         │  ← 可培养转化为：士兵/武将
└─────────────┘
```

**人口增长公式**：
- 基础增长率 = 城市等级对应比率 (0.5%~2.0%/月)
- 实际增长 = 当前人口 × 增长率 × 季节修正 × 剩余粮食修正
- 上限 = 城市等级对应最大人口 (500~30000)
- 全女性国家特殊：西梁女国(子母河×3)、东女国(母系传承+25%)、阿玛宗(收养制度)

### 2.3 发展依赖链

```
阶段0: 选择城邦
  ↓
阶段1: 城邦初立 (解锁: 城邦管理、资源采集、人口管理)
  │  需要: 完成城邦基础任务 (进贡100金)
  │  获得: 管理权、初始公民10人、初始奴隶10人
  ↓
阶段2: 部落征途 (解锁: 奴隶买卖→农业→商业→税收→马场→锻造)
  │  需要: 完成1-3个部落任务
  │  每个部落任务需要: 金币+粮食+靠近部落
  │  关键路径: 
  │    部落任务1 → 奴隶买卖/资源采集/农业/马场
  │    部落任务2 → 奴隶劳作/商业/税收/锻造
  │    部落任务3 → 奴隶培训/高级能力
  ↓
阶段3: 城市发展 (城市等级2+)
  │  需要: 完成至少3个部落任务
  │  解锁: 外交系统、征兵系统、科技研发
  │  城市Lv2 → 外交+征兵+科技
  │  城市Lv3 → 宣战+商路+谍报
  ↓
阶段4: 扩张征伐
  │  需要: 城市等级3+，完成目标国50%以上部落任务
  │  可执行: 宣战、攻城、吞并、附庸
  ↓
阶段5: 统一西域
     征服或外交统一所有36国
```

## 3. 武将系统

### 3.1 武将属性

| 属性 | 范围 | 影响 |
|------|------|------|
| 统率(leadership) | 30-100 | 带兵数量、军队士气 |
| 武力(force) | 20-100 | 单挑胜负、近战伤害 |
| 智力(strategy) | 20-100 | 计策成功率、防御策略 |
| 政治(politics) | 10-100 | 内政效率、外交效果 |
| 魅力(charisma) | 10-100 | 招募成功率、部下忠诚 |

### 3.2 武将等级与军衔

| 军衔 | 所需等级 | 最大带兵 | 加成 |
|------|----------|----------|------|
| 伍长 | Lv1 | 500 | +0% |
| 什长 | Lv3 | 1,000 | +5% |
| 百夫长 | Lv5 | 2,000 | +10% |
| 千夫长 | Lv8 | 5,000 | +15% |
| 万夫长 | Lv12 | 10,000 | +20% |
| 将军 | Lv15 | 20,000 | +30% |
| 大将军 | Lv20 | 50,000 | +40% |

### 3.3 武将获取途径

#### 途径一：任务奖励（主要来源）
- **城邦任务**：完成城邦管理权任务后，城邦长老赠送1名初始武将
- **部落任务**：完成部落盟约任务，部落首领派遣1名武将投奔
- **剧情事件**：特定剧情触发后，历史人物主动投奔
- **国家征服**：征服一国后，该国武将有30%概率投降归顺

#### 途径二：奴隶培养
- **侍妾→女武将**：消耗500金+100粮，30回合培养期
- **匠奴→男武将**：消耗400金+100粮，25回合培养期
- **战奴→男武将**：消耗350金+80粮，20回合培养期
- **舞女→女武将**：消耗450金+90粮，25回合培养期（特殊路线）

#### 途径三：城邦招聘
- 消耗500金，冷却3回合
- 随机生成一名武将，属性随机
- 城市等级越高，出现高品质武将概率越大

#### 途径四：剧情/事件获得
- **随机事件**："流浪武者"、"落难公主"、"逃亡将军"等事件
- **全女性国家特殊事件**：阿玛宗比武招亲、西梁女国女王赐婚、东女国女王赐将
- **联盟事件**：与盟国关系达到"亲密"后，盟国赠送武将
- **探索事件**：探索遗迹、古墓时发现隐世高手

#### 途径五：战俘招降
- 战斗胜利后，敌方武将有30%概率被俘
- 被俘武将可通过赠礼/说服使其归顺
- 基础忠诚度40，需通过提升好感至60+才能完全收服

## 4. 三大全女性国家

### 4.1 阿玛宗（Amazon Warriors）⚔️

**定位**：军事型全女性国家，尚武好战

**基本信息**：
- 首都位置：(75, 25) 天山深处
- 人口：8,000（全女性）
- 地形：山地
- 特色：骑兵训练
- 难度：★★★困难（靠近会被征讨）
- 态度：好战(aggressive)

**国家特色**：
- ⚔️ **战神之女**：骑兵攻击+30%，训练速度×2
- 💰 **以战养战**：每场胜仗获得战利品翻倍
- 🌿 **游牧采集**：山地食物产出+20%
- 💘 **武斗招亲**：可通过比武招亲联姻
- 👶 **收养制度**：收养女婴增加人口

**人口繁衍方式**：
阿玛宗族通过与外族男子"武斗招亲"——在年度比武大会上，外族男子可参加，胜者与阿玛宗女子共度一夜后必须离开。怀孕后若生女则留在族中培养为战士，若生男则送还父亲或遗弃。此外也大量收养战争孤儿中的女婴。**收养制度**额外增加人口增长率+15%。

**独特奴隶市场**：
阿玛宗专营**女奴市场**，只售卖女性奴隶（女劳奴、女战奴、舞女、侍妾等），绝不售卖任何男性。所有男性战俘和劳奴自动按市价转化为金币。这是她们的重要经济来源。

**故事线**：
1. **初遇阿玛宗**：玩家靠近阿玛宗领地时，遭遇巡逻的女战士，触发战斗或外交选择
2. **比武招亲**：参加阿玛宗年度比武大会，获胜可获得联姻资格和武将奖励
3. **战神试炼**：完成阿玛宗的三重试炼（骑射、格斗、战术），获得"战神之女"称号
4. **阿玛宗联盟**：与阿玛宗女王结盟，获得精锐女骑兵支援
5. **征服阿玛宗**：军事路线，需先完成其周边3个部落任务

**关键角色**：
- **阿玛宗女王·赫拉**（统率95/武力90/智力75）：阿玛宗族的传奇女王
- **骑兵统领·阿尔忒**（统率88/武力92/智力65）：最强女骑兵将领
- **弓骑队长·彭忒**（统率80/武力85/智力70）：神射手
- **驯马师·希波**（统率60/武力55/智力80）：天马驯养专家

- 特色：纺织医术
- 难度：★简单（友善，会馈赠）

- 🌺 **百花种植**：药草和花卉产出+40%

**故事线**：

5. **西梁联盟**：与西梁女国结盟，获得丝绸贸易垄断权
6. **征服西梁**：军事路线（极不推荐，会触发全国抵抗事件）

## 5. 36国故事线

### 5.1 主要大国

#### 楼兰 🏜️
- **类型**：定居城邦 | **难度**：★★普通 | **地形**：沙漠
- **特色**：玉石贸易、丝路关税
- **主线任务链**：
  1. "罗布泊的呼唤"：初到楼兰，帮助商队击退沙匪（战斗/外交）

## 6. 主线剧情

### 第一章：无名小卒（回合1-12）

**起始**：玩家选择一个城邦国家，以流浪者身份进入。

**剧情线**：
1. 抵达城邦，被城门守卫拦住，需选择：贿赂(10金)/展示武力/巧言说服
2. 进入城邦，遇到引路人（城邦长老/商人/武者，根据选择的国家类型不同）
3. 接受城邦考验任务：进贡100金，获得初步认可
4. 完成考验后获得管理权，开始经营自己的小领地
5. 触发第一个部落事件，学习基础能力

**关键选择**：
- 政治路线：与城邦贵族交好，通过政治手段获取权力
- 商业路线：从商队做起，积累财富后购买权力
- 军事路线：从军入伍，通过战功获得晋升

### 第二章：部落征途（回合13-36）

**核心**：完成周边部落任务，解锁各种能力。

**剧情线**：
1. 探索周边部落，每个部落有独特的文化和考验
2. 完成部落考验→获得资源和能力
3. 完成部落盟约→获得武将和领土
4. 遭遇第一个全女性国家（根据地理位置）
5. 触发"三国会盟"前置事件

**关键事件**：
- "流浪武者"：随机遇到一位隐世高手，可选择招募
- "商队遇袭"：护送商队击退匪徒，获得商业声望
- "部落冲突"：两个部落发生冲突，选择支持一方
- "女国来使"：全女性国家派遣使者，开启外交

### 第三章：城市发展（回合37-72）

**核心**：将城市发展壮大，解锁高级能力。

**剧情线**：
1. 城市升级至Lv2，解锁外交和征兵
2. 与周边国家建立外交关系
3. 组建第一支正规军
4. 开辟第一条远程商路
5. 触发"西域风云"事件——大国之间的博弈

**关键选择**：
- 与汉朝结盟 vs 保持独立
- 与匈奴对抗 vs 与匈奴合作
- 支持哪个全女性国家（影响后续联盟）

### 第四章：扩张征伐（回合73-120）

**核心**：攻城略地，扩大领土。

**剧情线**：
1. 城市升级至Lv3，解锁宣战和谍报
2. 完成目标国50%以上部落任务
3. 发动征服战争或通过外交吞并
4. 逐步统一周边小国
5. 触发"三国联盟"或"三国对抗"事件

**关键战役**：
- 沙漠之战：在塔克拉玛干沙漠边缘的攻防战
- 草原之战：与游牧大国的骑兵对决
- 山地之战：攻取全女性国家的山地要塞
- 丝路之战：控制丝绸之路关键节点

### 第五章：统一西域（回合121+）

**核心**：最终统一所有36国。

**剧情线**：
1. 面对最后的强敌（乌孙/康居/突骑施等大国）

3. 最终决战

**结局**：

## 附录A：全女性国家征服继承规则

### 核心规则：前2城继承制度

**任何国家攻掠的前2个城邦，其资源、制度（包括福利）都需要跟原国家保持一致。**

#### 全女性国家征服规则

当全女性国家（阿玛宗/西梁女国/东女国）攻下前2个城邦时：

| 项目 | 规则 |
|------|------|
| **城邦属性** | 自动变为全女性城邦，与母国制度一致 |
| **男性武将** | 不能招募男性武将；已有的男性武将自动按属性折算为金币遣散 |
| **男性公民** | 不能拥有男性公民；现有男性公民自动转化为金币（每人50金） |
| **男性奴隶** | 不能拥有男性奴隶；现有男性奴隶自动按市价转化为金币 |
| **女性公民** | 所有公民均为女性，可正常分配为商人/农民/工匠/士兵 |
| **女性奴隶** | 所有奴隶均为女性（女劳奴、舞女、侍妾、女战奴等） |
| **兵种** | 只能招募女性兵种（女步兵、女骑兵、女弓骑兵等） |
| **人口增长** | 继承母国的人口增长加成（子母河/收养制度/母系传承） |
| **资源产出** | 继承母国的资源产出加成 |
| **建筑** | 保留原有建筑，但功能调整为全女性版本 |

#### 全女性国家人口增长加成（含基础增长率）

| 国家 | 基础增长率 | 额外加成 | 最终效果 |
|------|-----------|----------|----------|
| **阿玛宗** | 0.5%~2.0%/月 | 收养制度+15% | 基础×1.15 |
| **西梁女国** | 0.5%~2.0%/月 | 子母河+30% | 基础×1.30 |
| **东女国** | 0.5%~2.0%/月 | 母系传承+25% | 基础×1.25 |

#### 普通国家征服规则

当普通国家攻下前2个城邦时：
- 继承母国的资源制度（税收政策、贸易政策等）
- 继承母国的福利制度（公民待遇、奴隶政策等）
- 保留原有男女比例，正常运作

#### 第3个及之后的征服城邦

从第3个征服城邦开始，玩家可以选择：
- **同化**：将新城邦制度改为与母国一致（需要额外金币和回合）
- **自治**：保留新城邦原有制度（产出略低，但无转化成本）
- **全女性国家特殊**：第3个及之后的城邦可选择是否变为全女性城邦

## 附录G：月度结算流程

```
每月结算顺序：
1. 农业产出 → 粮食入库（受季节+等级+奴隶加成）
2. 工业产出 → 矿石/木材入库
3. 铁匠铺转化 → 矿石→铁矿
4. 商业产出 → 金币入库（受季节+等级+商人+文官加成）
5. 军事消耗 → 粮食/金币扣除（冬季额外消耗）
6. 人口增长 → 人口增加（受等级+粮食+国家特色影响）
7. 娱乐收入 → 舞女/乐师产出金币
8. 税收收入 → 按人口征收金币
9. 属地收入 → 控制部落贡献粮食+金币
10. 商路收入 → 远程商路产生金币
11. 武将经验 → 全体武将+10经验（可能升级）
12. 文官经验 → 全体文官+8经验（可能升级）
13. 奴隶培训 → 检查是否有培训完成
14. 随机事件 → 触发剧情/灾难/机遇事件
```
