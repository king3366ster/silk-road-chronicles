## 1. 世界观概述

**时代背景**：西汉末年至东汉初年，丝绸之路刚刚打通，西域39国林立，汉匈争霸的余波未平。玩家扮演一个无名小卒，从一穷二白开始，通过政、商、军三条路线逐步崛起，最终统一西域诸国。

**核心循环**：
```
农业→粮食→人口→劳动力→资源采集→工业→兵器→军事→征服→领土→农业
  ↕                                                                    ↕
商业←←←←←←←←←←←←←←←← 金币 ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
  ↕
外交→联盟→贸易协定→商业加成→金币→城市建设→人口→军事
```

### 2.2 消耗类型区分：一次性消耗 vs 月度消耗

> **核心设计**：所有资源消耗严格区分为**一次性消耗**（发生即扣除）和**月度消耗**（每月结算时扣除）。这是经济规划的基础。

#### 一次性消耗（发生时立即扣除）

| 消耗项目 | 消耗资源 | 触发时机 | 说明 |
|----------|----------|----------|------|
| **征召士兵** | 1公民+1兵器+15~35金 | 征兵时 | 公民脱离生产，兵器装备消耗 |
| **训练士兵** | 10~30金/人 | 训练开始时 | 新兵→普通兵10金，→精兵20金，→老兵30金 |
| **建设建筑** | 金币+木材+石材+劳力 | 建设开始时 | 见附录G建筑表 |
| **购买奴隶** | 50~150金/人 | 购买时 | 含距离浮动+市场浮动 |
| **培养武将/文官** | 350~500金+60~100粮 | 培养开始时 | 培养期间奴隶无法工作 |
| **外交赠礼** | 各种资源 | 赠送时 | 丝绸+20好感、玉石+25、香料+30等 |
| **外交谈判** | 100~500金 | 谈判时 | 联盟/互不侵犯/联姻 |
| **科技研发** | 200~800金+科技点 | 研发开始时 | 解锁新建筑/兵种/能力 |
| **开辟商路** | 200金+2马匹 | 开辟时 | 建立远程贸易路线 |
| **攻城战消耗** | 兵器损耗+兵力 | 战斗结束时 | 见兵器损耗表 |
| **职业重分配** | 1回合（时间成本） | 重新分配时 | 公民换职业 |
| **奴隶转化** | 300金+20回合 | 转化开始时 | 奴隶→公民 |
| **同化城邦** | 500~2000金 | 同化时 | 将新城邦制度改为与母国一致 |
| **市场购买资源** | 金币（含距离浮动） | 购买时 | 所有15种资源均可购买 |
| **酿造美酒** | 2粮→1美酒 | 酿造时 | 消耗粮食产出美酒 |
| **锻造兵器** | 2铁矿→1兵器 | 锻造时 | 铁匠铺转化 |
| **加工玉器** | 5玉石→1玉器 | 加工时 | 售价×3 |
| **炼丹** | 5药草→1丹药 | 炼制时 | 武将经验/治疗 |
| **治疗伤兵** | 10药草 | 治疗时 | 恢复100兵 |

#### 月度消耗（每月结算时扣除）

| 消耗项目 | 消耗资源 | 频率 | 说明 |
|----------|----------|------|------|
| **男公民粮食** | 1粮/人/月 | 每月 | 所有男公民（含农民/矿工/商人等） |
| **女公民粮食** | 0.8粮/人/月 | 每月 | 所有女公民（含农民/纺织工/商人等） |
| **奴隶粮食** | 0.6粮/人/月 | 每月 | 所有奴隶（含劳奴/匠奴/战奴/舞女/侍妾） |
| **士兵粮食** | 1.5粮/人/月（新兵/普通兵）或1.8粮/人/月（精兵/老兵） | 每月 | 从公民中征召的士兵 |
| **马匹马料** | 2粮/匹/月 | 每月 | 所有马匹（不区分战马/普通马匹） |
| **盐消耗** | 总人口÷100/月 | 每月 | 人口必需品，不足满意度-20% |
| **税收** | 人口×0.5金/月 | 每月 | **收入**（城市Lv2后） |
| **军费维护** | 兵力×1金/100兵/月 | 每月 | 基础军费 |
| **兵器日常损耗** | 0.5%/月 | 每月 | 所有装备兵器的士兵 |
| **兵器冬季额外损耗** | +1%/月 | 冬季（10-12月） | 严寒导致兵器脆裂 |
| **低治安兵器损耗** | +1%/月 | 治安<40时 | 盗窃、破坏 |
| **无铁匠铺额外损耗** | +2%/月 | 无铁匠铺时 | 无法修复 |

| **商路收入** | 金币/月 | 每月 | **收入**（开辟商路后） |
| **酒吧收入** | +5金/酒吧/月 | 每月 | **收入**（有酒吧建筑时） |

#### 消耗流程图

```
每月结算顺序（含消耗类型标注）：

1. 【月度产出】农业/工业/纺织/商业产出 → 资源入库
2. 【月度消耗】男公民粮食(1粮/人) + 女公民粮食(0.8粮/人) + 奴隶粮食(0.6粮/人) + 士兵粮食(1.5~1.8粮/人)
3. 【月度消耗】马匹马料(2粮/匹)
4. 【月度消耗】盐消耗(人口÷100)
5. 【月度消耗】铁匠铺燃料(5木/月)
6. 【月度消耗】兵器日常损耗(0.5%)
7. 【月度消耗】军费维护(兵力÷100金)
8. 【月度消耗】武将/文官俸禄(10~50金/人)
9. 【月度消耗】建筑维护费
10.【月度收入】税收(人口×0.5金)
11.【月度收入】部落贡赋(各种资源)
12.【月度收入】商路收入(金币)
13.【月度收入】酒吧收入(+5金/酒吧)
14.【月度消耗】冬季额外消耗(皮毛+兵器损耗)
15.【一次性】检查是否有建设完成/训练完成/培养完成
16.【一次性】随机事件触发（可能产生一次性消耗或收入）
```

#### 关键经济指标

```
月度净收入 = 月度总收入 - 月度总消耗
  月度总收入 = 税收 + 商路收入 + 部落贡赋 + 酒吧收入 + 舞女收入
  月度总消耗 = 粮食消耗 + 盐消耗 + 军费 + 俸禄 + 建筑维护 + 燃料

一次性储备金 = 当前金库 - 预计一次性支出
  预计一次性支出 = 建设预算 + 征兵预算 + 外交预算 + 购买预算

健康经济标准：
  月度净收入 > 0（每月盈余）
  一次性储备金 > 月度总消耗×3（至少3个月缓冲）
  粮食储备 > 月度粮食消耗×6（至少6个月口粮）
```

### 2.5 人口-公民-奴隶-资源关系模型

> **核心设计**：人口数不是抽象数字，而是由具体的公民和奴隶组成。每个公民和奴隶都有职业分配，直接决定资源产出。人口越多→可分配的劳动力越多→资源产出越高→但粮食消耗也越大。

#### 人口构成公式

```
总人口 = 男公民 + 女公民 + 男奴隶 + 女奴隶 + 武将 + 文官 + 士兵

其中：
  公民 = 男公民 + 女公民（自然增长/奴隶转化获得）
  奴隶 = 男奴隶 + 女奴隶（市场购买/战俘/部落贡赋获得）
  武将/文官 = 从公民或奴隶中培养/任务奖励/招降获得（不计入劳动力）
  士兵 = 从公民中征召（脱离生产，消耗粮食1.5/月，精兵/老兵1.8/月）
```

#### 人口-资源产出计算公式

```
月度资源总产出 = Σ(公民产出) + Σ(奴隶产出) + 部落贡赋 + 建筑加成

具体计算：
  粮食产出 = 农民(♂×3+♀×2.5) + 男劳奴×3 + 女劳奴×3 + 猎人×1 + 牧民×1
           × 城市等级倍率 × 季节修正 × 地形修正 × 文官加成

  铁矿产出 = 矿工(♂×2+♀×1.5) + 男劳奴×2
           × 城市等级倍率 × 文官加成

  木材产出 = 伐木工(♂×3+♀×2.5) + 男劳奴×3
           × 城市等级倍率

  石材产出 = 采石工(♂×2+♀×1.5) + 男劳奴×2
           × 城市等级倍率

  丝绸产出 = 纺织工(♂×1+♀×2) + 女劳奴×2 + 女匠奴×3
           × 城市等级倍率 × 国家特色加成

  金币收入 = 商人×3(本地) + 商人×5(商路) + 舞女×10 + 税收(人口×0.5)
           × 城市等级倍率 × 季节修正 × 文官加成

月度资源总消耗：
  粮食消耗 = 男公民×1 + 女公民×0.8 + 奴隶×0.6 + 士兵×1.5(精兵/老兵×1.8) + 马匹×2
  盐消耗 = 总人口÷100
  金币消耗 = 军费 + 建设费 + 奴隶购买 + 科技研发
```

#### 人口增长对资源的影响链

```
人口增长 → 劳动力增加 → 资源产出增加 → 但粮食消耗也增加

关键平衡点：
  每新增1名男公民→分配农民→+3粮产出-1粮消耗=净+2粮/月 ✅
  每新增1名女公民→分配农民→+2.5粮产出-0.8粮消耗=净+1.7粮/月 ✅
  每新增1名男劳奴→分配农场→+2.4粮产出-0.6粮消耗=净+1.8粮/月 ✅
  每新增1名士兵→不产出→0-1.5粮消耗=净-1.5粮/月 ❌ 需要其他人养

最佳人口结构（粮食自给率最高）：
  农民45% + 纺织12% + 商业12% + 矿工8% + 伐木5% + 采石3% + 狩猎3% + 其他7% + 士兵5%
  → 粮食自给率≈110%（微盈余）
  → 但军事力量弱

  农民35% + 纺织10% + 商业10% + 矿工8% + 伐木5% + 采石3% + 士兵25% + 其他4%
  → 粮食自给率≈85%（需贸易补足）
  → 军事力量强

结论：人口越多→总产出越高→但需要合理分配职业→粮食是核心瓶颈
```

| 城市等级 | 最大人口 | 粮食自给上限人口 | 建议士兵比例 | 建议奴隶比例 | 核心瓶颈 |
|----------|----------|-----------------|-------------|-------------|----------|
| Lv1 村落 | 500 | 400(无贸易) | 5% | 15% | 人口少，产出低 |
| Lv2 镇集 | 2000 | 1500(无贸易) | 8% | 18% | 粮食开始紧张 |
| Lv3 城邦 | 5000 | 3500(无贸易) | 12% | 20% | 需要贸易补粮 |
| Lv4 大都 | 15000 | 8000(无贸易) | 15% | 22% | 严重依赖贸易 |
| Lv5 王都 | 30000 | 12000(无贸易) | 18% | 25% | 必须多城互补 |

> **核心结论**：城市等级越高、人口越多，粮食自给率越低（因为士兵和非农业人口比例增加）。**多城邦资源互补**是后期发展的必然选择——产粮城供养军事城，商业城提供金币。这也解释了为什么统一西域需要多城邦管理。

#### 途径二：任务奖励（主要来源）

**城邦任务奖励武将**：
- 完成城邦故事线后，奖励1-2名武将/文官
- **男女比例：5:5**（50%男性，50%女性）
- **母系社会国家**：男女比例3:7（30%男性，70%女性）
- **全女性国家**：奖励武将/文官均为女性
- **文官/武将区分**：根据任务类型决定（军事类→武将，商业/农业/外交类→文官）
- 城邦任务武将/文官属性较高（平均70+），有独特技能

**部落任务奖励武将**：
- 完成部落故事线后，奖励1名武将/文官（随机决定类型）
- **男女比例：3:7**（30%男性，70%女性）
- **全女性国家周边部落**：奖励武将/文官100%为女性
- **文官/武将区分**：部落奖励中约50%为武将（侧重武力/统率），50%为文官（侧重政治/智力）
- 部落武将/文官属性中等（平均50-70），有部落特色能力

**任务奖励武将属性侧重**：

| 任务类型 | 武将侧重 | 文官侧重 |
|----------|----------|----------|
| 军事类任务 | 统率+武力高 | 智力+统率中 |
| 商业类任务 | 魅力+智力高 | 政治+魅力高 |

| 探索类任务 | 智力+武力高 | 智力+魅力高 |
| 工艺类任务 | 智力+政治高 | 智力+政治高 |

#### 途径三：攻掠国家招降（批量来源）

- **征服一国后**：该国武将/文官有30%概率投降归顺
- **招降条件**：赠礼（100-500金）+ 说服（外交值判定）
- **忠诚度**：初始40，需提升至60+才能完全信任
- **男女比例**：按被征服国家的男女比例决定
- **全女性国家被征服**：招降武将100%为女性
- **属性保留**：招降武将保留原属性的80%，可通过后续培养恢复

#### 途径四：随机招聘（补充来源）

> **核心机制**：随机招聘不是主动操作，而是通过**公民养成事件触发**。每年系统会进行判定，满足条件时有概率触发招募事件。

- **触发方式**：通过以下公民养成事件概率触发
  - **军功提拔**：士兵参与战斗≥5场后，有一定概率触发「民间英才」事件，推荐武将候选人
  - **科举选拔**：城市等级≥Lv3后，每年春季可举行「科举」，有概率发现文官候选人
  - **民间推荐**：每回合有一定基础概率，由武将/文官推荐民间人才
- **触发频率**：大国每年约3人，中等国约2人，小国约2人（非固定，取决于事件触发概率）
- **招聘条件**：城市等级≥Lv2，金库≥200金
- **招聘消耗**：每人100-300金（根据属性决定）
- **男女比例**：根据城邦制度确定
  - **普通国家**：男女比例5:5
  - **母系社会国家**：男女比例3:7（30%男性，70%女性）
  - **全女性国家**：100%女性
- **属性范围**：平均45-65（低于任务奖励，高于奴隶培养初期）
- **文官/武将区分**：触发事件类型决定——军功类→武将，科举/内政类→文官，民间推荐→随机
- **特殊规则**：随机招聘的武将/文官忠诚度初始50（低于其他来源），需要通过赠礼/任务提升

> **随机招聘年度表**：

| 国家类型 | 年度招聘人数 | 触发事件 | 年度总成本 | 男女比例 |
|----------|-------------|----------|-----------|----------|
| 大国（8个） | ~3人/年 | 军功/科举/民间推荐 | 300-900金 | 见城邦制度 |
| 大国(全女)（1个） | ~3人/年 | 军功/科举/民间推荐 | 300-900金 | 100%女性 |

> **文官任务价值评估**：
> - **内政管理**是发展基石——同时提升粮食、金币、建设速度，一个优秀文官可抵50个劳动力
> - **外交谈判**是和平保障——一次成功的外交可避免一场战争（战争成本≈2000金+大量兵力）
> - **贸易管理**是经济命脉——贸易价格优化直接影响所有资源的获取成本
> - **科技研发**是长期投资——解锁高级建筑/兵种后，产出效率可提升20-30%
> - **人口管理**是扩张前提——人口是所有产出的基础，人口管理决定发展上限

### 3.6.2 奴隶-公民-武将/文官完整转化机制

> **核心设计**：人口在奴隶→公民→武将/文官之间有完整的转化路径，每一步都有明确的条件、消耗和时间。

```
【完整转化路径图】

奴隶 ──→ 公民 ──→ 武将/文官
  │         │         │
  │         │         └→ 带兵/外交/任务/故事线
  │         └→ 分配职业（农民/矿工/商人/士兵等）
  ├──→ 战奴整编 → 普通兵(Lv1) → 精兵/老兵

条件：奴隶服役≥10回合 + 城市等级≥Lv2
消耗：300金 + 20回合培养期

【转化路径三：奴隶 → 武将（舞女/侍妾路线）】
条件：舞女或侍妾 + 城市等级≥Lv3
消耗：450-500金 + 90-100粮 + 25-30回合培养期
效果：舞女/侍妾变为见习武将(Lv1)
产出：魅力+智力侧重（平均55-65），特殊技能（魅惑/谍报）

消耗：400金 + 80粮 + 25回合培养期
效果：匠奴变为见习文官(Lv1)
产出：智力+政治侧重（平均50-60），技术型文官

条件：武将/文官忠诚度<30 或 玩家主动解职
消耗：无

## 3.7 军事系统：兵源、训练与战斗

> **核心设计**：军事系统围绕"兵源→训练→战斗→损耗→补充"闭环设计。士兵不是抽象数字，而是从公民中征召的活人，有等级、有装备、有消耗。城邦治安、兵器损耗等机制都有实际的战略意义。

### 3.7.1 士兵等级体系

| 等级 | 名称 | 战斗力 | 训练时间 | 训练消耗 | 兵器损耗率 | 月粮耗 | 实际意义 |
|------|------|--------|----------|----------|-----------|--------|----------|
| Lv0 | **新兵** | 基础×0.5 | — | — | 5%/战 | 2粮 | 刚征召的公民，无战斗经验，阵亡率高 |
| Lv1 | **普通兵** | 基础×1.0 | 3回合 | 10金/人 | 3%/战 | 2粮 | 经过基础训练，可执行常规任务 |
| Lv2 | **精兵** | 基础×1.5 | 6回合 | 20金/人 | 1.5%/战 | 2.5粮 | 训练有素，可执行精锐任务，战损低 |
| Lv3 | **老兵** | 基础×2.0 | 10回合 | 30金/人 | 0.5%/战 | 2.5粮 | 百战余生，可训练新兵，极低战损 |

> **战斗力计算**：基础战斗力由兵种决定（步兵10/骑兵15/弓骑兵12）。实际战斗力 = 基础×等级倍率×武将加成×士气修正×地形修正。

#### 士兵升级路径

```
公民 → 征召(消耗1兵器+15金) → 新兵(Lv0)
新兵 → 武将训练3回合(消耗10金/人) → 普通兵(Lv1)
普通兵 → 武将训练6回合(消耗20金/人) → 精兵(Lv2)
精兵 → 战斗经验累积+武将训练10回合(消耗30金/人) → 老兵(Lv3)

战奴 → 整编(1兵器+10金) → 普通兵(Lv1，战力×0.7，忠诚40)，可后续升级
```

#### 各等级士兵实际价值对比

| 指标 | 新兵(Lv0) | 普通兵(Lv1) | 精兵(Lv2) | 老兵(Lv3) |
|------|-----------|-------------|-----------|-----------|
| **战斗力** | 5~7 | 10~15 | 15~22 | 20~30 |
| **阵亡率(攻城)** | 40% | 25% | 15% | 8% |
| **阵亡率(野战)** | 30% | 18% | 10% | 5% |
| **兵器损耗/战** | 5% | 3% | 1.5% | 0.5% |
| **可执行任务** | 守城 | 守城/巡逻/护卫 | 全部军事任务 | 全部+训练新兵 |
| **培养总成本** | 15金+1兵器 | 25金+1兵器 | 45金+1兵器 | 75金+1兵器 |
| **性价比** | ❌极低 | ✅基础 | ✅✅高 | ✅✅✅最高 |

> **核心结论**：精兵和老兵虽然培养成本高，但阵亡率和兵器损耗极低，长期来看是最经济的选择。一支1000精兵的军队战斗力等于1500新兵，但粮食消耗相同、兵器损耗仅为新兵的30%。

### 3.7.3 兵种克制关系

> **核心设计**：三大兵种之间存在循环克制关系，克制方对被克制方造成**+10%伤害**，被克制方受到**+10%额外伤害**。

```
克制循环图：

    步兵 ──克制──→ 弓骑兵
     ↑                 │
     │                 ↓
   骑兵 ←──克制── 步兵

说明：
  步兵 → 克制弓骑兵（密集方阵抵消远程优势）
  弓骑兵 → 克制骑兵（远程射击消耗高速目标，先手优势）
  骑兵 → 克制步兵（骑兵冲锋碾压步兵阵型）
```

| 攻方 | 守方 | 克制关系 | 伤害修正 | 说明 |
|------|------|----------|----------|------|
| **步兵** | 弓骑兵 | ✅克制 | 攻+10%/守伤+10% | 密集方阵抵消远程 |
| **弓骑兵** | 骑兵 | ✅克制 | 攻+10%/守伤+10% | 远程先手消耗 |
| **骑兵** | 步兵 | ✅克制 | 攻+10%/守伤+10% | 冲锋碾压阵型 |
| 弓骑兵 | 步兵 | ❌被克制 | 攻-10%/守伤+10% | 被密集方阵抵消 |
| 骑兵 | 弓骑兵 | ❌被克制 | 攻-10%/守伤+10% | 被远程先手消耗 |
| 步兵 | 骑兵 | ❌被克制 | 攻-10%/守伤+10% | 被冲锋碾压 |
| 同兵种 | 同兵种 | 无克制 | ×1.0 | 正常伤害 |

> **战术意义**：克制关系意味着纯兵种军队容易被针对性克制。最佳军队配置需要混合兵种，利用克制优势。例如：敌方以弓骑兵为主→我方多训练步兵；敌方以骑兵为主→我方多训练弓骑兵。

### 3.7.4 城邦治安系统

> **城邦治安**不是抽象数值，而是直接影响经济产出、人口增长、军事稳定的核心指标。

| 治安等级 | 治安值 | 效果 | 触发事件概率 |
|----------|--------|------|-------------|
| **太平** | 80-100 | 产出+5%、人口增长+10%、商路安全+20% | 无负面事件 |
| **安定** | 60-79 | 正常运作 | 盗窃-5%概率（损失少量金币） |
| **不安** | 40-59 | 产出-10%、人口增长-5%、商路收入-15% | 暴乱10%/月、逃兵5%/月 |
| **动荡** | 20-39 | 产出-25%、人口增长-15%、商路中断-30% | 暴乱25%/月、逃兵15%/月、投敌5% |
| **混乱** | 0-19 | 产出-50%、人口流失-30%、商路完全中断 | 暴乱40%/月、逃兵30%/月、叛变10% |

#### 治安影响因素

```
治安值 = 基础50 + 驻军加成 + 武将巡逻加成 + 建筑加成 + 事件修正

驻军加成：
  每百名驻军(普通兵+) → 治安+3
  每百名驻军(新兵) → 治安+1（新兵维持治安效果差）
  驻军不足人口5% → 治安-10

武将巡逻加成：
  武将执行"巡逻护卫"任务 → 治安+15

建筑加成：
  城墙 → 治安+5
  瞭望塔 → 治安+5
  衙门 → 治安+10

负面因素：
  粮食不足 → 治安-20
  税率过高(>15%) → 治安-10
  战败 → 治安-15
  敌国谍报 → 治安-10
  冬季 → 治安-5
```

#### 治安的实际经济影响（以人口5000城邦为例）

| 治安等级 | 月产出损失 | 商路损失 | 人口流失 | 逃兵损失 | 总经济损失/月 |
|----------|-----------|----------|----------|----------|-------------|
| 太平(80+) | 0 | 0 | 0 | 0 | 0 |
| 安定(60-79) | 0 | 0 | 0 | 0 | ≈0 |
| 不安(40-59) | -250金 | -100金 | -25人 | -5兵 | ≈-400金 |
| 动荡(20-39) | -625金 | -200金 | -75人 | -15兵 | ≈-1000金 |
| 混乱(0-19) | -1250金 | -400金 | -150人 | -30兵 | ≈-2000金 |

> **结论**：治安低于40时，每月经济损失可达1000金以上，远超雇佣武将巡逻的成本（武将巡逻≈50金/月）。**治安是投资，不是消耗**。

### 3.7.5 兵器损耗系统

> **兵器损耗**不是简单的数字减少，而是反映军队维护成本的核心机制。

#### 兵器损耗来源

| 损耗来源 | 损耗率 | 说明 |
|----------|--------|------|
| **战斗损耗** | 见士兵等级表 | 新兵5%/战，精兵1.5%/战，老兵0.5%/战 |
| **日常维护** | 0.5%/月 | 所有装备兵器的士兵每月损耗 |
| **恶劣环境** | +1%/月(冬季) | 严寒导致兵器脆裂 |
| **低治安** | +1%/月(治安<40) | 盗窃、破坏 |
| **缺乏维护** | +2%/月(无铁匠铺) | 无法修复损坏兵器 |

#### 兵器损耗的经济影响

```
以1000士兵为例（全部装备兵器）：

新兵军团(1000人)：
  战斗损耗：5%×1000=50兵器/战 → 50×20金=1000金/战
  日常损耗：0.5%×1000=5兵器/月 → 5×20金=100金/月
  年度维护成本≈1200金+100金×12=2400金/年

精兵军团(1000人)：
  战斗损耗：1.5%×1000=15兵器/战 → 15×20金=300金/战
  日常损耗：0.5%×1000=5兵器/月 → 5×20金=100金/月
  年度维护成本≈300金+100金×12=1500金/年

老兵军团(1000人)：
  战斗损耗：0.5%×1000=5兵器/战 → 5×20金=100金/战
  日常损耗：0.5%×1000=5兵器/月 → 5×20金=100金/月
  年度维护成本≈100金+100金×12=1300金/年

结论：精兵/老兵的兵器损耗仅为新兵的30%/10%，长期节省大量金币
```

#### 兵器修复机制

| 修复方式 | 修复率 | 成本 | 说明 |
|----------|--------|------|------|
| **铁匠铺修复** | 损耗的50% | 1铁矿/修复5兵器 | 基础修复 |
| **兵器坊修复** | 损耗的70% | 1铁矿/修复8兵器 | 高级修复 |
| **武将训练加成** | 损耗-15% | 无额外成本 | 训练有素的士兵更爱惜装备 |

### 3.7.6 交战系统

> **核心设计**：交战系统参考《战国兰斯》回合制框架，分为**掠夺**和**攻占**两种模式。每场战斗由多位武将各自带兵参与，按「远程→骑兵→步兵」回合顺序推进，结合武将特技和士气系统决定胜负。战后根据模式和目标城邦特产计算战利品（资源+奴隶）。

#### 3.7.6.1 两种交战模式

| 项目 | 掠夺模式 | 攻占模式 |
|------|---------|---------|
| **目标** | 抢夺资源+奴隶，不占领 | 彻底征服，占领城邦 |
| **战斗方式** | 野战速攻，守军出城迎战 | 攻城战，需破城墙 |
| **兵力损耗率** | 标准×0.5（速战速决） | 标准×1.5（攻城惨烈） |
| **兵器损耗率** | 标准×0.6 | 标准×1.5 |
| **战斗回合数** | 1~3回合 | 3~8回合（含破城阶段） |
| **收益** | 按比例掠夺特产+人口 | 全部残存资源+领土+人口 |
| **风险** | 低（可随时撤退） | 高（无法撤退，战败损失大） |
| **城防加成** | 守方无城防（出城野战） | 守方城墙+城防工事 |
| **外交影响** | 中等（宣战行为） | 严重（灭国行为，全西域警戒） |

**掠夺模式**：选择掠夺后，守军出城迎战。攻方无需攻破城墙，只需在野战中击溃守军即可洗劫。掠夺时可选择"劫掠"指令，武将带领部分兵力脱离主战场执行掠夺。掠夺持续1~3回合，回合数越长掠夺比例越高，但被援军截击风险越大。

**攻占模式**：选择攻占后，需先经历**围城→破墙→巷战**三阶段。每阶段可消耗攻城器械加速推进。破城后守军士气大幅下降，可选择劝降或强攻。攻占成功则获得城邦完全控制权（需后续同化）。

**模式切换规则**：
- 掠夺过程中可选择"转攻占"（消耗1回合重整阵型，城墙耐久重置为满）
- 攻占过程中不可转掠夺（已破城）
- 任一模式均可选择"撤退"（掠夺撤退无损，攻占撤退损失已损耗兵力）

#### 3.7.6.2 回合制战斗流程

参考《战国兰斯》回合制交战系统，每回合按五个阶段依次执行：

```
【每回合战斗顺序】

阶段0：战前布阵（仅第1回合）
  攻方：选择出战武将（最多5名），分配各武将带兵数
  守方：选择守城武将（最多3名），分配城防策略
   ⟶ 生效：武将统率决定带兵上限，智力决定先手判定

阶段1：远程射击（弓骑兵阶段）
  双方弓骑兵同时射击，造成远程伤害
  伤害 = 弓骑数量 × 12(基础战力) × 等级倍率 × 地形修正 / 10
  弓骑兵克制骑兵：对骑兵伤害+10%
   ⟶ 生效：武将特技（如火攻可在本阶段发动）

阶段2：骑兵冲锋（骑兵阶段）
  攻方骑兵优先冲锋，守方骑兵迎击
  伤害 = 骑兵数量 × 15 × 等级倍率 × 地形修正 / 10
  骑兵克制步兵：对步兵伤害+10%
  冲锋后可选择"追击败军"(扩大战果)或"撤回重整"(减少损失)
   ⟶ 生效：武将特技（如冲锋/单挑可在本阶段发动）

阶段3：步兵接战（主力决战阶段）
  剩余所有兵种（含未出战的预备队）投入近战
  本方总战斗力 vs 敌方总战斗力 → 决定本回合胜负
   ⟶ 生效：武将特技（如鼓舞/坚守可在本阶段发动）

阶段4：武将特技发动（可随时插入前三个阶段的任意时机）
  每个武将每场战斗战意值上限100，初始60，每回合+15
  发动特技消耗战意值（详见3.7.6.6武将特技表）
   ⟶ 效果：改变本回合战斗结算

阶段5：士气结算（每回合末）
  基础士气变化：
    胜方：士气+10
    败方：士气-15
  额外影响：武将魅力>70 → 本方士气额外+5；武将魅力<40 → 本方士气额外-5
  特殊事件：单挑胜→敌方士气-30；主将被斩→敌方士气-50
  士气归零 → 该部队溃败（下一回合开始时若士气仍为0则退出战斗）
```

**战争结束条件**：

| 模式 | 攻方胜利 | 守方胜利 | 平局/撤退 |
|------|---------|---------|----------|
| 掠夺 | 守方兵力<50% 或 守方士气<30 → 掠夺成功 | 攻方兵力<30% 或 攻方士气<20 → 掠夺失败 | 攻方主动撤退 |
| 攻占 | 城墙耐久=0且守方兵力<20% 或 守方士气=0 → 占领成功 | 攻方兵力<40% 或 攻方士气<20 → 攻占失败 | 攻方主动撤退（损失已损耗兵力） |

**战场撤退规则**：
- 掠夺模式：任何时候可撤退，无损
- 攻占模式：围城阶段可撤退（损已耗兵力），巷战阶段不可撤退（只能死战或投降）

> - **舞女/侍妾**多来自文教/商贸国家（龟兹、于阗、姑墨、楼兰、疏勒、粟特、鄯善等），好战国家（乌孙、康居、莎车、大宛、北庭、若羌等）产出概率极低或无。

**部落掠夺规则**：
- 每个部落被掠夺后，3个月内停止贡赋；掠夺期间无法获取该部落的任何资源
- 可消耗金币+特产修复关系（消耗=部落月产出×10金），修复后贡赋恢复
- 同一部落连续被掠夺，性价比逐次递减：

| 掠夺次数(连续) | 掠夺收益比例 | 守军强度 | 战后关系恢复难度 |
|---------------|------------|---------|----------------|
| 第1次 | 100% | 标准 | 可修复（标准成本） |
| 第2次 | 70% | ×1.2（部落有了防备） | 修复成本×2 |
| 第3次 | 45% | ×1.5 | 修复成本×4 |
| 第4次+ | 30% | ×2.0 | 修复成本×8（基本等于永久敌对） |

> **连续掠夺重置**：间隔≥6个月未掠夺该部落，计数重置为0。连续掠夺的"守军强度"加成也应用于部落守军战斗力计算。
> **部落不会永久毁灭**：即使连续掠夺多次，部落只是难以修复关系，不会从地图上消失。但修复成本过高（第4次+需8倍月产出）意味着实际上已无法再利用该部落。

> **不可掠夺的部落**：
> - **己方部落**：玩家已控制城邦对应的初始部落，以及通过征服获得的附属部落，不可掠夺
> - **联盟部落**：已缔结联盟关系的友方城邦对应的部落，不可掠夺（但可解除联盟后掠夺）
> - **归附部落**：通过外交或军事手段使其归附的部落，自动成为己方资源来源，不可掠夺
> - 对于上述部落，只能通过外交/任务互动（如修缮关系、提升好感），无法发起掠夺指令

**掠夺奴隶公式**：
```
掠夺奴隶数 = 守方总人口 × 掠夺比例 × 武将魅力修正
  大胜：5% → 约50~750人（取决于城邦大小）
  小胜：3%
  惨胜：1%

奴隶类型分配：
  普通城邦：按人口结构随机（男劳奴50%/女劳奴30%/匠奴10%/战奴10%）
  全女性城邦：100%女性奴隶（女劳奴/女匠奴/舞女/女战奴）
  母系城邦：女性奴隶概率+30%
```

#### 3.7.6.6 战斗力计算公式（保留原公式作为回合结算基础）

```
部队总战斗力 = Σ(单兵战斗力) × 武将加成 × 士气修正 × 地形修正 × 情报修正

其中：
  单兵战斗力 = 兵种基础 × 等级倍率
  武将加成 = 1 + (统率-50)×0.5% + 军衔加成
  士气修正 = 0.5~1.5（受粮食/季节/连胜影响）
  地形修正 = 0.7~1.35（见兵种地形表）
  情报修正 = 有谍报+1.15，无谍报×1.0，被谍报×0.85
```

**回合胜负判定**（替代原一次性判定）：

```
每回合胜负 = 攻方战斗力 vs 守方战斗力 × 当前防御加成
  当前防御加成（随攻城进度变化）：
    围城阶段：守军城内防御 ×1.5（有城墙×1.8）
    破墙阶段：城墙耐久每减少25%，防御加成-0.15
    巷战阶段：×1.2

胜方本回合损失 = 败方战斗力 / 胜方战斗力 × 败方损失率
  大胜(攻>守×1.5)：胜方损失率5%，败方损失率30%
  小胜(攻>守×1.2)：胜方损失率10%，败方损失率20%  
  惨胜(攻>守×1.0)：胜方损失率20%，败方损失率15%
  败北：胜方损失率5%，败方(攻方)损失率25%
```

#### 3.7.6.7 战斗实例

```
实例1：掠夺模式 — 楼兰掠夺于阗

攻方：楼兰，武将·尉屠(统率75/武力70/智力60)，带兵1000(骑兵400+步兵600)
守方：于阗，武将·尉迟(统率65/武力60/智力55)，带兵600(步兵400+弓骑兵200)

  远程阶段：于阗弓骑200×12×1.0(普通兵)=2400 ÷10=240伤害
    → 楼兰损失24人
  骑兵阶段：楼兰骑兵400×15×1.0=6000÷10=600伤害
    → 于阗损失60人（弓骑兵被骑兵克制+10%→660伤害→66人）
  步兵阶段：楼兰战力(576×10)=5760 vs 于阗战力(534×10+200×12)=7740
    → 楼兰战力5760 < 于阗7740×1.0 → 楼兰败北
    → 楼兰损失25%×576=144人，于阗损失15%×600=90人
  士气结算：楼兰士气-15→45，于阗士气+10→80

第2回合：
  楼兰武将发动"鼓舞"(战意20)：士气+15→60
  远程阶段：于阗弓骑134×12=1608÷10≈160伤害→楼兰损失16人
  骑兵阶段：楼兰骑兵376×15=5640÷10=564→于阗损失56人

掠夺收益（2回合）：
  玉石：于阗玉石库500×25%=125玉石
  丝绸：于阗丝绸库300×18%=54丝绸
  金币：于阗金库2000×25%=500金
  奴隶：于阗人口10000×3%=300人（女劳奴210+女匠奴60+舞女30）


实例2：攻占模式 — 疏勒攻占姑墨

攻方：疏勒，武将·萨法尔(统率80/武力75/智力55)，带兵1500(步兵800+骑兵500+弓骑200)

  姑墨选择"坚守"：防御+10%，士气-5

第3-5回合(破墙)：
  冲车每回合造成30点城墙伤害
  第3回合：城墙耐久500→470，守军弓骑远程攻击→疏勒损失30人
  第4回合：城墙耐久470→440，守军弓骑攻击→疏勒损失25人

### 3.7.7 军事与经济的关系链

```
【军事投入链】
  公民 → 征召(失去劳动力) → 新兵 → 训练(消耗金币) → 精兵/老兵
  兵器 → 装备(消耗铁矿) → 维护(每月损耗) → 修复(消耗铁矿)
  战马 → 骑兵(消耗马匹) → 马料(每月消耗粮食)

【军事回报链】
  胜利 → 战利品(金币+资源) → 战俘(奴隶) → 领土(人口+产出)
  军事实力 → 外交筹码 → 贸易优惠 → 经济加成
  驻军 → 城邦治安 → 产出稳定 → 经济安全

【军事风险链】
  战败 → 兵力损失(人口减少) → 兵器损失 → 金币损失 → 治安下降
  常备军 → 粮食消耗(每兵1.5粮/月+骑兵马料2粮/匹/月) → 农业压力 → 需要更多农民
  过度征兵 → 劳动力不足 → 产出下降 → 经济崩溃

核心平衡：
  1000精兵的年度总成本 ≈ 1500金(训练)+1200金(兵器维护)+18000粮(粮食1.5×1000×12)
  1000精兵的年度回报 ≈ 治安保障(避免-1000金/月损失)+战争收益(胜仗+2000金/次)
  → 军事投入约18个月可回本（纯经济角度）
  → 但没有军事力量，城邦随时可能被征服（一切归零）
```

> **区别于三大全女性国家**：母系社会国家有男有女，但女性在政治、经济、军事中占据主导地位。男性可担任低级官职和士兵，但高级决策权在女性手中。

**母系社会国家共同特征**：
- 女性公民可担任所有职务（商人/农民/工匠/士兵/官员）

- 舞女/侍妾类奴隶产出+15%
- 可通过"母系改革"事件转变为更激进的女性主导模式

## 6. 剧情任务体系

### 6.1 三条初始剧情线（互斥，选择后不再触发）

玩家开局时选择一条初始剧情线，决定整体游戏走向：

#### 线路A：西域统一线 🏆
**主题**：从无名小卒到西域霸主，统一39国
**触发**：选择任意非全女性国家开局时默认激活
**核心剧情**：
1. **无名小卒**（回合1-12）：抵达城邦→通过考验→获得管理权→经营小领地
2. **部落征途**（回合13-36）：完成周边部落任务→解锁能力→遭遇全女性国家
3. **城市发展**（回合37-72）：城市升级→外交征兵→组建正规军→开辟商路
4. **扩张征伐**（回合73-120）：攻城略地→统一周边→大国对决
5. **统一西域**（回合121+）：最终决战→建立新王朝

**结局分支**：霸王结局（军事）/ 商王结局（经济）/ 圣王结局（文化外交）

#### 线路B：三国女国联盟线 🌸
**主题**：联合三大全女性国家，建立母系王朝
**触发**：选择全女性国家（阿玛宗/西梁女国/东女国）开局时激活；或选择母系社会国家且完成"女国来使"事件后可切换
**核心剧情**：
1. **女儿初醒**（回合1-12）：在全女性国家中成长→完成国内考验→获得女王信任
2. **三国寻访**（回合13-48）：依次寻访三大全女性国家→完成各国独特考验→建立初步联系
3. **女国联盟**（回合49-96）：促成三国会盟→解决三国矛盾→建立女国联盟
4. **母系西征**（回合97-144）：以女国联盟力量征服/外交收服其他国家
5. **母系王朝**（回合145+）：建立母系王朝→选择治理模式

**结局分支**：战神女王结局（阿玛宗路线）/ 织女女王结局（西梁路线）/ 女王联盟结局（三国均衡）

#### 线路C：丝路传奇线 💰
**主题**：以商业力量控制西域，建立丝路帝国
**触发**：选择粟特/疏勒等商贸国家开局时激活；或在其他线路中商业声望达到"传奇"后可切换
**核心剧情**：
1. **商队学徒**（回合1-12）：加入商队→学习经商→完成第一笔贸易
2. **丝路开拓**（回合13-36）：开辟新商路→建立贸易站→遭遇商路匪患
3. **商业帝国**（回合37-72）：控制关键商路→建立金融网络→商业外交
4. **经济霸权**（回合73-120）：经济制裁敌国→收买武将→商业吞并
5. **丝路之王**（回合121+）：控制全部贸易→经济统一西域

**结局分支**：商王结局（经济统一）/ 暗影之王结局（经济+谍报）

> **线路切换规则**：线路A/B/C在游戏前期（回合36前）可自由切换一次。之后锁定当前线路直到通关。城邦特殊剧情线独立于初始线路，始终可触发。

#### 自给自足核心公式

```
月度资源平衡 = 基础产出 + 部落贡赋 + 贸易购买 - 人口消耗 - 军队消耗 - 建设消耗

各资源自给优先级：
  1. 金币 = 税收 + 特产出售 + 部落金币贡赋（核心，用于购买一切缺口）
  2. 粮食 = 农业 + 部落粮贡赋 + 购买（人口消耗最大项）
  3. 铁矿 = 矿场 + 购买（兵器锻造基础）
  4. 木材 = 伐木 + 购买（建设基础，全女性国家必须购买）
  5. 石材 = 采石 + 购买（建筑维护）
  6. 丝绸 = 纺织 + 购买（高价值贸易品/外交品）
  7. 兵器 = 锻造 + 购买（征兵装备）
  8. 马匹 = 马场 + 部落贡赋 + 购买（骑兵/商队）
  9. 奴隶 = 市场购买 + 战俘（劳动力/武将来源）
  10.盐/皮毛/药草/美酒/骆驼/香料/玉石 = 特产产出 + 购买（贸易/外交品）

自洽判定标准：
  ✅ 完全自给：产出≥消耗
  ✅ 贸易自给：产出<消耗，但金币盈余足以购买缺口
  ❌ 无法自给：产出<消耗，且金币不足以购买缺口（设计中不存在此情况）
```

> **结论**：所有39国（含3个全女性国家）的全部资源均可实现自给自足。核心机制是"特产出售→金币→购买缺口资源"三角模型。全女性国家虽然木材完全依赖购买、铁矿/兵器不足，但高价值特产（天衣丝绸/药草/战争收益）的金币收入足以覆盖所有缺口。没有任何国家存在无法自洽的资源。

## 附录B：距离价格浮动系统

> **核心规则**：所有奴隶贸易、商品贸易的价格，会随着与对应城邦/市场的距离远近进行上浮。最近的价格上浮0%（即基础价），最远的价格上浮30%。

### 距离价格公式

```
实际价格 = 基础价格 × (1 + 距离浮动率)

距离浮动率 = MIN(30%, 距离 / 最大地图距离 × 30%)

距离计算：两点间曼哈顿距离 = |x1-x2| + |y1-y2|
最大地图距离 = 100（地图对角线距离）
```

### 距离浮动率速查表

| 距离 | 浮动率 | 示例（劳奴50金） | 示例（丝绸100金） |
|------|--------|------------------|-------------------|
| 0-10 | 0% | 50金 | 100金 |
| 11-20 | 3% | 51.5金 | 103金 |
| 21-30 | 6% | 53金 | 106金 |
| 31-40 | 9% | 54.5金 | 109金 |
| 41-50 | 12% | 56金 | 112金 |
| 51-60 | 15% | 57.5金 | 115金 |
| 61-70 | 18% | 59金 | 118金 |
| 71-80 | 21% | 60.5金 | 121金 |
| 81-90 | 25% | 62.5金 | 125金 |
| 91-100 | 30% | 65金 | 130金 |

### 距离浮动对各国的影响

| 国家类型 | 平均购买距离 | 平均浮动 | 策略建议 |
|----------|-------------|----------|----------|
| 大国（中心位置） | 20-35 | 6-10% | 位置优越，贸易成本低 |
| 大国（边缘位置） | 35-55 | 10-16% | 需发展商路降低成本 |
| 中等国家 | 25-45 | 7-13% | 依赖邻近市场 |
| 小国 | 30-50 | 9-15% | 优先发展本地特产 |
| 全女性国家 | 40-65 | 12-20% | 贸易成本高，需发展高价值特产 |

> **商路减免**：建立商路后，对应方向的贸易距离浮动率-5%/商路等级。最高可减免15%（3级商路）。

## 附录C：六大奴隶市场

> **奴隶基础价格**：劳奴50金、匠奴80金、战奴100金、女战奴110金、舞女120金、侍妾150金。不同市场有品类和价格差异。**所有买卖价格均需叠加距离浮动（附录B）**。

### 买入规则

```
买入价格 = 基础价格 × (1 + 距离浮动率) × 买入折扣

买入折扣：特色品类享折扣（见下表"买入折扣"列），非特色品类为×1.0（无折扣也无加价）
买入溢价：部分市场对非主营品类有买入溢价（见下表"买入溢价"列），如战奴+20%即×1.2
```

### 卖出规则

```
卖出价格 = 基础价格 × (1 - 距离浮动率) × 卖出价格比例

卖出价格比例：
  通用比例：55%（所有市场通用，即卖出价为市价的55%）
  特色溢价：部分市场对特定品类有更高回收比例（见下表"卖出溢价"列）
  
距离浮动对卖家的影响：距离越远，运输成本越高，到手价格越低。这确保了远程倒卖无利可图。
```

| 市场 | 位置 | 买入折扣 | 买入溢价 | 卖出溢价 | 女性比例 | 月收购上限 |
|------|------|---------|---------|---------|----------|----------|
| **疏勒奴隶大市** | (16,34) | 匠奴-20%、舞女-15% | 战奴+20% | 通用55%、匠奴65% | 35% | 200人 |
| **粟特商人奴市** | (8,30) | 匠奴-15%、侍妾-20% | 战奴+25% | 通用55%、侍妾65% | 40% | 150人 |
| **康居战奴市场** | (8,14) | 战奴-30%、劳奴-25% | 舞女+30%、侍妾+35% | 通用55%、战奴70%、劳奴65% | 15% | 300人 |
| **阿玛宗女奴市场** | (75,25) | 女劳奴-20%、舞女-25%、侍妾-15% | 不卖男性 | 通用55%、舞女65%、女战奴60% | 100%女 | 100人 |
| **于阗女奴市场** | (25,59) | 女劳奴-20%、舞女-15%、侍妾-10% | 战奴+20% | 通用55%、女匠奴60%、舞女60% | 65% | 120人 |

> **买入示例**：从楼兰(52,46)到疏勒奴隶大市(16,34)购买匠奴，距离48→距离浮动14.4%，买入折扣-20%，实际价格=80×(1-20%)×(1+14.4%)=80×0.8×1.144≈73金。
>
> **卖出示例**：在楼兰向疏勒奴隶大市出售匠奴，同样距离48→距离浮动14.4%，卖出比例65%（匠奴特色溢价），实际到手=80×(1-14.4%)×65%=80×0.856×0.65≈45金。买入73金卖出45金，差价28金（38%），且受月收购上限限制（疏勒月收匠奴上限200人），无法批量倾销套利。
>
> **价格均衡设计**：卖出价始终低于买入价（最高卖出比例70% < 最低买入折扣后比例70%），且距离浮动对卖出方更不利（卖时减去距离浮动，买时加上），配合月收购上限，彻底杜绝跨市场倒卖套利空间。
