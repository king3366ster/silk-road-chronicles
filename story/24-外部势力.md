## 一、外部势力概述

外部势力是游戏**阶段七**正式介入的超国家/帝国级势力。当玩家的势力达到「所占领/同盟/附庸的国家数 ≥ 36个」时，阶段七触发，外部势力正式登场。

外部势力的介入是游戏后期的核心变数，打乱西域原有的势力平衡，迫使玩家以势力领袖身份对抗或者联合外部势力完成最终的主线故事线。

## 二、主要外部势力

### 2.1 汉朝（大汉帝国）

| 属性 | 说明 |
|------|------|
| **类型** | 大一统中央帝国 |
| **政治制度** | 君主专政制（皇帝独裁） |
| **经济制度** | 国家专营制（盐铁专营） |
| **军事制度** | 常备军制（中央军+边军） |
| **介入动机** | 维护丝路安全、扩张势力范围、纳入朝贡体系 |
| **西域策略** | 以夷制夷：扶持亲汉势力、册封西域都护、构建朝贡体系 |
| **威胁等级** | ⭐⭐⭐⭐⭐（高） |
| **相性制度** | 倾向于与普通国合作；对全女国态度中立但不承认合法性；与母系国可维持表面外交关系 |

#### 汉朝介入方式

| 方式 | 触发条件 | 效果 |
|------|----------|------|
| **使者来访** | 阶段七初期 | 要求玩家势力纳入汉朝朝贡体系 |
| **军事驻扎** | 玩家拒绝朝贡 | 在边境重镇屯驻边军（军力评判） |
| **西域都护任命** | 汉朝影响力≥60 | 设立西域都护府，直接在玩家势力范围内设立行政机构 |
| **联军征讨** | 玩家敌对行为严重 | 汉朝联合其他西域国家组成联军讨伐 |
| **强制改制** | 玩家战败/受降 | 强制玩家势力改用君主专政制（普通国），纳入朝贡体系 |

### 2.2 匈奴（匈奴汗国）

| 属性 | 说明 |
|------|------|
| **类型** | 草原游牧帝国 |
| **政治制度** | 部落酋长制（单于集权+部落联盟） |
| **经济制度** | 游牧畜牧制 |
| **军事制度** | 部落兵制（全民皆兵，骑兵为主） |
| **介入动机** | 劫掠财富、争夺丝路控制权、削弱汉朝影响力 |
| **西域策略** | 武力威吓、强迫臣服、征收贡赋 |
| **威胁等级** | ⭐⭐⭐⭐⭐（极高，机动作战能力强） |
| **相性制度** | 倾向于与部落制/游牧制势力结盟；与城邦制/中央集权制势力敌对 |

#### 匈奴介入方式

| 方式 | 触发条件 | 效果 |
|------|----------|------|
| **骑兵南下** | 阶段七初期 | 匈奴骑兵劫掠玩家势力边境城市 |
| **强迫臣服** | 玩家军力不足 | 匈奴使者要求玩家缴纳贡赋、称臣纳贡 |
| **结盟邀请** | 玩家为部落酋长制/游牧制 | 邀请玩家共同对抗汉朝 |
| **大军压境** | 玩家拒绝投降/结盟 | 匈奴主力骑兵入侵（战力极高） |
| **扶持傀儡** | 玩家战败 | 在玩家势力内扶持亲匈奴政权 |

### 2.3 贵霜帝国

| 属性 | 说明 |
|------|------|
| **类型** | 中亚帝国 |
| **政治制度** | 君主专政制（皇帝统治） |
| **经济制度** | 商路贸易制 |
| **军事制度** | 常备军制+雇佣兵制混合 |
| **介入动机** | 扩张商路利益、争夺丝路南道 |
| **西域策略** | 经贸渗透、文化传播（佛教）、军事威慑 |
| **威胁等级** | ⭐⭐⭐⭐（中等偏高） |
| **相性制度** | 倾向于与城邦市政制/商路贸易制势力合作；传播佛教支持宗教领袖制 |

#### 贵霜介入方式

| 方式 | 触发条件 | 效果 |
|------|----------|------|
| **商队武装** | 阶段七初期 | 贵霜商队全副武装通过玩家领地（试探态度） |
| **经贸协定** | 玩家以商路贸易制为主 | 提供优惠贸易协定、技术转让 |
| **佛教传播** | 玩家未明确拒绝 | 佛教文化大规模传入，影响政治/经济制度选择 |
| **军事干预** | 玩家威胁贵霜商路利益 | 贵霜精锐部队介入西域战争 |

### 2.4 帕提亚帝国（安息）

| 属性 | 说明 |
|------|------|
| **类型** | 中亚西亚帝国 |
| **政治制度** | 贵族议政制（国王+贵族议会） |
| **经济制度** | 商路贸易制+庄园贡赋制混合 |
| **军事制度** | 贵族私兵制（贵族骑兵为主） |
| **介入动机** | 控制丝绸之路西端、对抗罗马/贵霜在东方的扩张 |
| **西域策略** | 外交纵横、经济封锁/开放、军事代理人战争 |
| **威胁等级** | ⭐⭐⭐⭐（中等偏高） |
| **相性制度** | 倾向于与贵族议政制/庄园贡赋制势力合作 |

#### 帕提亚介入方式

| 方式 | 触发条件 | 效果 |
|------|----------|------|
| **外交使节** | 阶段七初期 | 帕提亚使者提议建立反汉朝/反匈奴联盟 |
| **军火供应** | 与帕提亚结盟 | 提供帕提亚骑兵装备和战术 |
| **代理人战争** | 帕提亚不愿直接下场 | 提供资金和武器支持玩家与汉朝/匈奴对抗 |
| **封锁商路** | 玩家拒绝与帕提亚合作 | 封锁丝绸之路西段，切断玩家贸易收入 |

## 三、外部势力介入阶段与触发条件

### 3.1 阶段七触发条件

| 条件 | 说明 |
|:----:|------|
| **核心触发** | 玩家势力所占领/同盟/附庸的国家数 ≥ 36个 |
| **下限** | 西域五十国，36个已基本统治西域 |
| **意义** | 玩家的势力已经足够强大，引起了三大帝国的关注和警惕 |

### 3.2 外部势力介入的时间线

| 阶段 | 外部势力状态 | 主要事件 |
|:----:|-------------|----------|
| **阶段一~四** | 未介入 | 外部势力仅作为背景存在（在[12-主线剧情.md](12-主线剧情.md)中提及） |
| **阶段五** | 开始关注 | 玩家的扩张引起周边帝国注意，可能有使者来访但无实质干预 |
| **阶段六** | 外交试探 | 汉朝/匈奴开始派遣使者建立联系；贵霜/帕提亚试探性接触 |
| **阶段七** | **正式介入** | 触发条件达成后，所有外部势力开始采取实际行动 |

### 3.3 各阶段外部势力影响

| 游戏阶段 | 外部势力影响 | 玩家反应 |
|----------|-------------|----------|
| **阶段一** | 无 | 不受影响 |
| **阶段二** | 无 | 不受影响 |
| **阶段三** | 偶有边境冲突提及（事件文本） | 可无视 |
| **阶段四** | 可选择派遣使者建立初步联系 | 可选主动接触 |
| **阶段五** | 外部势力的使者首次到访 | 可接待/拒绝 |
| **阶段六** | 势力已成气候，外部势力开始认真接触 | 需选择外交立场 |
| **阶段七** | **全面介入**，决定西域最终局势 | 以势力领袖身份应对外部势力 |

## 四、外部势力介入事件链

### 4.1 阶段七引导事件：三大帝国来朝

当玩家势力达到36个国家时，触发引导事件：

> **【三大帝国来朝】**
>
> 西域五十国中已有三十六国归附于你。消息传至长安、龙城、尼雅……三大帝国的使节几乎同时抵达你的宫殿。
>
> 汉朝使者宣读圣旨：「西域都护府将正式设立，尔等须向天子称臣，岁岁来朝。」
>
> 匈奴单于的使者则带来草原之王的战书：「若降，可保荣华富贵；若抗，草原铁骑必将踏平西域。」
>
> 贵霜使团则带着佛经与金币：「伟大的王啊，不如与贵霜结盟，共谋丝路繁华……」

| 玩家选择 | 后果 |
|----------|------|
| **向汉朝称臣** | 势力变为汉朝藩属，需缴纳贡赋（收入-15%），获得汉朝保护（汉军可协防）。政治制度被强制限定为君主专政制（普通国版）。 |
| **与匈奴结盟** | 与匈奴建立同盟，获得骑兵支援（战力+20%），但需共同对抗汉朝（战争风险+50%）。政治制度被强制限定为部落酋长制。 |
| **与贵霜结盟** | 获得贵霜贸易和技术支持（收入+30%，手工业产出+20%），但不直接参与对汉/匈战争。政治制度不受强制改变。 |
| **拒绝所有帝国，独立称雄** | 同时将汉朝、匈奴、贵霜、帕提亚全部变为敌对势力。四方联军可能联合讨伐。但若能抵御成功，最终实现真正的「君临天下」。 |

### 4.2 后续事件链

| 事件 | 触发条件 | 描述 |
|------|----------|------|
| **汉朝讨伐** | 拒绝称臣/与匈奴结盟 | 汉朝派兵西征，西域都护率汉军攻打玩家势力 |
| **匈奴入侵** | 拒绝结盟/向汉朝称臣 | 匈奴骑兵大规模南下劫掠 |
| **贵霜调停** | 汉匈同时敌对 | 贵霜提出调停方案，玩家可选择接受中立方案 |
| **帕提亚商机** | 丝路贸易受阻 | 帕提亚暗中提供替代商路 |
| **四方混战** | 玩家独立称雄 | 汉、匈、贵霜、帕提亚四国联军与玩家势力全面战争 |
| **最终决战** | 击败所有外部势力/或与某一势力结盟完成 | 决定西域最终归属 |

### 4.3 外部势力强制改制事件

外部势力介入后，可以对玩家势力强制改变政治/经济/军事制度：

| 强制方 | 强制方向 | 要求 | 拒绝后果 |
|--------|----------|------|----------|
| **汉朝** | 政治→君主专政制（普通国） | 玩家势力必须改为普通国君主专政制，纳入朝贡体系 | 汉朝全面入侵 |
| **匈奴** | 政治→部落酋长制<br>经济→游牧畜牧制<br>军事→部落兵制 | 玩家势力必须改为草原体制 | 匈奴骑兵铲平城池 |
| **贵霜** | 经济→商路贸易制<br>（政治可选佛教→宗教领袖制） | 开放商路、接受佛教传播 | 经济封锁+文化渗透+军事威胁 |
| **帕提亚** | 经济→商路贸易制<br>政治→贵族议政制（可选） | 建立反汉/反匈联盟 | 商路西段封锁 |

> 注：当玩家已经被某一势力强制改制后，其他势力的强制要求可能无法同时满足，会触发势力间的冲突。

## 五、外部势力与各条路线的关系

| 主线路线 | 与外部势力的最佳关系 | 策略说明 |
|----------|---------------------|----------|
| **A·君临天下** | 独立对抗所有外部势力 | 最终决战击败四方联军，统一西域对抗三大帝国 |
| **B·富甲丝路** | 与贵霜/帕提亚合作 | 保持中立利用各方商路，以金币换安全 |
| **C·万国来朝** | 名义上依附汉朝 | 以朝贡换取汉朝支持，文化影响力最大化 |
| **D·凤鸣昆仑** | 灵活应对 | 母系/全女国在外部势力中寻找可接受自身制度的合作方（贵霜、帕提亚较宽松） |
| **E·纵横捭阖** | 在各方之间游走 | 利用汉匈矛盾、贵霜帕提亚竞争，最大化自身利益 |
| **F·天朝藩屏** | 依附汉朝 | 最理想的藩屏路线，获得汉朝最大程度保护 |

## 六、外部势力影响力系统

### 6.1 影响力数值

| 外部势力 | 初始影响力 | 阶段七起始值 | 影响力增长方式 |
|----------|:----------:|:------------:|---------------|
| **汉朝** | 30 | 50 | 玩家称臣+5/月；拒绝-2/月；贸易+1/月 |
| **匈奴** | 20 | 40 | 玩家结盟+5/月；拒绝-3/月；边境安定-2/月 |
| **贵霜** | 15 | 30 | 贸易协定+3/月；佛教传播+2/月；冲突-4/月 |
| **帕提亚** | 10 | 25 | 商路合作+2/月；反汉协议+3/月；敌对-5/月 |

### 6.2 影响力决定的事件

| 影响力阈值 | 触发事件 |
|:----------:|----------|
| ≥80 | 全面控制：该势力实际支配玩家势力 |
| ≥60 | 深度渗透：势力可要求政治/经济/军事改制 |
| ≥40 | 正式接触：使节常驻、贸易通道建立 |
| ≤20 | 冷淡关系：无交流，相互戒备 |
| ≤10 | 敌对状态：边境冲突频繁 |
