### 1.1 🌸 凤鸣觉醒 — 普通国→母系国

#### 1.1.1 事件概述

| 维度 | 说明 |
|------|------|
| **事件名称** | 凤鸣觉醒（The Phoenix's Awakening） |
| **转型方向** | 普通国家 → 母系国家 |
| **本质** | 女性解放运动，社会性别权力结构的根本性重组 |
| **周期** | 6~12个月（分三个阶段完成） |
| **可逆性** | 可逆（可通过父权复辟转回） |

#### 1.1.2 触发原因

| 原因类型 | 具体原因 | 权重 |
|----------|----------|:----:|
| **人口结构变化** | 女性公民比例≥65%（接近母系国70%标准），女性在经济生产中的贡献已超过男性 | 核心条件 |
| **经济发展驱动** | 女性在纺织、酿造、贸易等领域积累了大量财富，经济独立带来政治诉求 | 催化条件 |
| **外部思潮影响** | 与母系国或全女国的接触→女性知识分子接触到"女性治国的可能性" | 催化条件 |
| **男性统治危机** | 连续战败/外交失败/经济衰退→男性统治的合法性受到质疑 | 催化条件 |
| **文化觉醒** | 女诗人、女学者、女祭司掀起"找回女性传统"的文化运动 | 辅助条件 |

#### 1.1.3 触发条件

| 条件 | 要求 | 说明 |
|------|------|------|
| **人口条件** | 女性公民比例 ≥ 65% | 核心条件，不可绕过 |
| **稳定度** | ≥ 50 | 社会有一定秩序，非战乱状态 |
| **经济条件** | 国库 ≥ 2,000 金 | 转型需要财政支持 |
| **文化条件** | 有 ≥ 1 位魅力 ≥ 70 的女性角色（文官/武将/贵族） | 需要有女性领袖人物 |
| **时间条件** | 游戏进入第二阶段后（回合 ≥ 37） | 早期不足以积累社会变革条件 |
| **外部条件（可选）** | 与 ≥ 1 个母系/全女国建立外交关系 | 缩短触发所需的冷却期 |

#### 1.1.4 触发机制

```
触发判定流程：

[每个月判定一次]
    ↓
女性公民比例 ≥ 65%？
    ├── 否 → 不触发，积累"觉醒进度"+5/月
    │         （觉醒进度 > 100 时自动触发）
    │
    └── 是 → 觉醒进度 ≥ 100？
                ├── 否 → 觉醒进度 + 15/月
                └── 是 → 🌸 凤鸣觉醒触发！
                          ↓
                    事件弹出：玩家选择
                    ├── ① 接受觉醒（进入转型流程）
                    └── ② 压制觉醒（稳定度-20，觉醒进度归零，
                               进入24回合冷却，此后第1次触发
                               概率翻倍）
```

**觉醒进度说明：**

| 条件满足度 | 每月觉醒进度增加 | 说明 |
|-----------|:--------------:|------|
| 仅满足人口条件 | +5 | 缓慢积累 |
| 人口条件 + 1个催化条件 | +10 | 中度加速 |
| 人口条件 + 2个催化条件 | +15 | 快速积累 |
| 全部条件满足 | +25 | 爆发式增长 |
| 有女性魅力≥85的领袖 | +5（额外） | 领袖效应 |

#### 1.1.5 转型流程（三个阶段）

##### 第一阶段：萌芽期（1-3个月）

| 事件 | 效果 |
|------|------|
| 女性知识分子秘密集会 | 稳定度-2/月（秘密活动引发猜疑） |
| 女性商人开始联合抵制男性商人的排挤 | 商业收入-10%短期 |
| 女诗人/女学者创作"觉醒文学" | 文化影响力+5 |

**剧情描写：**

> 在疏勒城西市的一间昏暗的茶馆后室里，七个女人围坐在一张矮桌旁。她们中有富商**马三娘**、有学识渊博的女学者**苏婉儿**、有疏勒王宫的女医官**莲心**，还有几个面戴纱巾的普通女市民。
>
> 苏婉儿展开一卷帛书，上面是她抄录的——来自遥远瑶池国的《母系治国论》。"姐妹们，"她的声音低沉而坚定，"瑶池国——一个由女人治理的国家。女人当国王，女人做将军，女人管赋税……她们的国家比我们任何邻国都要繁荣。"
>
> 马三娘冷笑一声："繁荣有什么用？在这疏勒城，我们女人的财产可以被丈夫随意处置，我们的女儿只能嫁给父亲指定的人——我们辛辛苦苦赚的钱，最后全进了男人的口袋。"
>
> 茶馆外传来巡逻兵的脚步声——她们迅速收起了帛书，装作在喝茶聊天。但一种新的思想已经在她们心中生根——一种连她们自己都不敢完全承认的思想。

##### 第二阶段：公开化（4-6个月）

| 事件 | 效果 |
|------|------|

**剧情描写：**

> 三个月后的早晨，疏勒王宫门前聚集了上千名女性。站在最前面的是马三娘——她手持一卷长长的帛书，上面密密麻麻写满了名字和手印。
>
> 疏勒王**安国**在卫兵的护卫下出现在宫门城楼上。他低头看着那些女人——她们中有白发苍苍的老妪，有怀抱婴儿的少妇，有面如冰霜的女商人，还有几个他认识的女官。
>
> "陛下，"马三娘的声音穿透了清晨的薄雾，"我们——疏勒城一万两千名女性——请求陛下接见。我们有一份请愿书要呈上。"
>
> 安国沉默了很久。他身边的宰相低声说："陛下，按祖宗规矩，女人不可干政。您若是见了她们，就是在鼓励更多反叛。"
>
> 但安国的目光落在了一位老妇身上——她是他儿时的乳母。她也在人群中，满脸皱纹的眼睛里含着泪水——但不是恐惧的泪水，而是失望。

##### 第三阶段：转型期（7-12个月）——关键抉择

| 选项 | 后果 |
|------|------|
| **选项一：和平过渡（推荐）** | ① 召开"朝野对话"，国王与女性代表共同起草《女权宪章》；② 国王保留名义权力，实权逐步过渡给女性议会；③ 男性公民可保留全部权利，但统治权归女性；④ 稳定度→100；⑤ 转型耗时12个月 |
| **选项二：激进革命** | ① 女性发动大规模起义，推翻旧政权；② 旧王室被驱逐/处决；③ 女性议会掌权；④ 男性公民权利受限（政治权利-30%）；⑤ 稳定度→60（短期）；⑥ 转型耗时6个月（更快但更痛苦）；⑦ 与旧盟友外交关系恶化（好感-20）；⑧ 可能遭遇邻国干涉 |
| **选项三：镇压革命** | ① 国王派兵镇压女性运动；② 女性领袖被捕/处决；③ 稳定度→50（短期恢复）；④ 女性满意度-30；⑤ 觉醒进入24个月冷却期；⑥ 觉醒进度下次触发时双倍速度；⑦ 文化影响力-20 |

**剧情描写（和平过渡）：**

> 安国最终选择了对话。他走下城楼，亲手打开了宫门。在朝堂之上，马三娘和她的姐妹们呈上了《疏勒女权十二条》——内容从"女子有受教育之权"到"女子可参与市政议事"。
>
> 保守派大臣怒不可遏："陛下！这是要亡国啊！女人管政事，那还是疏勒吗？"
>
> 安国看着那份请愿书，缓缓说了一句让所有人都震惊的话："疏勒是什么？疏勒不是一座城、一面旗、一个姓氏。疏勒是住在这里的人。如果一半的人说她们被忽视了——那这城就不是完整的疏勒。"
>
> 当天，安国宣布成立"改制委员会"——由六名大臣和六名女性代表共同组成，负责在一年内完成制度转型。消息传出，整个西域为之震惊——疏勒，一个百年男权城邦，正在变成母系国家。

**剧情描写（激进革命）：**

> 安国选择了镇压。骑兵冲散了人群，马三娘被捕，苏婉儿在逃亡中失踪。但这场镇压激起了更大的反弹——三个月后，一群蒙面的女骑兵在深夜突袭了王宫。她们是苏婉儿秘密训练的女兵——虽然只有两百人，但她们的决心超越了人数。
>
> 当黎明照进王宫时，安国被围在寝殿中。女兵首领——苏婉儿的妹妹**苏玉**——掀开了面纱："陛下，我姐姐在逃跑时摔断了腿，现在还被关在你的地牢里。打开牢门，释放所有女囚——我们可以谈。否则，今天就是你这王朝的最后一天。"
>
> 安国看着苏玉手中的剑——那剑上还滴着卫兵的血。他知道——旧的秩序已经终结了。

### 1.2 🔥 女儿当国 — 母系国→全女国

#### 1.2.1 事件概述

| 维度 | 说明 |
|------|------|
| **事件名称** | 女儿当国（Daughters Rule the Nation） |
| **转型方向** | 母系国家 → 全女性国家 |
| **本质** | 极端女权运动，彻底排除男性的政治和社会参与 |
| **周期** | 9~18个月（分四阶段完成） |
| **可逆性** | **不可逆**（全女国制度不可直接转回母系国，只能通过凤归母巢） |

#### 1.2.2 触发原因

| 原因类型 | 具体原因 | 权重 |
|----------|----------|:----:|
| **男性威胁感知** | 邻国父权复辟、男性叛乱、外国渗透等事件让女性统治者认为"男性不可信任" | 核心原因 |
| **人口纯化动力** | 男性公民比例持续下降至≤15%，社会基础已接近全女 | 核心条件 |
| **理想主义运动** | 年轻一代女贵族/女兵认为"既然女性可以治国，为什么还需要男性？" | 催化条件 |
| **安全焦虑** | 男性可能成为"第五纵队"（与外部父权势力勾结），全女化是安全选择 | 催化条件 |
| **文化激进** | 女祭司/女神职者宣扬"男性是罪恶之源"的教义 | 辅助条件 |

#### 1.2.3 触发条件

| 条件 | 要求 | 说明 |
|------|------|------|
| **人口条件** | 男性公民比例 ≤ 15% | 核心条件 |
| **稳定度** | ≥ 50 | 社会有序，运动可和平进行 |

| **军事条件** | 女性武将 ≥ 2 员 或 女兵占比 ≥ 30% | 需要武力保障转型 |
| **外部条件** | 与 ≥ 1 个全女国家有外交关系 | 有先例可循 |
| **触发阶段** | 游戏进入第三阶段后（回合 ≥ 61） | 母系国需足够发展时间 |

> "能用——但要打八折。"
>
> 疏勒的商业税官**赵衡**站在一边，看着这种混乱的交易场面直摇头。他回到王宫，向疏勒王提交了一份长篇报告——《论西域统一币制之必要》。报告中写道：

### 5.6 🌙 月下之盟 — 母系国家的联姻网络

#### 事件概述

| 维度 | 说明 |
|------|------|
| **触发对象** | 母系国家（与普通国有外交关系时触发） |
| **触发原因** | 母系国试图通过"女娶男嫁"的联姻方式扩大外交影响力 |
| **触发条件** | ①母系国且②与≥2个普通国好感≥50且③有适龄的公主/女贵族（颜值≥70） |
| **本质** | 通过与传统男权文化的联姻实现政治渗透 |

#### 选项与后果

| 选项 | 效果 |
|------|------|
| **公主外嫁（联姻）**（2,000金+6个月） | 母系国公主嫁给普通国国王/王子；联姻国关系+30；但公主的"母系思想"可能影响联姻国的政治（母系文化影响力+10/年） |

**剧情描写：**

> 瑶池国的女王**瑶姬**有一个十九岁的女儿——**明月公主**——不仅容貌出众，而且精通六国语言、熟读兵书、擅长骑射。这样一位才貌双全的公主，自然成了西域各国王室求婚的热门对象。

>
> 瑶姬看着女儿眼中闪烁的光芒——那不是新婚的羞涩，而是战士的野心。
