# 西域群英传 — 剧情·任务线

> **文档版本**：v2.0 | **最后更新**：2026年5月
> **所属游戏**：西域群英传 - Silk Road Chronicles（RPG + 策略战棋）

## 一、主线剧情结构

### 1.1 故事背景

西汉末年至东汉初年，丝绸之路刚刚打通，西域39国林立，汉匈争霸的余波未平。玩家扮演一个无名小卒，从一穷二白开始，通过政、商、军三条路线逐步崛起，最终统一西域诸国。

### 1.2 五阶段主线进程

#### 阶段一：无名小卒（回合1-12）

```
剧情梗概：
  玩家抵达城邦→通过考验→获得管理权→经营小领地

关键节点：
  1. 抵达城邦：选择起始国家
  2. 通过考验：完成城邦基础任务（进贡100金）
  3. 获得管理权：初始公民10人+初始奴隶10人
  4. 经营小领地：学习资源采集、人口管理

解锁内容：城邦管理、资源采集、人口管理
```

#### 阶段二：部落征途（回合13-36）

```
剧情梗概：
  完成周边部落任务→解锁能力→遭遇全女性国家

关键节点：
  1. 部落任务1：解锁奴隶买卖/资源采集/农业/马场
  2. 部落任务2：解锁奴隶劳作/商业/税收/锻造
  3. 部落任务3：解锁奴隶培训/高级能力
  4. 遭遇全女性国家：首次接触特殊文明

解锁内容：奴隶系统、农业、商业、马场、税收、锻造
条件：完成1-3个部落任务，每个需要金币+粮食+靠近部落
```

#### 阶段三：城市发展（回合37-72）

```
剧情梗概：
  城市升级→外交征兵→组建正规军→开辟商路

关键节点：
  1. 城市Lv2：解锁外交+征兵+科技
  2. 城市Lv3：解锁宣战+商路+谍报
  3. 组建正规军：首次拥有正规军事力量
  4. 开辟商路：建立远程贸易网络

解锁内容：外交系统、征兵系统、科技研发、宣战、商路、谍报
条件：完成至少3个部落任务
```

#### 阶段四：扩张征伐（回合73+）

```
剧情梗概：
  宣战→攻城→吞并→附庸→逐步统一

关键节点：
  1. 首次宣战：正式进入战争状态
  2. 攻城略地：占领敌方城邦
  3. 吞并/附庸：选择直接统治或间接控制
  4. 多线作战：同时应对多个敌人

条件：城市等级3+，完成目标国50%以上部落任务
```

#### 阶段五：统一西域

```
剧情梗概：
  征服或外交统一所有39国

胜利条件（与五大主线一一对应）：
  - ⚔️ 军事征服（线路A）：征服所有39个城邦（直接统治或附庸）→ 西域霸王
  - 💰 经济霸权（线路B）：控制丝路80%贸易额 + 与≥20国商业同盟 → 丝路之主
  - 📜 文化同化（线路C）：文化影响力覆盖≥30国 + 联姻/联盟统一 → 西域圣者
  - 🌸 母系联盟（线路D）：统一所有母系/全女性国家（9国）+ 征服其余 → 西域女帝
  - 🏛️ 帝国平衡（线路E）：六大帝国全部友好 + 39国联合体系 → 西域之王
```

## 二、五大主线剧情路线

> **六大帝国贯穿设计**：每条主线都受到不同帝国势力的深层影响，帝国的博弈关系贯穿始终，玩家的每一个重大决策都会影响与六大帝国的关系。

### 2.1 线路A：军事征服线 ⚔️

**主题**：以武力统一西域，建立军事帝国
**触发**：选择好战型国家（乌孙、康居、阿玛宗等）开局时激活
**帝国影响**：**汉朝🐉 vs 匈奴🐺** 的丝路控制权争夺贯穿全线。玩家必须在两大死敌之间选边站，或利用双方矛盾渔利。

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权
     【帝国事件】匈奴骑兵首次骚扰边境，汉朝使者首次到访
  2. 部落征途（回合13-36）：征服周边部落→建立军事基础→遭遇强敌
     【帝国事件】匈奴要求进贡或结盟；汉朝提供初级军事技术
     【关键抉择】投靠匈奴（获得骑兵加成但失去汉朝好感）vs 效忠汉朝（获得步兵技术但匈奴敌意增加）
  3. 铁血征途（回合37-72）：组建正规军→首次攻城→吞并邻国
     【帝国事件】汉朝都护府兴衰影响军事格局；匈奴南北分裂带来机遇
     【关键抉择】利用匈奴内乱北伐 vs 趁汉朝衰弱独立扩张
  4. 西域霸主（回合73-120）：多线作战→击溃联军→称霸西域
     【帝国事件】大月氏部落联盟从西北施压；羌人从南部骚扰
     【关键抉择】先北后南（先击匈奴再平羌人）vs 先南后北
  5. 一统天下（回合121+）：最终决战→统一西域→建立帝国
     【帝国事件】汉朝派遣远征军或承认你的霸权；匈奴最终决战
     【结局】军事征服胜利→西域霸王
```

### 2.2 线路B：商业帝国线 💰

**主题**：以商业力量控制西域，建立丝路帝国
**触发**：选择粟特/疏勒等商贸国家开局时激活；或在其他线路中商业声望达到"传奇"后可切换
**帝国影响**：**贵霜🏛️ vs 安息🦁** 的贸易垄断争夺贯穿全线。玩家在两大贸易帝国之间周旋，争夺丝路贸易主导权。

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权
     【帝国事件】首次接触贵霜商队和安息货物
  2. 丝路开拓（回合13-36）：开辟新商路→建立贸易站→遭遇商路匪患
     【帝国事件】贵霜开放西部市场；安息设置高额关税壁垒
     【关键抉择】与贵霜合作（获得稀有商品但安息敌意增加）vs 独立开拓（效率低但不受制于人）
  3. 商业帝国（回合37-72）：控制关键商路→建立金融网络→商业外交
     【帝国事件】安息试图垄断中转贸易；贵霜推广佛教换取商业特权
     【关键抉择】打破安息垄断（风险高但收益大）vs 与安息分利（稳定但受限）
  4. 经济霸权（回合73-120）：经济制裁敌国→收买武将→商业吞并
     【帝国事件】汉朝丝路需求激增；大月氏提供北方商路通道
     【关键抉择】垄断东向贸易（与汉朝合作）vs 垄断西向贸易（与贵霜/安息博弈）
  5. 丝路之主（回合121+）：垄断丝路贸易→经济统一西域
     【帝国事件】六大帝国都试图分一杯羹，玩家以经济力量制衡各方
     【结局】经济霸权胜利→丝路之主
```

### 2.3 线路C：外交文化线 📜

**主题**：以文化和外交手段统一西域
**触发**：选择龟兹、于阗、西梁女国等文教国家开局时激活
**帝国影响**：**大月氏🌙 + 羌人⛰️** 作为次级势力在局部区域施加影响。玩家通过文化融合和外交手段化解帝国压力。

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权
     【帝国事件】大月氏部落使者到访；羌人山民首次接触
  2. 文化传播（回合13-36）：建设文化建筑→传播文化→结交盟友
     【帝国事件】贵霜佛教传入带来文化冲击；大月氏血缘部落寻求联盟
     【关键抉择】接受佛教（文化+但传统派反感）vs 保持传统（稳定但文化发展慢）
  3. 外交纵横（回合37-72）：联姻结盟→调解纠纷→建立文化圈
     【帝国事件】羌人部落联盟提出联合抗汉；大月氏提议部落联姻
     【关键抉择】与大月氏联姻（西北部落联盟加成）vs 与羌人结盟（南部安全+药草资源）
  4. 文化霸权（回合73-120）：文化渗透→外交孤立敌国→和平演变
     【帝国事件】汉朝文化影响力扩大；匈奴文化在草原部落中衰退
     【关键抉择】借助汉朝文化影响力（快速扩张但依附汉朝）vs 独立发展西域文化（慢但自主）
  5. 万国来朝（回合121+）：文化影响力覆盖全西域→和平统一
     【帝国事件】六大帝国都认可你的文化成就，和平共处
     【结局】文化胜利→西域圣者
```

### 2.4 线路D：全女性国家特殊线 🌸

**主题**：全女性国家的独特崛起之路
**触发**：选择阿玛宗、西梁女国、东女国开局时激活
**帝国影响**：**羌人⛰️** 是全女性国家最重要的外部关系，昆仑山南麓的通道控制权决定全女性国家的命运。同时**汉朝🐉**对全女性国家持好奇与警惕并存的态度。

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权
     【帝国事件】汉朝使者首次到访，对全女性社会充满好奇
  2. 母系传承（回合13-36）：探索母系文化→建立女性治理体系→遭遇外部威胁
     【帝国事件】羌人部落要求通行费或结盟；匈奴探子试图渗透
     【关键抉择】与羌人结盟（保障南部安全）vs 封闭边境（独立但孤立）
     【资源挑战】重体力劳动产出降低，需通过贸易补足木材/铁矿
  3. 女性崛起（回合37-72）：发展特色经济→外交突破→军事自卫
     【帝国事件】贵霜商人高价收购天衣丝绸/药草；汉朝提出册封
     【关键抉择】接受汉朝册封（获得技术援助但失去独立性）vs 保持独立（自由但发展受限）
  4. 巾帼霸业（回合73-120）：打破偏见→建立联盟→争夺霸权
     【帝国事件】大月氏女王提议联盟；匈奴试图征服全女性国家
     【关键抉择】建立母系联盟（联合所有母系/全女性国家）vs 融入主流外交体系
  5. 女帝之路（回合121+）：统一西域→建立母系帝国
     【帝国事件】六大帝国见证女性力量的崛起
     【结局】全女性特殊胜利→女帝
```

### 2.5 线路E：帝国纵横线 🏛️

**主题**：在六大帝国之间纵横捭阖，以外交平衡术统一西域
**触发**：选择处于帝国交汇点的国家（疏勒、车师、楼兰）开局时激活；或在线路A/B/C中同时与≥3个帝国达到"友好"关系后可切换
**帝国影响**：**六大帝国全部参与**，玩家必须在所有帝国之间维持精妙的平衡，任何一方的过度强大都威胁西域的独立。

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权
     【帝国事件】同时接触汉朝使者和匈奴探子，初步了解帝国格局
  2. 平衡之术（回合13-36）：学习六大帝国的关系→选择初步立场→建立外交网络
     【帝国事件】汉朝与匈奴同时拉拢；贵霜与安息同时示好
     【关键抉择】在汉匈之间保持中立（两边获利但两边不信任）vs 选边站（一方信任但另一方敌对）
  3. 纵横捭阖（回合37-72）：利用帝国矛盾→挑拨离间→渔翁得利
     【帝国事件】安息与贵霜爆发贸易战；大月氏与匈奴结盟对抗汉朝
     【关键抉择】挑拨安息vs贵霜（延长贸易战从中获利）vs 调停和解（获得双方好感但失去短期利益）
  4. 帝国棋局（回合73-120）：成为帝国博弈的关键棋手→左右西域命运
     【帝国事件】汉朝远征军抵达；匈奴全面南侵；贵霜东扩
     【关键抉择】联合次要敌人对抗主要威胁（灵活联盟）vs 建立西域统一战线（所有39国联合抗外）
  5. 丝路共主（回合121+）：六大帝国都承认你的地位→建立多帝国平衡体系
     【帝国事件】六大帝国签署和平条约，西域成为中立缓冲区
     【结局】外交统一胜利→西域之王（最高称号：丝路传奇）
```

## 三、城邦专属剧情线

> **核心规则**：每个城邦拥有2-3条独立剧情线，每条线包含5个任务节点。完成剧情线可获得专属武将和永久加成。大国3条线，中等国2条线，小国2条线。

### 3.1 大国剧情线（9国，各3条线）

> 大国9个：楼兰、于阗、龟兹、乌孙、疏勒、粟特、康居、车师、西梁女国。每个大国拥有3条剧情线。

#### 楼兰 🏜️（3条线）

- **线1「绿洲明珠」**：绿洲保卫→水源争夺→沙漠治理→绿洲复兴→楼兰王
- **线2「丝路门户」**：驿站建设→商路开拓→丝路关卡→贸易帝国→丝路之王
- **线3「沙漠秘宝」**：古墓发现→沙漠探险→古物收集→宝藏揭秘→楼兰秘宝王
- **可获得武将**：驿站官·楼驿(♂·文官)、绿洲守·兰盾(♂·武将)、古墓学者·沙悟(♂·文官)(线3)

#### 龟兹 🎵（3条线，普通大国）

- **线1「乐舞之都」**：乐舞学习→乐舞大会→乐舞外交→文化霸权→龟兹乐王
- **线2「铁矿帝国」**：铁矿开发→兵器锻造→铁骑军团→军事强国→龟兹铁王
- **线3「佛教中心」**：佛寺建设→佛经翻译→佛教传播→文化输出→龟兹法王
- **可获得武将**：乐师·龟音(♀·文官)、铁匠·龟铁(♂·武将)、高僧·龟法(♂·文官)(线3)

#### 于阗 💎（3条线，母系大国·6个母系国之一）

- **线1「玉石之路」**：玉石开采→玉器加工→玉石贸易→玉石帝国→于阗玉王
- **线2「丝绸之国」**：蚕桑引进→丝绸纺织→丝绸贸易→丝绸帝国→于阗丝王
- **线3「母系王朝」**：女王登基→母系改革→女官体系→母系盛世→于阗女王
- **可获得武将**：玉匠·阗琢(♀·文官)、织女·阗锦(♀·文官)、女王·阗华(♀·武将)(线3)

#### 乌孙 🐎（3条线）

- **线1「天马之国」**：天马驯养→骑兵建设→天马军团→草原帝国→天马之王
- **线2「草原联盟」**：部落联合→草原议会→联盟扩张→草原帝国→草原之王
- **线3「铁骑风暴」**：重骑建设→铁骑训练→铁骑征服→铁骑帝国→铁骑之王
- **可获得武将**：驯马师·孙驰(♂·武将)、部落长·孙联(♂·文官)、铁骑将·孙铁(♂·武将)(线3)

#### 疏勒 🕌（3条线）

- **线1「商贸之王」**：大巴扎的考验→万国通译→雇佣兵之王→东西交汇→疏勒王

## 四、部落系统

### 4.1 部落总览

> **核心规则**：
> - **数量**：大国周边5个部落、中等国周边4个部落、小国周边3个部落（阿玛宗例外4个）。总计(9×5)+(13×4)+(16×3+1×4) = 45+52+52 = **149个部落**。
> - **制度一致**：每个周边部落的制度与所属城邦一致。普通城邦的周边部落为普通部落，母系城邦的周边部落为母系部落，全女性城邦的周边部落为全女性部落。
>   - 例：于阗(母系大国)的5个部落均为母系部落；西梁女国(全女性大国)的5个部落均为全女性部落；楼兰(普通大国)的5个部落均为普通部落。
> - **任务**：每个部落有且仅有1条独特故事线，包含3个任务节点。完成部落故事线后解锁对应能力并获得该部落的资源贡赋。

| 国家类型 | 周边部落数 | 部落任务奖励 |
|----------|-----------|-------------|
| 大国（9个，含西梁女国） | 5个 | 每个部落：资源贡赋+解锁能力+1名武将/文官 |
| 中等国（13个，含东女国） | 4个 | 同上 |
| 小国（17个，含阿玛宗） | 3个（阿玛宗4个） | 同上 |

### 4.2 部落任务与能力解锁

```
部落任务解锁进程：

部落任务1 → 解锁：奴隶买卖/资源采集/农业/马场
部落任务2 → 解锁：奴隶劳作/商业/税收/锻造
部落任务3 → 解锁：奴隶培训/高级能力

完成≥3个部落任务 → 进入城市发展阶段
完成目标国≥50%部落任务 → 可对该国宣战
```

### 4.3 部落类型

| 部落类型 | 互动方式 | 说明 |
|----------|----------|------|
| **独立部落** | 通过任务/外交/军事互动 | 有独立故事线，完成后提供贡赋 |
| **争议部落** | 多国共享，需竞争控制 | 完成任务可获得控制权 |
| **归附部落** | 通过外交或军事手段使其归附 | 自动成为己方资源来源，不可掠夺 |

```
  剧情：建立城邦的故事
  系统：教程阶段，学习基础操作
  闭环：剧情推动玩家学习系统，系统掌握推动剧情进展

部落征途 ←→ 奴隶系统+农业商业+军事基础
  剧情：征服/联盟周边部落的故事
  系统：扩张阶段，学习军事和外交
  闭环：部落征服带来资源和人口，支撑进一步发展

城市发展 ←→ 外交系统+科技+商路+谍报
  剧情：城市繁荣与外交博弈的故事
  系统：深化阶段，多系统协同运作
  闭环：外交关系影响商路收入，科技提升产出

扩张征伐 ←→ 全面战争+贸易垄断+外交联盟
  剧情：争夺西域霸权的故事
  系统：高级阶段，所有系统全面运作
  闭环：军事力量保障商路安全，经济实力支撑军事消耗

统一西域 ←→ 多路径胜利条件
  剧情：统一西域的最终章
  系统：终局阶段，验证所有系统的积累
  闭环：玩家选择的发展路线决定最终结局
```

### 7.2 任务-资源闭环

```
任务奖励 → 资源获取 → 系统建设 → 新任务解锁

具体闭环：
  城邦任务 → 初始资源+管理权 → 基础建设 → 部落征途
  部落任务 → 资源贡赋+解锁能力 → 扩张发展 → 城市发展
  城邦剧情线 → 专属武将+永久加成 → 实力增强 → 扩张征伐
  随机事件 → 机遇/挑战 → 应对决策 → 影响发展路线
```
