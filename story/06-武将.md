## 一、武将双轨体系总论

### 1.1 双轨制核心概念

> 武将系统分为**两套独立但可交互的体系**：**势力武将/文官**（归属国家）和**玩家随从/侍卫**（归属玩家个人）。阶段五以后逐步融合，阶段六完成合并。

| 维度 | 势力武将/文官 | 玩家随从/侍卫 |
|------|-------------|-------------|
| **归属** | 国家/势力 | **玩家角色个人** |
| **忠诚对象** | 国家/统治者 | **玩家角色** |
| **招募方式** | 奴隶培养/任务奖励/招降/科举 | 玩家主动招募/剧情获得 |
| **消耗** | 国库俸禄 | **玩家私库** |
| **军事指挥** | 受国家军事制度约束 | 受玩家身份等级约束 |
| **数量上限** | 无硬性上限（俸禄约束） | 受玩家身份等级限制 |
| **阶段五核心** | — | **拉拢势力武将为玩家武将** |
| **阶段六核心** | **排除异己+合并双轨** | **合并完成** |
| **阶段七核心** | 统一体系 | 统一体系 |

### 1.2 双轨阶段演进（对齐01-世界观七大阶段）

| 阶段 | 玩家身份 | 势力武将 | 玩家随从 | 两轨关系 |
|------|---------|---------|---------|---------|
| **一·城邦初立** | 奴隶/公民 | 国家已有（不可控） | 首批随从1~3人 | 独立运行 |
| **二·部落征途** | 幕僚/参谋 | 可下达部分指令 | 随从任务解锁 | 独立运行 |
| **三·城市发展** | 初级官员 | 可下达部分指令 | 侍卫→参与城防 | 侍卫协助城邦 |
| **四·丝路争霸** | 中高级官员 | 完全控制 | 侍卫→文官/武将 | 可执行相同任务 |
| **五·帝国博弈** | 丞相/国师 | 完全控制 | **拉拢势力武将** | 交互开始 |
| **六·势力崛起** | 势力实际统治者※ | **排除异己后合并** | **合并** | 统一体系 |
| **七·一统西域** | 西域霸主/女帝 | 统一体系 | 统一体系 | 无区分 |

> **※阶段六实际统治者身份因国家类型与性别而异**：普通国→国王/女王/汗王；母系国→女王/摄政王(男)/教皇(男)；全女国→女帝/圣女(女)/唯一合法男性(男，需忠诚度≥80)；宗教领袖制→教皇/圣女/宗教领袖

## 五、阶段五·幕府与拉拢系统

### 5.1 幕府系统（阶段五核心解锁）

> 玩家达到阶段五（丞相/国师身份，Lv≥8）时解锁**幕府**——玩家私人权力机构，独立于国家官僚体系。

| 幕府职位 | 人数 | 等级要求 | 功能 |
|----------|:---:|:------:|------|
| 幕僚长 | 1 | Lv9+ | 管理全部玩家武将 |
| 首席文官 | 1 | Lv7+ | 代行内政/外交 |
| 首席武将 | 1 | Lv7+ | 代行军事指挥 |
| 幕僚 | ≤5 | Lv5+ | 执行文官/武将任务 |
| 侍卫长 | 1 | Lv6+ | 管理近卫军/私兵 |

### 5.2 拉拢势力武将机制

> **阶段五核心玩法**：将势力武将拉拢为玩家武将。拉拢成功后武将脱离国家体系，转为玩家个人所有。

#### 拉拢条件

| 条件 | 说明 |
|------|------|
| 玩家等级 | ≥Lv8 |
| 游戏阶段 | ≥阶段五 |
| 目标武将忠诚度 | ≤59（动摇或不满） |
| 目标武将派系 | 自身派最容易(★)、改革派中等(★★)、王权派最难(★★★★) |
| 幕府职位空缺 | 有对应职位空位 |

#### 拉拢方式（三种）

| 方式 | 消耗 | 成功概率 | 适合对象 |
|------|------|:-------:|---------|
| **私下赠礼** | 500~2000金+特产/宝物 | 30%~50% | 自身派、忠诚度40~59 |
| **利益承诺** | 承诺更高职位/封地/特权 | 40%~60% | 改革派、自身派 |
| **理念共鸣** | 完成特定剧情对话+玩家魅力≥70 | 50%~70% | 改革派 |

#### 拉拢流程

```
1. 选择目标势力武将/文官（忠诚度≤59）
2. 选择拉拢方式（赠礼/承诺/共鸣）
3. 计算成功概率：
   基础概率 = 方式基础概率
   + 玩家魅力加成(魅力每10点+5%)
   + 忠诚度加成(越低越容易：40~59时+0%，20~39时+10%，0~19时+20%)
   + 派系匹配加成(自身派+10%，改革派+0%，王权派-20%)
   - 目标武将统率/智力抵抗(每10点-3%)
4. 随机判定
5. 成功→武将转入幕府(忠诚度重置为60)
6. 失败→武将忠诚度-10，若为王权派则可能向统治者告密(触发"阴谋暴露"事件)
```

#### 拉拢失败后果

| 失败类型 | 后果 | 处理 |
|---------|------|------|
| 被拒绝 | 目标忠诚度-5，3月冷却期 | 可再次尝试 |
| 被告密 | 王权派向统治者告密→触发"阴谋暴露"事件 | 玩家政治影响力-15，3月内不可拉拢 |
| 反向策反 | 自身派武将被敌国收买→叛逃 | 失去武将，好感下降 |

## 六、阶段六·排除异己与势力合并

### 6.1 排除异己机制

> **阶段六核心玩法**：玩家成为统治者后，必须**排除异己**——清除不忠诚的势力武将/文官，确保权力稳固。排除异己后，势力武将与玩家武将合并为统一体系。

#### 排除异己方式（四种）

| 方式 | 适用对象 | 效果 | 风险 |
|------|---------|------|------|
| **🤐 流放** | 忠诚度20~39(不满) | 武将离开势力，不带走军队 | 稳定度-5；被流放者可能投敌 |
| **⛓️ 降奴** | 忠诚度0~19(背叛) | 武将变为奴隶/战奴 | 稳定度-2；但武将属性归零 |
| **💀 处决** | 忠诚度0~19+有叛乱行为 | 武将死亡，震慑效果 | 稳定度-10短期(+15长期)；其他武将忠诚度+10(恐惧)或-15(反感) |
| **🤝 收买** | 自身派+忠诚度40~59 | 花费1000~3000金收买→忠诚度→80 | 成本高；但保留武将 |

#### 排除异己判定流程

```
1. 进入阶段六 → 触发"权力审查"事件
2. 所有势力武将/文官进行忠诚度重评：
   - 王权派：若玩家身份为正统统治者→忠诚+20；若为篡权者→忠诚-20
   - 改革派：若玩家推动改革→忠诚+15；若保守→忠诚-10
   - 自身派：根据玩家势力实力判定(国力前3→忠诚+10；否则-5)
3. 忠诚度重评后，低于40的武将自动标记为"异己"
4. 玩家逐个决定异己的处理方式（流放/降奴/处决/收买）
5. 处理完毕后→势力武将与玩家武将合并
```

### 6.2 势力合并机制

> 排除异己完成后，势力武将/文官与玩家随从/侍卫**合并为统一体系**。合并规则如下：

| 合合前 | 合并后 | 说明 |
|-------|-------|------|
| 势力武将(忠诚≥60) | **玩家武将** | 忠诚对象转为玩家 |
| 势力文官(忠诚≥60) | **玩家文官** | 忠诚对象转为玩家 |
| 玩家随从/侍卫(Lv5+) | **玩家文官/武将** | 自动提升为正式文官/武将 |
| 玩家随从(Lv1~4) | **幕府随从** | 保留随从身份，辅助幕府运作 |
| 势力武将(忠诚<40) | **被排除** | 流放/降奴/处决/收买后处理 |
| 玩家私兵 | **禁卫军** | 私兵升级为禁卫军(阶段六专属兵种) |

**合并后数量上限**：阶段六无上限（受俸禄/私库约束）

## 七、贵族私兵系统

### 7.1 私兵核心概念

> 私兵是玩家从**封地招募**的个人军队，不从国家兵源征召，不受国家军事制度约束。私兵是玩家权力上升的核心军事保障，阶段六时升级为**禁卫军**。

#### 私兵核心规则

| 规则 | 说明 |
|------|------|
| **指挥权** | 私兵**只能由玩家或玩家专属武将（随从/侍卫/幕府武将）率领**，不可交由势力武将指挥 |
| **数量上限** | 私兵数量上限**3,000人**（禁卫军阶段5,000人），不可突破 |
| **战斗力** | 私兵战斗力**高于国家正规军**（基础战力×1.2），因忠诚度高+装备自选+训练定制 |
| **性别与阶级** | 私兵**不受性别和阶级限制**——公民/奴隶/战奴均可编入私兵（全女国例外：因全国无男性，私兵仍为全女兵） |
| **维护来源** | 从玩家私库支出，不消耗国库 |
| **战损补充** | 从封地人口补充（含专属奴隶），补充速度-20%但忠诚度+15 |

#### 私兵专属用途（仅私兵可执行）

> 以下行动**必须使用私兵**，国家正规军不可执行：

| 专属用途 | 说明 | 最低私兵数 |
|----------|------|:---------:|
| **平叛** | 镇压内部叛乱（男权反扑/低稳定度/保守势力叛乱） | 200 |
| **铲除异己** | 阶段六排除异己武将的军事行动 | 500 |
| **篡位** | 篡夺国家最高权力 | 1,000 |
| **逼宫** | 迫使当前统治者让位 | 1,000 |
| **政变** | 发动军事政变夺取政权 | 2,000 |

#### 私兵护卫与反暗杀系统

> 私兵是**保护玩家人身安全**的唯一军事力量。异己武将、间谍、叛军均可能对玩家实施暗杀，私兵负责护卫与反制。

| 威胁来源 | 触发条件 | 暗杀概率 | 私兵护卫效果 |
|----------|----------|:--------:|-------------|
| **异己武将暗杀** | 阶段六排除异己期间 | 5%~15%/月 | 私兵≥200→概率-80%；侍卫长武力≥70→-90% |
| **间谍暗杀** | 敌国谍报任务 | 3%~10%/次 | 私兵≥100→概率-70%；侍卫智力≥60→提前2回合预警 |
| **叛乱军暗杀** | 叛乱期间玩家亲临前线 | 10%~25%/战 | 私兵≥500→概率-85%；禁卫军→-95% |
| **后宫阴谋** | 阶段六后宫势力不满 | 2%~8%/月 | 侍卫长+女侍卫≥3→-75% |
| **保守势力暗杀** | 玩家夺权后保守势力反扑 | 5%~20%/月 | 私兵≥300→-80%；禁卫军→-95% |

**反暗杀机制**：
- 私兵护卫成功→暗杀者被擒获→可审讯获取情报
- 私兵护卫失败→玩家受伤（战力-30%，3月恢复）或死亡（游戏结束）
- **禁卫军护卫**：阶段六后禁卫军接管护卫，暗杀概率≤1%

#### 私兵参战与战利品规则

| 参战类型 | 战利品归属 | 军功 | 说明 |
|----------|----------|------|------|
| **征服他国** | **归国家** | 私兵军功×1.5 | 战利品归国库，但军功+50% |
| **掠夺他国/外部势力** | **归玩家** | 正常 | 战利品/奴隶/归降武将/敌国后宫眷属**全部归玩家私库** |
| **城邦自卫** | — | 正常 | 私兵参与城邦防御战 |

#### 国家正规军职责范围

> 国家正规军职责**仅限外部征战**，不可用于内部政治行动：

| 正规军职责 | 说明 |
|-----------|------|
| **城邦自卫** | 防御敌国进攻 |
| **掠夺他国/外部势力** | 正规军掠夺，战利品归国库 |
| **征服他国/外部势力** | 正规军征服，领土/资源归国家 |

### 7.4 封地专属奴隶系统

> 玩家解锁封地（Lv5+）后，可拥有**专属奴隶**——归属玩家私库，不由国家管理，由玩家专属武将/文官管理。

#### 专属奴隶获取

| 获取途径 | 说明 | 消耗 |
|----------|------|------|
| **市场购买** | 从奴隶市场购买，归属玩家而非国家 | 市场价×1.1（私购溢价10%） |
| **私兵掠夺** | 私兵掠夺他国/外部势力获得的奴隶 | 无额外消耗（掠夺收入） |
| **封地招募** | 流民/难民自动归附封地 | 20金/人安置费 |
| **敌国后宫眷属** | 私兵掠夺获得的敌国后宫眷属 | 无额外消耗（掠夺收入） |

#### 专属奴隶用途

| 用途 | 说明 | 管理者 |
|------|------|--------|
| **私兵兵源** | 专属奴隶可编入私兵（不受阶级限制） | 侍卫长/近卫长 |
| **封地劳力** | 分配至封地农田/工坊/矿场劳动 | 首席文官/幕僚 |
| **私库收入** | 侍妾→人口增长+私库金收入；舞女→私库金收入 | 首席文官 |
| **武将培养** | 侍妾/舞女→玩家随从→武将/文官 | 幕僚长 |
| **转正为公民** | 300金+10回合→封地公民（扩大封地人口） | 首席文官 |

#### 专属奴隶管理规则

| 规则 | 说明 |
|------|------|
| **数量上限** | 封地人口×50%（奴隶不超过封地总人口一半） |
| **管理者** | 由玩家**专属武将/文官**管理，不可交由势力文官 |
| **消耗** | 从玩家私库支出（粮食0.6粮/人/月+管理费5金/人/月） |
| **产出** | 全部归玩家私库（不归国库） |
| **满意度** | 封地奴隶满意度独立计算，低于30可能逃亡 |

### 7.5 禁卫军（阶段六终极私兵）

> 阶段六玩家身份升级为统治者后，私兵自动升级为**禁卫军**——西域最精锐的私人军队。

| 维度 | 禁卫军规格 |
|------|-----------|
| **规模** | 最大5,000人 |
| **基础战力** | ×1.3（比普通兵高30%） |
| **忠诚度** | 固定90（不可降低） |
| **装备** | 精英装（全属性+20%） |
| **特殊能力** | ①不可被策反/叛变；②守城时防御+40%；③攻城时攻击+20%；④玩家亲自率军时士气+30% |
| **兵种构成** | 玩家自定义（步兵/骑兵/弓兵比例自由调配） |
| **维护费用** | 3金/人/月（从玩家私库支出） |

## 八、兵种系统

### 8.1 兵种培养链路

```
【步兵】公民/战奴→新兵(Lv0)→普通(Lv1)→精锐(Lv2)→老兵(Lv3)
  ├→ 长矛兵专精→反骑大师(克制骑兵+15%)
  ├→ 盾兵专精→铁壁守卫(守城防御+25%)
  └→ 重步兵专精→破阵先锋(攻城+20%)

【骑兵】公民/战奴→新兵(Lv0)→普通(Lv1)→精锐(Lv2)→老兵(Lv3)
  ├→ 轻骑兵专精→游击大师(速度+30%)
  ├→ 重骑兵专精→突击铁骑(冲锋+25%)
  └→ 弓骑兵专精→射骑高手(远程+20%)

【升级条件】
新兵→普通：武将训练3回合+10金/人
普通→精锐：武将训练6回合+20金/人+战斗≥2场
精锐→老兵：武将训练10回合+30金/人+战斗≥5场+统率≥60

长矛兵：民兵营Lv2+统率≥50
```
