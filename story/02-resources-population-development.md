## 一、资源体系

### 1.1 十五种资源商品总览

> **设计原则**：精简为15种核心资源/商品（7基础+8贸易），每种都有完整的获取→存储→消耗闭环。所有商品和奴隶均可通过市场贸易购买，不同市场有品类和价格差异。

【香料链】  香料→外交赠礼/寺庙供奉/贵族消费
【驼运链】  骆驼→商队运输/沙漠农业/外交赠礼/建设（需驼厩）

```

### 2.3 月度结算顺序

```
每月结算顺序（含消耗类型标注）：

1.  【月度产出】农业/工业/纺织/商业产出 → 资源入库
2.  【月度消耗】男公民粮食(1粮/人) + 女公民粮食(0.8粮/人) + 奴隶粮食(0.6粮/人) + 士兵粮食(1.5~1.8粮/人)
3.  【月度消耗】马匹马料(2粮/匹)
4.  【月度消耗】盐消耗(人口÷100)
5.  【月度消耗】铁匠铺燃料(5木/月)
6.  【月度消耗】兵器日常损耗(0.5%)
7.  【月度消耗】军费维护(兵力÷100金)
8.  【月度消耗】武将/文官俸禄(10~50金/人)
9.  【月度消耗】建筑维护费
10. 【月度收入】税收(人口×0.5金)
11. 【月度收入】部落贡赋(各种资源)
12. 【月度收入】商路收入(金币)
13. 【月度收入】酒吧收入(+5金/酒吧)
14. 【月度消耗】冬季额外消耗(皮毛+兵器损耗)
15. 【一次性】检查是否有建设完成/训练完成/培养完成
16. 【一次性】随机事件触发（可能产生一次性消耗或收入）
```

### 2.4 关键经济指标

```
月度净收入 = 月度总收入 - 月度总消耗
  月度总收入 = 税收 + 商路收入 + 部落贡赋 + 酒吧收入 + 舞女收入
  月度总消耗 = 粮食消耗 + 盐消耗 + 军费 + 俸禄 + 建筑维护 + 燃料

一次性储备金 = 当前金库 - 预计一次性支出
  预计一次性支出 = 建设预算 + 征兵预算 + 外交预算 + 购买预算

健康经济标准：
  月度净收入 > 0（每月盈余）
  一次性储备金 > 月度总消耗×3（至少3个月缓冲）
  粮食储备 > 月度粮食消耗×6（至少6个月口粮）
```

> **全女性国家限制**：西梁女国、东女国、苏毗女国、阿玛宗、弱水国、斯基泰只能购买/拥有女性奴隶（女劳奴、女匠奴、女战奴、舞女、侍妾）。男性奴隶自动按市价转化为金币。

> **奴隶买卖价格公式**：
> - **买入价格** = 基础价格 × (1 + 距离浮动率) × 1.05

> - 同一市场各品类每月有收购上限，防止单次批量倾销扰乱市价

### 3.3 人口增长公式

#### 公民人口增长

- **基础增长率** = 城市等级对应比率 (0.5%~2.0%/月)
- **实际增长** = 当前公民人口 × 增长率 × 季节修正 × 剩余粮食修正
- **上限** = 城市等级对应最大人口（大国50,000 / 中等国35,000 / 小国25,000）
- **全女性国家特殊**：西梁女国(子母河+20%)、东女国(外部通婚+20%)、苏毗女国(神女传承+20%)、阿玛宗(收养制度+20%)、弱水国(弱水生育+15%)、斯基泰(孤儿收养+15%)
- **母系社会国家特殊**：瑶池国/月氏国/玄女国/赤谷国/青鸾国/素女国/灵山国/玉河国(母系传承+10%)
- **注意**：侍妾**不再**对公民增长率有加成

#### 母系性别比例约束机制

> 母系国家的核心特征是女性占多数（初始男女比30:70）。随着游戏发展，玩家可能通过各种手段改变性别比例。为维持母系社会的本质特征，设置以下约束：

- **性别比例红线**：当男性人口占比超过40%（即男女比>40:60）时，触发**"母系失衡"**状态
- **失衡效果**：
  - 🚫 **禁止购买男性奴隶**：奴隶市场中的男性奴隶选项被锁定
  - 🚫 **禁止男性奴隶转化**：无法将男性奴隶转化为男性公民
  - 🚫 **男性公民自然增长停止**：人口增长时，新增公民全部为女性
  - ⚠️ **母系文化加成减半**：所有母系文化加成效果降低50%
  - ⚠️ **外交联姻成功率-20%**：母系国家认为你已背离传统
- **恢复条件**：当男性人口占比回落至40%以下（男女比≤40:60）时，自动恢复正常状态
- **设计目的**：防止玩家通过大量购买男性奴隶或转化手段将母系国家变为事实上的普通国家，保持母系国家的独特游戏体验

#### 奴隶人口自然增长

> **核心规则**：奴隶人口也会自然增长，年增长速度与侍妾占所有奴隶的比例挂钩。

- **奴隶年增长率** = 基础增长率(2%) × (1 + 侍妾占比加成)
- **侍妾占比加成** = (侍妾数量 ÷ 奴隶总数) × 200%
  - 例：100个奴隶中有10个侍妾 → 加成 = (10/100)×200% = 20% → 年增长率 = 2%×(1+20%) = 2.4%
  - 例：100个奴隶中有50个侍妾 → 加成 = (50/100)×200% = 100% → 年增长率 = 2%×(1+100%) = 4%
  - 上限：年增长率不超过10%
- **新增奴隶性别分配**：新增奴隶由玩家根据性别分配为不同分工（男劳奴、女匠奴等），性别比例根据国家制度：
  - 普通国家：55%男 / 45%女
  - 母系社会国家：30%男 / 70%女
  - 全女性国家：0%男 / 100%女
- **新增奴隶分工**：玩家确认分工后**不可改变**（见3.6节奴隶分工不可变规则）

### 3.9 人口增长对资源的影响链

```
人口增长 → 劳动力增加 → 资源产出增加 → 但粮食消耗也增加

关键平衡点：
  每新增1名男公民→分配农民→+3粮产出-1粮消耗=净+2粮/月 ✅
  每新增1名女公民→分配农民→+2.5粮产出-0.8粮消耗=净+1.7粮/月 ✅
  每新增1名男劳奴→分配农场→+2.4粮产出-0.6粮消耗=净+1.8粮/月 ✅
  每新增1名士兵→不产出→0-1.5粮消耗=净-1.5粮/月 ❌ 需要其他人养

最佳人口结构（粮食自给率最高）：
  农民45% + 纺织12% + 商业12% + 矿工8% + 伐木5% + 采石3% + 狩猎3% + 其他7% + 士兵5%
  → 粮食自给率≈110%（微盈余）
  → 但军事力量弱

最佳人口结构（军事强国）：
  农民35% + 纺织10% + 商业10% + 矿工8% + 伐木5% + 采石3% + 士兵25% + 其他4%
  → 粮食自给率≈85%（需贸易补足）
  → 军事力量强

结论：人口越多→总产出越高→但需要合理分配职业→粮食是核心瓶颈
