## 一、外交系统

> 外交系统在城市发展阶段（城市Lv2）解锁。

### 1.1 外交操作

| 操作 | 效果 | 费用 | 条件 |
|------|------|------|------|
| **互不侵犯条约** | 双方承诺不攻击 | 100-300金谈判费 | 关系≥中立 |
| **贸易协定** | 双方贸易税率-10% | 100-200金谈判费 | 关系≥中立 |
| **军事同盟** | 共同防御/进攻 | 300-500金谈判费 | 关系≥友好 |
| **联姻** | 关系+30，获得武将 | 聘礼500-2,000金 | 关系≥友好+有合适武将 |
| **朝贡** | 向强国进贡换取保护 | 粮食+金币/月 | 关系≥冷淡 |
| **宣战** | 进入战争状态 | 威望-20 | 城市Lv3+宣战能力 |

### 1.2 联姻系统

```
联姻条件：
  - 双方关系≥友好（61+）
  - 至少一方有可婚配武将（未婚）
  - 支付聘礼（500-2,000金）

联姻效果：
  - 双方关系+30（永久基础加成）
  - 获得对方一名武将
  - 母系国家：联姻由女方主导，效果+50%
  - 全女性国家联姻：需要特殊剧情触发
```

### 1.3 谍报系统

> 谍报系统在城市发展阶段（城市Lv3）解锁，由武将执行侦察谍报任务实现：

| 功能 | 效果 | 条件 |
|------|------|------|
| **侦察** | 获取敌方城市信息（兵力/资源/武将） | 武将智力≥50 |
| **提前预警** | 提前2回合预警入侵 | 武将执行侦察任务 |
| **商路情报** | 探知市场行情→贸易最优时机 | 武将执行侦察任务 |
| **外交情报** | 掌握敌方弱点→外交谈判优势+20 | 武将执行侦察任务 |

### 1.4 宣战与战争系统

```
宣战必要条件：
  1. 城市等级≥Lv3
  2. 已解锁"宣战能力"
  3. 拥有≥1名武将
  4. 拥有≥100名士兵
  5. 完成目标国50%以上部落任务

宣战后果：
  - 与目标国家关系归零
  - 目标国家盟友可能参战
```

## 二、交战系统（战棋）

### 2.1 战场系统

战斗采用**回合制战棋**模式，在六角格地图上进行。

#### 战场规模

| 规模 | 地图大小 | 最大部队 | 回合限制 | 触发条件 |
|------|----------|----------|----------|----------|
| **小规模遭遇战** | 10×8 | 每方500兵 | 10回合 | 边境冲突/巡逻遭遇 |
| **中等战斗** | 16×12 | 每方2,000兵 | 15回合 | 部落争夺/资源点攻防 |
| **大规模会战** | 24×16 | 每方5,000兵 | 20回合 | 城市攻防/主力决战 |
| **决战** | 32×20 | 每方10,000兵 | 30回合 | 灭国战争/丝路争霸 |

#### 地形效果（战棋层面）

| 地形 | 移动消耗 | 防御加成 | 攻击影响 | 特殊效果 |
|------|----------|----------|----------|----------|
| **平地** | 1 | 0% | 0% | 无 |
| **草地** | 1 | +5% | 0% | 骑兵冲锋+10% |
| **沙漠** | 2 | -10% | -10% | 每回合-5%体力 |
| **山地** | 3 | +30% | -15% | 骑兵无法冲锋 |
| **丘陵** | 2 | +15% | -5% | 弓兵射程+1 |
| **森林** | 2 | +20% | -10% | 伏击伤害+30% |
| **河流** | 3 | +10% | -20% | 渡河时防御-20% |
| **城墙** | 1 | +50% | -20% | 守城方巨大优势 |
| **绿洲** | 1 | +10% | 0% | 每回合恢复5%体力 |

### 2.2 战斗流程

```
每回合战斗流程：

1. 部署阶段
   - 攻方先部署（选择初始位置）
   - 守方后部署（可利用地形优势）

2. 行动阶段（交替进行）
   - 武将指挥：选择武将执行战术动作
   - 部队移动：按兵种移动范围移动
   - 攻击行动：对范围内敌军发起攻击
   - 技能释放：武将使用特殊技能
   - 阵型调整：改变部队阵型

3. 结算阶段
   - 计算双方伤亡
   - 检查士气状态
   - 判断胜负条件

胜负条件：
  - 歼灭敌方全部部队
  - 敌方士气降为0（溃败）
  - 占领敌方大本营/城池
  - 达到回合限制（平局，双方损失）
```

### 2.3 战斗公式

```
伤害计算：
  基础伤害 = 兵种攻击力 × (1 + 武将武力×0.01)
  实际伤害 = 基础伤害 × 克制系数 × 地形系数 × 士气系数 × 随机(0.8-1.2)

防御计算：
  实际减伤 = 兵种防御力 × (1 + 地形防御加成) × (1 + 阵型加成)
  最终伤害 = max(实际伤害 - 实际减伤, 1)

士气系统：
  初始士气 = 100
  士气下降：每损失10%部队 → 士气-10
  士气恢复：武将鼓舞技能 → 士气+20
  士气崩溃：士气≤0 → 全军溃败

  士气影响：
    士气>80：攻击+10%
    士气60-80：正常
    士气40-60：攻击-10%，移动-1
    士气20-40：攻击-20%，移动-2，有概率逃跑
    士气<20：攻击-40%，每回合10%部队逃跑
```

| **长蛇** | 移动+3，侧翼脆弱 | 骑兵 | 快速机动 |

```
攻城战特殊规则：

城防等级：
  Lv1 土墙：防御+20%，耐久500
  Lv2 木墙：防御+30%，耐久1,000
```

## 三、战争结果体系

### 3.1 战争五结果

> **核心设计**：征服/联盟一个国家有五种结果——掠夺、同盟、联姻、附庸、占领。玩家可根据自身路线选择最优方式。

> **重要规则**：任何外交/军事操作均需完成目标国**指定数量的故事线**（大国3条/中等国2条/小国2条）。

### 3.2 掠夺

> **定义**：在不占领的情况下，出兵袭击目标国，劫掠金币和资源。**掠夺不计入统一进度**，主要用于经济压制和削弱。

| 维度 | 规则 |
|------|------|
| **触发条件** | ①城市等级≥Lv2 ②目标国关系≤中立 ③完成目标国≥1条故事线 |
| **执行方式** | 出兵→选择"掠夺"→行军到目标国→投入战斗（战斗过程同交战系统） |
| **掠夺收益** | ①金币：目标国金库×10%~20% ②粮食：目标国粮仓×10%~20% ③特产：一定概率掠夺特产（概率=目标国特产数量×10%）④俘虏：一定概率掠夺人口（目标国人口×2%~5%） |
| **掠夺代价** | ①与目标国关系-30 ②目标国可能宣战报复 ③掠夺军队需消耗粮食（行军消耗+战损）④威望-10 |
| **掠夺与统一** | ❌ 掠夺**不**计入统一进度 |
| **掠夺限制** | ①每国每季度最多只能对同一城/部落掠夺1次（敌方有多座城则至多每城一次）②对同一城/部落，同一年（第一次掠夺开始计算的滚动周期）连续掠夺收益递减：第二次50%，第三次25% ③冬季掠夺收益减半 ④掠夺不可与同盟/附庸同时存在 |
| **掠夺影响·部落** | 掠夺部落，则该部落当年对势力主的供奉减少（城邦不受影响） |
| **掠夺影响·城邦** | 掠夺城邦，则目标城邦的人口⭐、资源、城防下降 |
| **适用场景** | 前期快速获取资源、经济压制、军事路线的阶段性手段 |

### 3.3 联姻

> **定义**：通过外交联姻建立紧密联盟。联姻国**不计入统一进度**，需进一步转化为同盟/附庸/占领才计入。

| 维度 | 规则 |
|------|------|
| **触发条件** | ①双方关系≥友好（61+）②至少一方有可婚配武将（未婚）③支付聘礼（500~2,000金）④完成目标国≥1条故事线 |
| **执行方式** | 外交菜单→选择"联姻"→选择婚配武将→支付聘礼→对方接受（好感越高接受率越高） |
| **联姻效果** | ①双方关系+30（永久基础加成）②获得对方一名武将③双方不可互相攻击④共享情报（视野共享）⑤贸易收入+20%⑥**不计入统一进度**（需进一步转化为同盟/附庸/占领） |
| **母系国家加成** | 联姻由女方主导，效果+50%（关系+45、获得2名武将、贸易+30%） |
| **全女性国家** | 需特殊剧情触发（西梁女国：子母河水外交；东女国：外部通婚协议；阿玛宗：战魁联姻） |
| **解除条件** | ①联姻武将死亡→联姻降级为"同盟"②玩家主动撕毁→威望-50、对方宣战 |
| **计入统一进度** | ❌ 否（联姻国需进一步转化为同盟/附庸/占领才计入） |
| **适用场景** | 快速建立紧密联盟、获取武将、文化同化路线核心手段（联姻→同盟/附庸/占领的跳板） |

### 3.4 同盟

> **定义**：通过外交协议建立军事同盟，双方承诺共同防御/进攻。同盟国保留完全自治权。

| 维度 | 规则 |
|------|------|
| **触发条件** | ①双方关系≥友好（61+）②支付谈判费（300~500金）③完成目标国≥1条故事线 |
| **执行方式** | 外交菜单→选择"军事同盟"→支付谈判费→对方接受（好感越高接受率越高） |
| **同盟效果** | ①共同防御：一方被攻击，另一方自动参战②共同进攻：可邀请同盟国参与己方战争③共享情报④贸易税率-15% |
| **同盟限制** | ①同盟国保留独立外交权（可能与你的敌人也是同盟）②同盟国保留独立军事权（不一定响应你的进攻号召）③**同盟国对玩家不产生任何收益**（无贡赋、无资源、无军事调遣权） |
| **同盟破裂** | ①一方攻击另一方的盟友→同盟自动破裂②玩家主动退出→关系-30③同盟国好感<30→可能主动退出 |
| **转化方式** | ①同盟→联姻：满足联姻条件后可升级②同盟→附庸：通过经济/军事施压（军事实力3倍于对方+好感≥40→可要求对方成为附庸）③同盟→占领：撕毁同盟→宣战→军事征服 |
| **计入统一进度** | ✅ 是 |
| **适用场景** | 建立防御网络、文化同化路线、帝国纵横路线核心手段 |

### 3.5 附庸

> **定义**：通过军事压力或经济控制，使目标国成为附庸。附庸国保留内政权，每年向玩家供奉自身产出的30%，但不服从军事调遣。

| 维度 | 规则 |
|------|------|
| **触发条件（满足任一）** | ①**军事附庸**：战争中大胜（敌方损失>60%）→对方求和→选择"纳为附庸"②**经济附庸**：目标国对玩家债务>GDP的80%→强制变为经济附庸③**威慑附庸**：军事实力达到目标国**5倍**以上+完成目标国全部故事线→可要求对方成为附庸④**外交附庸**：目标国关系≥友好+玩家威望≥80→可邀请对方成为附庸（接受率=好感×1.5%） |
| **附庸效果** | ①附庸国**每年**缴纳贡赋（金币+粮食+特产，合计为附庸国自身年产出×30%）②**不服从军事调遣**（不可调遣附庸国军队参战）③玩家获得附庸国的通行权和商路权④附庸国不可与玩家的敌人结盟 |
| **附庸限制** | ①附庸国保留内政权（自行管理人口/资源/建设）②附庸国保留独立文化/制度（不需要同化，无同化成本）③附庸国忠诚度独立计算（忠诚<30可能叛乱）④**玩家不承担**附庸国的治安、粮食、必需品消耗⑤**玩家不可在附庸国征兵**⑥**玩家不可在附庸国掠夺** |
| **附庸→占领** | 可直接对附庸国宣战→军事征服→占领（附庸国兵力通常较弱，且玩家已熟悉其地形） |
| **附庸忠诚度** | 初始忠诚=60。每月变化：贡赋压力-2/月；驻军威慑+3/月（每100驻军）；文化影响+1/月；联姻绑定+5/月。忠诚<30→叛乱风险；忠诚<10→必定叛乱 |
| **转化方式** | ①附庸→占领：宣战→军事征服（附庸国兵力通常较弱）②附庸→同盟：附庸忠诚≥80+玩家主动释放→变为同盟关系③附庸→独立：忠诚<10→叛乱独立→成为敌对势力 |
| **计入统一进度** | ✅ 是 |
| **适用场景** | 经济霸权路线核心手段、军事征服路线的辅助手段、降低管理成本 |

### 3.6 占领

> **定义**：通过军事征服完全吞并目标国，将其纳入玩家领土。占领是控制程度最高的结果，但需要支付同化成本。

| 维度 | 规则 |
|------|------|
| **触发条件** | ①完成目标国**全部**故事线（大国3条/中等国2条/小国2条）②完成目标国≥50%部落任务 ③城市等级≥Lv3 ④宣战→战争中攻陷目标国首都（城防耐久降为0+占领大本营）⑤或军事威慑：军事实力达到目标国**10倍**→目标国自动投降 |
| **执行方式** | 宣战→击败目标国军队→攻陷首都→选择"占领"→支付同化成本→目标国纳入领土 |
| **占领收益** | ①获得目标国全部领土、资源产出、人口②获得目标国金库50%+全部库存资源③获得目标国部分武将（忠诚度高的武将可能逃跑或自杀）④获得目标国全部部落（自动归附） |
| **同化成本（一次性）** | 金币×人口数×制度差异系数：同族制度×0（免成本）；跨族制度×0.5金/人；全女性↔普通×1金/人。详见01-世界观.md第四阶段同化决策 |
| **占领后管理** | ①占领城邦按玩家选择的制度运行②人口性别比例按同化后的制度调整③占领城邦治安初始=30（需驻军或文化建设恢复）④占领城邦产出前3个月减半（"战后恢复期"） |
| **资金不足后果** | ①差额≤30%→强制借贷（粟特商会，本金+50%利息）②差额>30%→城邦进入"失控"状态（治安固定20、产出归零、叛乱风险累积） |
| **转化方式** | ①占领→不可逆（一旦占领不可退回）②占领城邦内部制度可通过第四阶段"统一决策"调整（需再次支付同化成本） |
| **计入统一进度** | ✅ 是 |
| **适用场景** | 军事征服路线（线路A）唯一统一方式、其他路线的核心领土来源 |

### 3.7 五种结果相互转化矩阵

> **核心设计**：五种结果之间可单向或双向转化，形成完整的"外交-军事"闭环。

```
转化路径图：

  掠夺 ──(无法直接转化)──→ 需先建立外交关系
    ↓ 完成故事线+外交谈判
  同盟 ──(联姻条件)──→ 联姻（联盟）
    ↓ 军事/经济施压          ↓ 联姻武将死亡
  附庸 ←──(释放附庸)──→ 同盟
    ↓ 宣战+军事征服
  占领（不可逆）

  同盟 ──(撕毁+宣战)──→ 战争 ──(胜利)──→ 占领
  附庸 ──(忠诚<10叛乱)──→ 独立（敌对）──(再次征服)──→ 占领
  联姻 ──(主动撕毁)──→ 敌对 ──(宣战)──→ 占领
```

| 从→到 | 掠夺 | 同盟 | 联姻 | 附庸 | 占领 |
|-------|------|------|------|------|------|

## 六、雇佣兵系统

> 雇佣兵是西域战争中的重要补充力量。当本国兵力不足（如制度转型遣散军队后），玩家可从**12大奴隶市场**雇佣西域各国的精锐部队临时作战。雇佣兵**不消耗本国粮食、武器等耗材**，仅每月收取金钱；雇佣兵**不参与性别比例或人口规模计算**。

### 6.3 雇佣兵规则

| 维度 | 规则 | 说明 |
|------|------|------|
| **雇佣条件** | ①城市等级≥Lv2 ②通过任一奴隶市场 ③国库有足够资金 | 无需与提供国直接外交 |
| **雇佣数量上限** | 雇佣兵总数不可超过本国总兵力的**80%**（硬性上限，不可突破） | 消费国视角：限制雇佣兵依赖度 |
| **雇佣期限** | 最少1月，按月续约。可随时解约（不退当月费用） | 解约后雇佣兵立即撤离 |
| **消耗规则** | **不消耗本国粮食、武器、护甲等任何耗材** | 雇佣兵自带装备和补给，仅收取金钱 |
| **人口规则** | **不参与性别比例或人口规模计算** | 雇佣兵为独立编制，不影响本国人口统计 |
| **战斗规则** | 雇佣兵由玩家统一指挥，参与战斗时与本国军队相同 | 雇佣兵士气独立计算，不受本国士气影响 |
| **战损处理** | 雇佣兵战损由提供国补充（每月自动恢复10%战损） | 玩家无需额外支付战损补充费用 |
| **解约惩罚** | 战斗中解约→雇佣兵立即撤退→该回合防御-20% | 非战斗时解约无惩罚 |

### 6.4 提供国机制

> 提供雇佣兵的国家需投入资源培养雇佣兵，每月消耗粮食和武器，换取高额金钱收入。这是雇佣兵系统的经济闭环。

#### 6.4.1 提供国资源消耗

> 每个雇佣兵每月消耗资源为**精锐兵种（Lv2）的0.8倍**。

| 维度 | 规则 | 说明 |
|------|------|------|
| **粮食消耗** | 精锐兵种粮食消耗×0.8/人/月 | 雇佣兵需持续供给口粮 |
| **武器消耗** | 精锐兵种武器损耗×0.8/人/月 | 雇佣兵装备需定期维护 |
| **总消耗** | 约为精锐兵种维护成本的80% | 比本国精锐兵略低 |

#### 6.4.2 提供国收入

| 维度 | 规则 |
|------|------|
| **提成比例** | 雇佣价的**40%**作为提成收入归提供国 |
| **结算周期** | 每月结算时自动发放至提供国金库 |
| **提成计算** | 提成 = 当前在外雇佣兵总数 × 单价 × 40% |
| **收入上限** | 无上限——雇佣兵越多，提供国收入越高 |
| **净利润** | 收入（40%提成） - 支出（0.8倍精锐维护成本） = **高额净利润** |

**示例**：若乌孙有500名天马铁骑被雇佣（单价15金），则乌孙每月：
- 收入：500 × 15 × 40% = **3,000金/月**
- 支出：500 × 精锐维护成本×0.8 ≈ 500 × 8 = **4,000资源等价/月**
- 净利润：金币收入远超资源消耗成本，**以粮食/武器换取高额金钱**

#### 6.4.3 投放市场规则

> 玩家（作为提供国）可主动将本国士兵投放至雇佣兵市场，换取金钱收入。

| 维度 | 规则 | 说明 |
|------|------|------|
| **投放上限** | 不超过**总人口的30%** | 防止过度投放导致国防空虚 |
| **投放条件** | ①城市等级≥Lv2 ②投放士兵已训练完毕 ③投放后本国兵力不低于最低防御线 | 不可投放未训练的新兵 |
| **投放类型** | 根据国家类型决定可投放的雇佣兵类型 | 普通国→全男兵；母系国→混编；全女国→全女兵 |
| **投放收入** | 按上述提成规则执行（40%单价/人/月） | 被雇佣后持续获得收入，未被雇佣则无收入但也不消耗 |
| **召回机制** | 可随时召回投放的雇佣兵（需1回合） | 召回后恢复为本国士兵 |

### 6.5 雇佣兵与制度转型的配合

> 雇佣兵系统与制度转型（凤鸣觉醒/女儿当国）紧密配合，为遣散军队后的玩家提供即时战力补充。

| 转型事件 | 遣散情况 | 推荐雇佣兵 | 说明 |
|----------|----------|------------|------|
| **凤鸣觉醒**（普通→母系） | 遣散50%男性士兵 | 天马铁骑/赤甲重兵/帕米尔卫队/大宛铁骑 | 补充常规战力，度过转型期 |
| **女儿当国**（母系→全女） | 遣散全部男性士兵（约70%军队） | **猎头女骑/弓骑女兵**（全女兵） | 全女兵雇佣兵完美适配全女国军队体系 |

> **设计意图**：雇佣兵系统为制度转型提供了"缓冲带"——遣散军队虽然导致战力骤降，但玩家可立即用遣散补偿金从12大奴隶市场购买雇佣兵维持军力。全女兵雇佣兵（猎头女骑/弓骑女兵）的存在，确保了转型为全女国的玩家也能获得高战力补充，不会因制度转型而陷入军事真空。同时，提供国通过投放雇佣兵（上限总人口30%），以粮食/武器换取高额金钱，形成"以兵养国"的经济循环。

## 七、军事闭环验证

### 6.1 军事-经济闭环

```
经济产出（金币/粮食/铁矿）
    ↓
建设军事设施（兵营/马场/铁匠铺）
    ↓
征召士兵（消耗金币+兵器+人口）
    ↓
军队维护（持续消耗粮食+金币）
    ↓
战争胜利（获得领土/资源/俘虏）
    ↓
经济实力增强（更多产出/部落贡赋）
    ↓
建设更强军事力量（循环上升）

关键平衡点：
  - 军队消耗与经济产出的平衡
  - 兵力规模与人口上限的制约
  - 军事投资与民生发展的取舍
  - 战争收益与战争风险的权衡
```

### 6.2 武将-战斗闭环

```
获取武将（奴隶培养/任务奖励/招降/随机招聘）
    ↓
武将积累经验（月度+战斗+任务）
    ↓
武将升级（属性提升+技能解锁）
    ↓
战斗表现提升（更强指挥+特殊技能）
    ↓
战斗胜利（更多经验+战利品）
    ↓
武将继续成长（循环上升）

关键平衡点：
  - 武将数量与俸禄负担（10~50金/人/月）
  - 武将培养投入与回报
  - 忠诚度管理与人才保留
  - **舞女忠诚度加成**：舞女每月自动贡献+10金币，同时可分配给武将/文官，每月忠诚度+5；也可作为外交赠礼（+5好感）
  - 武将搭配与战术组合
```
