## 一、武将系统

### 1.1 武将属性

每位武将/文官拥有五项核心属性：

| 属性 | 范围 | 影响 |
|------|------|------|
| **统率(leadership)** | 30-100 | 带兵数量、军队士气 |
| **武力(force)** | 20-100 | 单挑胜负、近战伤害 |
| **智力(strategy)** | 20-100 | 计策成功率、防御策略 |
| **政治(politics)** | 10-100 | 内政效率、外交效果 |
| **魅力(charisma)** | 10-100 | 招募成功率、部下忠诚 |

### 1.3 武将/文官获取途径（四种）

> **核心规则**：武将和文官的来源有四种途径——奴隶培养、任务奖励、攻掠国家招降、随机招聘。不存在流浪投奔等其他来源。**仅舞女和侍妾可直接培养为武将/文官，其他奴隶必须先转化为公民或士兵，再通过军功或科举事件晋升。**

#### 途径一：奴隶培养（仅限舞女/侍妾直接晋升）

| 奴隶类型 | 培养目标 | 消耗 | 培养期 | 产出属性侧重 |
|----------|----------|------|--------|-------------|
| **侍妾** ♀ | 女文官 | 400金+80粮 | 10回合 | 政治+魅力高，统率+武力低 |
| **侍妾** ♀ | 女武将 | 500金+100粮 | 15回合 | 统率+魅力高，政治+智力中 |
| **舞女** ♀ | 女文官(外交型) | 350金+60粮 | 10回合 | 魅力+政治高，武力+统率低 |

> **其他奴隶（劳奴/匠奴/战奴）** 不可直接培养为武将/文官，必须先转化为公民或士兵，再通过**军功提拔**或**科举选拔**事件晋升。

#### 途径二：任务奖励（主要来源）

**城邦任务奖励武将**：
- 完成城邦故事线后，奖励1-2名武将/文官
- **男女比例**：普通国家5:5，母系社会国家3:7，全女性国家100%女性
- 城邦任务武将/文官属性较高（平均70+），有独特技能

**部落任务奖励武将**：
- 完成部落故事线后，奖励1名武将/文官（随机决定类型）
- **男女比例**：普通国家5:5，全女性国家周边部落100%女性
- 部落武将/文官属性中等（平均50-70），有部落特色能力

#### 途径三：攻掠国家招降（批量来源）

- **征服一国后**：该国武将/文官有30%概率投降归顺
- **招降条件**：赠礼（100-500金）+ 说服（外交值判定）
- **忠诚度**：初始40，需提升至60+才能完全信任
- **属性保留**：招降武将保留原属性的80%，可通过后续培养恢复

#### 途径四：随机招聘（补充来源）

> **核心机制**：随机招聘不是主动操作，而是通过**公民养成事件触发**。

- **触发方式**：
  - **军功提拔**：士兵参与战斗≥5场后，有概率触发「民间英才」事件
  - **科举选拔**：城市等级≥Lv3后，每年春季可举行「科举」
  - **民间推荐**：每回合有基础概率，由武将/文官推荐民间人才
- **触发频率**：大国每年约3人，中等国约2人，小国约2人
- **招聘条件**：城市等级≥Lv2，金库≥200金
- **招聘消耗**：每人100-300金
- **男女比例**：普通国家6:4，母系社会国家3:7，全女性国家100%女性
- **属性范围**：平均45-65（低于任务奖励，高于奴隶培养初期）
- **忠诚度**：初始50（低于其他来源）

### 1.5 武将与文官分工系统

> **核心规则**：武将和文官有明确的职责分工。武将专注于军事领域（3种任务），文官专注于内政领域（5种任务）。**武将不可执行文官任务，文官不可执行武将任务。**

#### 武将可执行任务（3种）

| 任务 | 关键属性 | 效果 | 属性要求 |
|------|----------|------|----------|
| **训练士兵** | 统率+智力 | 士兵经验+10%~30%/月，新兵→精兵速度×2 | 统率≥40 |
| **巡逻护卫** | 武力+智力 | 商路安全+20%~40%，城邦治安+15 | 武力≥40 |
| **侦察谍报** | 智力+魅力 | 获取敌方情报，提前2回合预警入侵 | 智力≥50 |

#### 文官可执行任务（5种）

| 任务 | 关键属性 | 效果 | 属性要求 |
|------|----------|------|----------|
| **内政管理** | 政治+智力 | 粮食产出+10%~30%，金币收入+10%~20%，建设速度+10%~20% | 政治≥40 |
| **外交谈判** | 魅力+政治 | 外交好感+15~30/次，贸易价格优惠5%~15%，联盟成功率+20% | 魅力≥50 |
| **贸易管理** | 魅力+智力 | 商路收入+15%~30%，市场购买价格-5%~15%，出售价格+5%~15% | 魅力≥40 |
| **科技研发** | 智力+政治 | 科技点+5~15/月，解锁新建筑/新兵种/新能力 | 智力≥50 |
| **人口管理** | 政治+魅力 | 人口增长+5%~15%，公民满意度+10%，奴隶转化速度+20% | 政治≥40 |

#### 武将/文官属性侧重分类

| 类型 | 属性侧重 | 适合任务 | 来源 |
|------|----------|----------|------|
| **猛将型武将** | 武力>统率>智力 | 巡逻护卫 | 战奴培养、军事任务 |
| **统帅型武将** | 统率>智力>武力 | 训练士兵 | 战奴培养、城邦任务 |
| **智将型武将** | 智力>统率>武力 | 侦察谍报 | 匠奴培养、探索任务 |
| **内政型文官** | 政治>智力>魅力 | 内政管理、人口管理 | 劳奴/侍妾培养 |
| **外交型文官** | 魅力>政治>智力 | 外交谈判、贸易管理 | 舞女/侍妾培养 |
| **技术型文官** | 智力>政治>魅力 | 科技研发、内政管理 | 匠奴培养 |

#### 武将文官搭配策略

```
早期（人口<500）：1文官（内政管理）+ 1武将（巡逻护卫）
  → 文官提升粮食/金币产出，武将保护商路安全

中期（人口500-2000）：2文官（内政+贸易）+ 2武将（训练+巡逻）
  → 内政文官管粮食，贸易文官管金币
  → 训练武将提升军队质量，巡逻武将保护扩张中的商路

后期（人口2000+）：3~5文官（内政+外交+贸易+科技+人口）+ 2~3武将（训练+巡逻+侦察）
  → 全方位管理，每个文官专注一个领域
  → 武将负责军事后勤保障（训练/护卫/侦察）
```

### 2.2 征兵系统

```
征兵流程：
  1. 确认征兵容量（受民兵营/精锐兵营限制）
  2. 从公民中征召（减少可用劳动力）
  3. 消耗兵器（步兵×2兵器/骑兵×1兵器/弓骑兵×1兵器）
  4. 支付征兵费用（15~35金/人，一次性）
  5. 新兵需要训练时间（武将训练3回合→普通兵）

征兵规则（按国家类型区分）：
  普通国家：只征召男兵
  母系社会国家（6个）：70%男兵+30%女兵（需城市Lv2）
  全女性国家：只征召女兵

女兵战力加成：
  同兵种女兵战斗力比男兵高5%
  女兵纪律性更强、阵亡率略低（-3%）、兵器损耗略低（-0.5%/战）
```

> - 解约时雇佣兵直接消失（不返还费用）

## 三、战略系统

### 3.1 外交系统

外交系统在城市发展阶段（城市Lv2）解锁。

#### 外交操作

| 操作 | 效果 | 费用 | 条件 |
|------|------|------|------|
| **互不侵犯条约** | 双方承诺不攻击 | 100-300金谈判费 | 关系≥中立 |
| **贸易协定** | 双方贸易税率-10% | 100-200金谈判费 | 关系≥中立 |
| **军事同盟** | 共同防御/进攻 | 300-500金谈判费 | 关系≥友好 |
| **联姻** | 关系+30，获得武将 | 聘礼500-2,000金 | 关系≥友好+有合适武将 |
| **朝贡** | 向强国进贡换取保护 | 粮食+金币/月 | 关系≥冷淡 |
| **宣战** | 进入战争状态 | 威望-20 | 城市Lv3+宣战能力 |

#### 联姻系统

```
联姻条件：
  - 双方关系≥友好（61+）
  - 至少一方有可婚配武将（未婚）
  - 支付聘礼（500-2,000金）

联姻效果：
  - 双方关系+30（永久基础加成）
  - 获得对方一名武将
  - 母系国家：联姻由女方主导，效果+50%
  - 全女性国家联姻：需要特殊剧情触发
```

### 4.2 战斗流程

```
每回合战斗流程：

1. 部署阶段
   - 攻方先部署（选择初始位置）
   - 守方后部署（可利用地形优势）

2. 行动阶段（交替进行）
   - 武将指挥：选择武将执行战术动作
   - 部队移动：按兵种移动范围移动
   - 攻击行动：对范围内敌军发起攻击
   - 技能释放：武将使用特殊技能
   - 阵型调整：改变部队阵型

3. 结算阶段
   - 计算双方伤亡
   - 检查士气状态
   - 判断胜负条件

胜负条件：
  - 歼灭敌方全部部队
  - 敌方士气降为0（溃败）
  - 占领敌方大本营/城池
  - 达到回合限制（平局，双方损失）
```

### 4.3 战斗公式

```
伤害计算：
  基础伤害 = 兵种攻击力 × (1 + 武将武力×0.01)
  实际伤害 = 基础伤害 × 克制系数 × 地形系数 × 士气系数 × 随机(0.8-1.2)

防御计算：
  实际减伤 = 兵种防御力 × (1 + 地形防御加成) × (1 + 阵型加成)
  最终伤害 = max(实际伤害 - 实际减伤, 1)

士气系统：
  初始士气 = 100
  士气下降：每损失10%部队 → 士气-10
  士气恢复：武将鼓舞技能 → 士气+20
  士气崩溃：士气≤0 → 全军溃败

  士气影响：
    士气>80：攻击+10%
    士气60-80：正常
    士气40-60：攻击-10%，移动-1
    士气20-40：攻击-20%，移动-2，有概率逃跑
    士气<20：攻击-40%，每回合10%部队逃跑
```

|------|------|----------|----------|
| **方阵** | 防御+15%，移动-1 | 步兵 | 克制冲锋 |
| **锋矢** | 攻击+20%，防御-10% | 骑兵 | 克制散阵 |
| **雁行** | 远程+15%，近战-10% | 弓兵 | 克制方阵 |
| **散阵** | 移动+2，防御-15% | 轻步兵/弓骑 | 克制锋矢 |

| **长蛇** | 移动+3，侧翼脆弱 | 骑兵 | 快速机动 |

### 4.5 攻城系统

```
攻城战特殊规则：

城防等级：
  Lv1 土墙：防御+20%，耐久500
  Lv2 木墙：防御+30%，耐久1,000
  Lv3 石墙：防御+40%，耐久2,000
  Lv4 城堡：防御+50%，耐久5,000
  Lv5 要塞：防御+60%，耐久10,000

攻城武器：
  云梯：步兵登墙，需近战
  攻城锤：对城门伤害×3
  投石车：远程攻击城墙（需木材建造）
  火箭：对木制建筑伤害×2

攻城方劣势：
  - 攻击-20%（仰攻）
  - 每回合额外损失（城墙射击）
  - 需要攻城武器才能有效破城

守城方优势：
  - 防御+20-60%（城墙等级）
  - 弓兵射程+2
  - 可使用城防设施（箭塔、滚石等）

围城：
  - 切断补给线 → 守方每月粮食-20%，士气-5
  - 围城≥6月 → 守方可能投降（概率=围城月数×5%）
```

### 4.6 战斗结果与战利品

```
战斗结果类型：
  大胜：敌方损失>70%，己方损失<20% → 战利品×2
  胜利：敌方损失>50%，己方损失<40% → 正常战利品
  惨胜：敌方损失>50%，己方损失>40% → 战利品×0.5
  平局：双方损失接近 → 无战利品
  失败：己方损失>50% → 损失所有携带资源

战利品计算：
  金币 = 敌方携带金币 × 30-50%
  粮食 = 敌方携带粮食 × 20-40%
  兵器 = 敌方损失兵器 × 10-20%
  俘虏 = 敌方损失兵力 × 10-30%（可转化为奴隶）
  武将俘虏 = 概率俘虏敌方武将

特殊战利品：
  攻占城市 → 获得城市全部资源储备
  灭国 → 获得该国所有武将、资源、领土
```

### 4.7 单挑系统

```
触发条件：
  - 战斗中主动发起（消耗1个行动点）
  - 对方武将接受（AI根据武力差决定）
  - 武力差>30时，弱方可能拒绝

单挑流程：
  回合制，每回合双方交替攻击
  
  攻击伤害 = 武力 × 随机(0.8-1.2) + 武器加成
  防御减伤 = 武力 × 0.3 + 护甲加成
  
  特殊事件：
    - 暴击（5%概率）：伤害×2
    - 闪避（敏捷差×1%）：完全回避
    - 反击（10%概率）：对方攻击后立即反击

单挑结果：
  胜利：对方武将受伤/被俘/被杀
  失败：己方武将受伤/被俘/被杀
  平局：双方受伤，单挑结束

单挑对战局影响：
  胜利方全军士气+15
  失败方全军士气-15
  被杀武将所属部队攻击-20%（1回合）
```

## 六、军事闭环验证

### 6.1 军事-经济闭环

```
经济产出（金币/粮食/铁矿）
    ↓
建设军事设施（兵营/马场/铁匠铺）
    ↓
征召士兵（消耗金币+兵器+人口）
    ↓
军队维护（持续消耗粮食+金币）
    ↓
战争胜利（获得领土/资源/俘虏）
    ↓
经济实力增强（更多产出/部落贡赋）
    ↓
建设更强军事力量（循环上升）

关键平衡点：
  - 军队消耗与经济产出的平衡
  - 兵力规模与人口上限的制约
  - 军事投资与民生发展的取舍
  - 战争收益与战争风险的权衡
```

### 6.2 武将-战斗闭环

```
获取武将（奴隶培养/任务奖励/招降/随机招聘）
    ↓
武将积累经验（月度+战斗+任务）
    ↓
武将升级（属性提升+技能解锁）
    ↓
战斗表现提升（更强指挥+特殊技能）
    ↓
战斗胜利（更多经验+战利品）
    ↓
武将继续成长（循环上升）

关键平衡点：
  - 武将数量与俸禄负担（10~50金/人/月）
  - 武将培养投入与回报
  - 忠诚度管理与人才保留
  - **舞女忠诚度加成**：舞女每月自动贡献+10金币，同时可分配给武将/文官，每月忠诚度+5；也可作为外交赠礼（+5好感）
  - 武将搭配与战术组合
```
