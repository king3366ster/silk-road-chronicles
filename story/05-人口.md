### 1.4 性别比例说明

#### 1.4.1 普通国家性别比例

> 普通国家初始男女比为**55:45**，男性略多于女性，反映古代西域社会的基本人口结构。

- **初始构成**：男公民30% + 女公民30% + 男奴隶15% + 女奴隶15% + 士兵10%
- **征兵规则**：只征召男兵，女公民不可被征召为士兵
- **奴隶购买**：可购买男性和女性各类奴隶，无性别限制
- **自然增长**：新增公民/奴隶性别比默认55%男/45%女
- **女战奴处理**：购买/掠夺/外交获得的女战奴无军事使用价值，可选择转化为女劳奴/女匠奴/舞女（消耗10金+1回合）
- **比例变化**：随游戏发展，玩家可通过购买更多男性/女性奴隶改变性别比，无硬性约束

#### 1.4.2 母系国家性别比例

> 母系国家的核心特征是女性占多数，初始男女比为**30:70**。女性在社会中占据主导地位，男性为辅助角色。

- **初始构成**：男公民18% + 女公民42% + 男奴隶9% + 女奴隶21% + 士兵10%
- **征兵规则**：征召70%男兵+30%女兵，女兵仅限中高级兵士（精锐卫队），需城市等级≥Lv2
- **奴隶购买**：可购买男性和女性各类奴隶，女性奴隶价格-10%，舞女/侍妾产出+15%
- **自然增长**：新增公民/奴隶性别比默认30%男/70%女
- **特殊生育**：可使用圣水受孕、外交通婚、男奴繁殖、侍女/舞女租借、收养孤儿等方式
- **约束机制**：见1.5节详述

#### 1.4.3 全女性国家性别比例

> 全女性国家初始男女比为**0:100**，男性始终为0。所有人口均为女性，是一种极端的母系社会形态。

- **初始构成**：女公民60% + 女奴隶30% + 女士兵10%（男性0%）
- **征兵规则**：只征召女兵，女公民均可被征召入伍
- **奴隶购买**：只能购买/拥有女性奴隶（女劳奴、女匠奴、女战奴、舞女、侍妾）。男性奴隶不可保留，购买后强制立即消耗（用于男奴繁殖或按市价转化为金币）
- **自然增长**：基础月增长率为0%，完全依赖特殊生育方式（圣水受孕、外交通婚等）
- **特殊生育**：可使用圣水受孕、外交通婚、男奴繁殖、侍女/舞女租借、收养孤儿等方式
- **新增人口**：所有新增公民/奴隶均为女性
- **女兵加成**：女兵额外+10%战斗力加成（总加成+20%），但战败惩罚×1.5，兵力恢复速度-25%

### 1.5 母系性别比例约束机制

> 母系国家的核心特征是女性占多数（初始男女比30:70）。随着游戏发展，玩家可能通过各种手段改变性别比例。为维持母系社会的本质特征，设置以下约束：

- **性别比例红线**：当男性人口占比超过40%（即男女比>40:60）时，触发**"母系失衡"**状态
- **失衡效果**：
  - 🚫 **禁止购买男性奴隶**：奴隶市场中的男性奴隶选项被锁定
  - 🚫 **禁止男性奴隶转化**：无法将男性奴隶转化为男性公民
  - 🚫 **男性公民自然增长停止**：人口增长时，新增公民全部为女性
  - ⚠️ **母系文化加成减半**：所有母系文化加成效果降低50%
  - ⚠️ **外交联姻成功率-20%**：母系国家认为你已背离传统
- **恢复条件**：当男性人口占比回落至40%以下（男女比≤40:60）时，自动恢复正常状态
- **设计目的**：防止玩家通过大量购买男性奴隶或转化手段将母系国家变为事实上的普通国家，保持母系国家的独特游戏体验

## 二、文官/武将

> **核心规则**：武将和文官是人口的最高层级，来源有四种途径——奴隶培养、任务奖励、攻掠国家招降、随机招聘。不存在流浪投奔等其他来源。**仅舞女和侍妾可直接培养为武将/文官，其他奴隶必须先转化为公民或士兵，再通过军功或科举事件晋升。**

### 2.1 武将/文官属性

每位武将/文官拥有五项核心属性：

| 属性 | 范围 | 影响 |
|------|------|------|
| **统率(leadership)** | 30-100 | 带兵数量、军队士气 |
| **武力(force)** | 20-100 | 单挑胜负、近战伤害 |
| **智力(strategy)** | 20-100 | 计策成功率、防御策略 |
| **政治(politics)** | 10-100 | 内政效率、外交效果 |
| **魅力(charisma)** | 10-100 | 招募成功率、部下忠诚 |

### 2.2 武将等级与军衔

| 军衔 | 所需等级 | 最大带兵 | 加成 |
|------|----------|----------|------|
| 伍长 | Lv1 | 500 | +0% |
| 什长 | Lv3 | 1,000 | +5% |
| 百夫长 | Lv5 | 2,000 | +10% |
| 千夫长 | Lv8 | 5,000 | +15% |
| 万夫长 | Lv12 | 10,000 | +20% |
| 将军 | Lv15 | 20,000 | +30% |
| 大将军 | Lv20 | 50,000 | +40% |

### 2.3 武将/文官获取途径（四种）

#### 途径一：奴隶培养（仅限舞女/侍妾直接晋升）

| 奴隶类型 | 培养目标 | 消耗 | 培养期 | 产出属性侧重 |
|----------|----------|------|--------|-------------|
| **侍妾** ♀ | 女文官 | 400金+80粮 | 10回合 | 政治+魅力高，统率+武力低 |
| **侍妾** ♀ | 女武将 | 500金+100粮 | 15回合 | 统率+魅力高，政治+智力中 |
| **舞女** ♀ | 女文官(外交型) | 350金+60粮 | 10回合 | 魅力+政治高，武力+统率低 |

> **其他奴隶（劳奴/匠奴/战奴）** 不可直接培养为武将/文官，必须先转化为公民或士兵，再通过**军功提拔**或**科举选拔**事件晋升。

#### 途径二：任务奖励（主要来源）

**城邦任务奖励武将**：
- 完成城邦故事线后，奖励1-2名武将/文官
- **男女比例**：普通国家5:5，母系社会国家3:7，全女性国家100%女性
- 城邦任务武将/文官属性较高（平均70+），有独特技能

**部落任务奖励武将**：
- 完成部落故事线后，奖励1名武将/文官（随机决定类型）
- **男女比例**：普通国家5:5，母系社会国家3:7，全女性国家周边部落100%女性
- 部落武将/文官属性中等（平均50-70），有部落特色能力

#### 途径三：攻掠国家招降（批量来源）

- **征服一国后**：该国武将/文官有30%概率投降归顺
- **招降条件**：赠礼（100-500金）+ 说服（外交值判定）
- **忠诚度**：初始40，需提升至60+才能完全信任
- **属性保留**：招降武将保留原属性的80%，可通过后续培养恢复

#### 途径四：随机招聘（补充来源）

> **核心机制**：随机招聘不是主动操作，而是通过**公民养成事件触发**。

- **触发方式**：
  - **军功提拔**：士兵参与战斗≥5场后，有概率触发「民间英才」事件
  - **科举选拔**：城市等级≥Lv3后，每年春季可举行「科举」
  - **民间推荐**：每回合有基础概率，由武将/文官推荐民间人才
- **触发频率**：大国每年约3人，中等国约2人，小国约2人
- **招聘条件**：城市等级≥Lv2，金库≥200金
- **招聘消耗**：每人100-300金
- **男女比例**：普通国家6:4，母系社会国家3:7，全女性国家100%女性
- **属性范围**：平均45-65（低于任务奖励，高于奴隶培养初期）
- **忠诚度**：初始50（低于其他来源）

## 三、公民

### 3.1 公民人口增长机制

#### 3.1.1 自然增长公式

> 公民人口增长的基础计算公式适用于所有国家，但月增长率因国家类型而异。

```
公民人口月增长 = 当前公民人口 × (公民月增长率 × 城市等级对应比率 × 季节修正 × 剩余粮食修正 + 额外增长率)
```

| 变量 | 说明 | 取值范围 |
|------|------|----------|
| **公民月增长率** | 基础月增长率，按国家类型固定 | 普通1% / 母系0.5% / 全女0% |
| **城市等级对应比率** | Lv1无加成，Lv2+10%，Lv3+15%，Lv4+20%，Lv5+25% | 1.0~1.25 |
| **季节修正** | 春季+10%，夏季+5%，秋季-5%，冬季-20% | 0.80~1.10 |
| **剩余粮食修正** | 粮食盈余>10%时+10%，盈余>30%时+20%，赤字时-10% | 0.90~1.20 |
| **额外增长率** | 圣水受孕/外交通婚等额外获得的月增长率（见3.1.3~3.1.4节） | 0%~1.5% |

#### 3.1.2 国家类型基础月增长率

| 国家类型 | 公民月增长率 | 奴隶月增长率 | 说明 |
|----------|-------------|-------------|------|
| **普通国家** | 1% | 1% | 标准自然增长，不可使用特殊生育方式 |
| **母系国家** | 0.5% | 0.5% | 自然增长率较低，可使用特殊生育方式补充 |
| **全女性国家** | 0% | 0% | 无自然增长，完全依赖特殊生育方式 |

#### 3.1.3 圣水受孕（基础特殊生育方式）

> 母系/全女性国家可通过消耗圣水类资源获得额外月增长率：

| 资源 | 消耗 | 效果 | 说明 |
|------|------|------|------|
| **子母河水** | 每消耗2单位 | 获得+1%月增长率 | 对公民和奴隶各+0.5%，总+1% |
| **瑶池圣水** | 每消耗1单位 | 获得+1%月增长率 | 对公民和奴隶各+0.5%，总+1% |
| **弱水泉水** | 每消耗1单位 | 获得+1%月增长率 | 对公民和奴隶各+0.5%，总+1% |

> **增长率上限**：母系/全女性国家的总月增长率上限为**1.5%**（含圣水受孕+外交通婚叠加）。以上两种方式获得的增长率均为基础增长率，适用于公式中的"额外增长率"变量。

#### 3.1.4 其他特殊生育方式

> 以下6种特殊生育方式中，前5种为全女/母系国家专属（月增长率共享上限**1.5%**），第3种"侍女/舞女租借"为所有国家通用。优先级：**优先消耗外交通婚额度，再消耗圣水额度**。

**方式1：圣水受孕** — 见3.1.3节详述。

**方式2：外交通婚**（仅母系/全女性国家）

| 项目 | 说明 |
|------|------|
| **条件** | 需与目标国外交关系≥友好（60+） |
| **消耗** | 每次消耗金币+外交点数 |
| **效果** | 一个国家的通婚+0.5%月增长率 |
| **上限** | 总月增长率上限1.5%（优先消耗外交通婚额度，再消耗圣水额度） |

**方式3：男奴繁殖**（所有国家通用）

| 奴隶类型 | 消耗 | 产出 | 时间 |
|----------|------|------|------|
| **男劳奴** ♂ | 消耗1个 | 产生10个女公民 | 6个月后 |
| **男匠奴** ♂ | 消耗1个 | 产生10个女公民 | 6个月后 |
| **男战奴** ♂ | 消耗1个 | 产生20个女公民 | 9个月后 |

> 男奴繁殖是一种一次性人口爆发手段，将男性奴隶转化为女性公民，适合急需补充人口的场景。

**方式4：侍女/舞女租借**（所有国家通用）

| 项目 | 说明 |
|------|------|
| **条件** | 在奴隶市场选择"侍女/舞女租借" |
| **租期** | 租借12个月 |
| **产出** | 12个月后产生2个女公民（租借后同时归还侍女/舞女） |
| **限制** | 租借期间不产生任何收益/加成，不可赠送/使用等 |

**方式5：收养孤儿**（被动触发，母系/全女性国家专属）

| 项目 | 说明 |
|------|------|
| **触发条件** | 毗邻国发生交战时自动收养战争孤儿 |
| **效果** | 邻国一个月内每发生过一次交战，月人口增长额外+0.5%（增加女公民，非奴隶） |
| **上限** | 上限不超过1%月增长率 |

> 此类增长适用于公式中的"额外增长率"变量。

### 3.2 公民人口减少机制

> 公民人口的减少来源多样，涵盖战争、制度转型、治安等多个维度。

| 减少来源 | 减少方式 | 影响程度 | 说明 |
|----------|----------|----------|------|
| **征兵** | 公民转为士兵，脱离公民池 | 每次征兵减少对应公民 | 士兵仍属总人口，但不再作为公民产出 |

> - **阿玛宗战败惩罚×1.5**：战败时人口损失+50%
> - **全女国兵力恢复速度-25%**：阵亡后补充更慢

### 4.2 奴隶人口增长机制

#### 4.2.1 自然增长公式

> **核心规则**：奴隶人口增长公式与公民类似，但受侍妾占比加成影响。

```
奴隶人口月增长 = 当前奴隶人口 × 奴隶月增长率 × 城市等级对应比率 × (1 + 侍妾占比加成)
```

| 变量 | 说明 | 取值范围 |
|------|------|----------|
| **奴隶月增长率** | 基础月增长率，按国家类型固定 | 普通1% / 母系0.5% / 全女0% |
| **城市等级对应比率** | Lv1无加成，Lv2+10%，Lv3+15%，Lv4+20%，Lv5+25% | 1.0~1.25 |
| **侍妾占比加成** | = (侍妾数量 ÷ 奴隶总数) × 100% | 0%~100% |

- **侍妾占比加成示例**：
  - 100个奴隶中有10个侍妾 → 加成 = (10/100)×100% = 10% → 月增长系数 = 1.1
  - 100个奴隶中有50个侍妾 → 加成 = (50/100)×100% = 50% → 月增长系数 = 1.5
- **新增奴隶性别分配**：新增奴隶由玩家根据性别分配为不同分工（男劳奴、女匠奴等），性别比例根据国家制度：
  - 普通国家：55%男 / 45%女
  - 母系社会国家：30%男 / 70%女
  - 全女性国家：0%男 / 100%女
- **新增奴隶分工**：玩家确认分工后**不可改变**（见4.4节奴隶分工不可变规则）

#### 4.2.2 其他增长来源

| 增长来源 | 获取方式 | 说明 |
|----------|----------|------|
| **奴隶市场购买** | 12大奴隶市场，花费金币购买 | 主要来源，详见08-发展.md |
| **掠夺获取** | 对敌国领土进行掠夺/征服行动 | 根据掠夺规模获得奴隶（目标国人口×2%~5%），直接加入奴隶人口池 |
| **部落贡赋** | 完成部落故事线后获得 | 每月获得各种资源，可能包含奴隶 |
| **收养孤儿** | 毗邻国交战时被动触发 | 仅增加女公民（非奴隶），月增长额外+0.5%，上限1% |
| **侍女/舞女租借** | 在奴隶市场选择租借 | 12个月后产生2个女公民（同时归还侍女/舞女） |
| **圣水受孕** | 消耗圣水类资源 | 对奴隶人口额外+0.5%月增长率 |

### 5.1 征兵系统

#### 5.1.1 征兵流程

```
征兵流程：
  1. 确认征兵容量（受民兵营/精锐兵营限制）
  2. 从公民中征召（减少可用劳动力）→ **即时完成，无需回合数和金币**
  3. 消耗兵器（步兵×2兵器/骑兵×1兵器/弓骑兵×1兵器）
  4. 普通公民征召后直接成为**普通新兵(Lv0)**

战奴 → 整编 ────────↗               │

                                    ├→ 重骑兵专精 → 突击铁骑（冲锋伤害+25%）
                                    └→ 弓骑兵专精 → 射骑高手（远程+20%，克制+15%）
```

#### 5.3.3 士兵升级条件

|----------|------|------|
| 新兵→普通兵 | 武将训练3回合 | 10金/人 |
| 普通兵→精兵 | 武将训练6回合 + 战斗经验≥2场 | 20金/人 |
| 精兵→老兵 | 武将训练10回合 + 战斗经验≥5场 + 武将统率≥60 | 30金/人 |

|------|----------|
| **长矛兵** | 民兵营Lv2 + 武将统率≥50 |

### 5.4 战奴整编规则

```
战奴 → 编入军队 → 普通兵(Lv1)，战斗力与普通新兵相同
  消耗：1战奴 + 1兵器 + 10金 + **3回合训练期**
  初始属性：战斗力与普通新兵相同，忠诚度初始40
  与普通征兵对比：普通公民征召→普通新兵（**无需回合数和金币**，即时完成）；战奴整编需额外3回合+10金
  与雇佣兵对比：雇佣兵更贵（40~150金/人）但战斗力更强（×0.9~×1.2）且即时可用；战奴更便宜但需训练等待
  忠诚提升：每参与1场胜仗，忠诚+2；每参与1场败仗，忠诚-5；忠诚<30时可能叛逃
```

### 5.5 雇佣兵系统

> **核心机制**：雇佣兵不消耗公民人口，不参与性别比例或人口规模计算，不消耗本国粮食、武器等耗材，仅每月收取金钱。

| 雇佣兵类型 | 雇佣费用 | 月维护费 | 战斗力 | 特殊说明 |
|-----------|----------|----------|--------|----------|
| **疏勒护卫兵** | 50金/人 | 2金/人/月 | ×0.9 | 防御型，适合守城 |
| **草原战奴兵** | 40金/人 | 1.5金/人/月 | ×0.7 | 成本低，可快速暴兵 |
| **大宛天马骑** | 150金/人 | 5金/人/月 | ×1.2 | 精锐骑兵，需天马资源 |
| **草原弓骑手** | 80金/人 | 3金/人/月 | ×1.0 | 远程机动，适合骚扰 |

> **雇佣兵限制**：
> - 雇佣兵总数不超过正规军的30%
> - 雇佣兵忠诚度固定50，不可提升
> - 雇佣兵不可升级/转专精
> - 解约时雇佣兵直接消失（不返还费用）

### 5.6 士兵消耗

| 消耗项目 | 数值 | 频率 | 说明 |
|----------|------|------|------|
| **士兵粮食** | 1.5粮/人/月（新兵/普通兵）或1.8粮/人/月（精兵/老兵） | 每月 | 从公民中征召的士兵 |
| **军费维护** | 兵力×1金/100兵/月 | 每月 | 基础军费 |
| **兵器日常损耗** | 0.5%/月 | 每月 | 所有装备兵器的士兵 |

| **低治安兵器损耗** | +1%/月 | 治安<40时 | 盗窃、破坏 |
| **无铁匠铺额外损耗** | +2%/月 | 无铁匠铺时 | 无法修复 |
| **冬季皮毛消耗** | 1皮毛/兵/月 | 冬季（10-12月） | 无皮毛则士气-20% |
| **征兵消耗** | 1公民+1兵器+15~35金 | 一次性 | 公民脱离生产，兵器装备消耗 |
| **训练消耗** | 10~30金/人 | 一次性 | 新兵→普通兵10金，→精兵20金，→老兵30金 |

### 5.7 士兵损耗机制

| 损耗来源 | 损耗方式 | 影响程度 | 说明 |
|----------|----------|----------|------|
| **战斗阵亡** | 战斗中直接死亡 | 根据战况而定 | 伤害公式见07-战争.md |
| **士气崩溃逃跑** | 士气<20时每回合10%部队逃跑 | 永久失去 | 士气≤0→全军溃败 |
| **伤兵转化** | 战后部分伤兵可恢复 | 10药草恢复100兵 | 无药草则伤兵逐渐死亡 |
| **兵器损耗** | 每月0.5%+冬季+1%+治安+1% | 累积计算 | 无铁匠铺额外+2% |
| **战败惩罚** | 阿玛宗战败人口损失+50% | 特殊国家惩罚 | 全女国兵力恢复速度-25% |

### 5.8 士兵转业

| 转业方式 | 条件 | 说明 |
|----------|------|------|
| **正常退役** | 玩家主动选择 | 消耗1回合重新分配职业 |

| **雇佣兵解约** | 非战斗时解约无惩罚 | 雇佣兵直接消失，不回归公民池 |

## 六、奴隶-公民-武将/文官完整转化机制

```
【完整转化路径图】

奴隶 ──→ 公民 ──→ 士兵(军功) → 武将
  │         │         │
  │         │         └→ 文官(科举) 
  │         └→ 分配职业（农民/矿工/商人/士兵等）
  ├──→ 战奴整编 → 普通兵(Lv1) → 精兵/老兵 → 军功 → 武将
  └→ 分配工作（农场/矿场/纺织/工坊等）

舞女/侍妾 ──→ 武将(直接培养)
            └─→ 文官(直接培养)

【转化路径一：奴隶 → 公民】
条件：奴隶服役≥10回合 + 城市等级≥Lv2
消耗：300金 + 20回合培养期
效果：奴隶变为公民，可分配所有职业

【转化路径二：战奴 → 士兵（整编路线）】
条件：男战奴/女战奴 + 城市等级≥Lv1 + 兵器库有兵器
消耗：1兵器 + 10金/人
效果：战奴直接编入军队成为普通兵(Lv1)
产出：战斗力×0.7，忠诚度初始40

> **普通国家女战奴转化**：普通国家只征男兵，女战奴无法编入军队，可选择转化为**女劳奴/女匠奴/舞女**（消耗10金+1回合）。母系/全女性国家**不可转化**。

【转化路径三：舞女/侍妾 → 武将（直接培养）】
条件：舞女或侍妾 + 城市等级≥Lv3
消耗：450-500金 + 90-100粮 + 25-30回合
效果：变为见习武将(Lv1)

【转化路径四：舞女/侍妾 → 文官（直接培养）】
条件：侍妾/舞女 + 城市等级≥Lv2
消耗：350-400金 + 60-80粮 + 20-25回合
效果：变为见习文官(Lv1)
```

## 七、人口与资源对应关系

### 7.1 人口-资源产出计算公式

```
月度资源总产出 = Σ(公民产出) + Σ(奴隶产出) + 部落贡赋 + 建筑加成

具体计算：
  粮食产出 = 农民(♂×3+♀×2.5) + 男劳奴×3 + 女劳奴×3 + 猎人×1 + 牧民×1
           × 城市等级倍率 × 季节修正 × 地形修正 × 文官加成

  铁矿产出 = 矿工(♂×2+♀×1.5) + 男劳奴×2
           × 城市等级倍率 × 文官加成

  木材产出 = 伐木工(♂×3+♀×2.5) + 男劳奴×3
           × 城市等级倍率

  石材产出 = 采石工(♂×2+♀×1.5) + 男劳奴×2
           × 城市等级倍率

  丝绸产出 = 纺织工(♂×1+♀×2) + 女劳奴×2 + 女匠奴×3
           × 城市等级倍率 × 国家特色加成

  金币收入 = 商人×3(本地) + 商人×5(商路) + 舞女×10 + 税收(人口×0.5)
           × 城市等级倍率 × 季节修正 × 文官加成

月度资源总消耗：
  粮食消耗 = 男公民×1 + 女公民×0.8 + 奴隶×0.6 + 士兵×1.5(精兵/老兵×1.8) + 马匹×2
  盐消耗 = 总人口÷100
  金币消耗 = 军费 + 建设费 + 奴隶购买 + 科技研发
```

### 7.3 人口增长对资源的影响链

```
人口增长 → 劳动力增加 → 资源产出增加 → 但粮食消耗也增加

关键平衡点：
  每新增1名男公民→分配农民→+3粮产出-1粮消耗=净+2粮/月 ✅
  每新增1名女公民→分配农民→+2.5粮产出-0.8粮消耗=净+1.7粮/月 ✅
  每新增1名男劳奴→分配农场→+2.4粮产出-0.6粮消耗=净+1.8粮/月 ✅
  每新增1名士兵→不产出→0-1.5粮消耗=净-1.5粮/月 ❌ 需要其他人养

最佳人口结构（粮食自给率最高）：
  农民45% + 纺织12% + 商业12% + 矿工8% + 伐木5% + 采石3% + 狩猎3% + 其他7% + 士兵5%
  → 粮食自给率≈110%（微盈余）
  → 但军事力量弱

最佳人口结构（军事强国）：
  农民35% + 纺织10% + 商业10% + 矿工8% + 伐木5% + 采石3% + 士兵25% + 其他4%
  → 粮食自给率≈85%（需贸易补足）
  → 军事力量强

结论：人口越多→总产出越高→但需要合理分配职业→粮食是核心瓶颈
```

### 7.4 月度结算顺序

```
每月结算顺序（含消耗类型标注）：

1.  【月度产出】农业/工业/纺织/商业产出 → 资源入库
2.  【月度消耗】男公民粮食(1粮/人) + 女公民粮食(0.8粮/人) + 奴隶粮食(0.6粮/人) + 士兵粮食(1.5~1.8粮/人)
3.  【月度消耗】马匹马料(2粮/匹)
4.  【月度消耗】盐消耗(人口÷100)
5.  【月度消耗】铁匠铺燃料(5木/月)
6.  【月度消耗】兵器日常损耗(0.5%)
7.  【月度消耗】军费维护(兵力÷100金)
8.  【月度消耗】武将/文官俸禄(10~50金/人)
9.  【月度消耗】建筑维护费
10. 【月度收入】税收(人口×0.5金)
11. 【月度收入】部落贡赋(各种资源)
12. 【月度收入】商路收入(金币)
13. 【月度收入】酒吧收入(+5金/酒吧)
14. 【月度消耗】冬季额外消耗(皮毛+兵器损耗)
15. 【一次性】检查是否有建设完成/训练完成/培养完成
16. 【一次性】随机事件触发（可能产生一次性消耗或收入）
```
