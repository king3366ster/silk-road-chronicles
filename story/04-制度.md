> 西域五十国按社会性别结构分为三大国家类型，每种类型有独特的社会结构和运行规则：

## 三、经济制度

### 3.1 六大经济制度体系详解

> **六大经济制度**是所有国家（全女/母系/普通）均可选择的基本经济体制。每种制度适用于特定场景，玩家可随发展阶段切换。不同国家类型在应用同一制度时，因人口结构、文化传统差异，策略重心和实际效果截然不同。

| 1 | **绿洲农耕制** 🌾 | 以农为本，稳中求进 | 农业产出+15%，粮食储备上限+25%，食物损耗-10% | 商业收入-10%，商路收益-15% | 普通农业国 | D-种田发展 |

## 四、军事制度

### 4.1 两大军事制度详解

> 西域五十国均有**2种军事制度**可选：征兵制与募兵制。不同国家类型在应用同一军事制度时，因人口结构与军事传统差异，兵种构成与战术风格截然不同。

| 制度 | 核心机制 | 核心加成 | 核心代价 | 适用场景 |
|------|----------|----------|----------|----------|
| **征兵制** | 公民强制征召，即时成军，消耗人口 | ⚔️征兵速度+30%（紧急动员）；💰军事维护费-20%（公民义务兵成本低） | ⚠️消耗公民人口（征召后脱离生产）；⚠️公民满意度-5%（强制征召不满）；⚠️士兵初始等级Lv0（需训练提升） | 大规模战争、紧急防御、兵力需求大 |
| **募兵制** | 雇佣兵按月付费，不消耗人口，上限80% | 💰不消耗公民人口（生产力不受影响）；⚔️士兵初始等级Lv1（雇佣兵有经验）；🛡️士兵可长期服役（等级提升快） | ⚠️雇佣成本50-100金/人（远高于征兵）；⚠️军队规模上限为人口×80%（雇佣兵有上限）；⚠️忠诚度波动（欠薪时叛变风险+10%） | 小规模精锐作战、长期驻防、人口稀缺 |

## 五、同化决策核心规则

> **同化**是指征服其他国家后，如何处理被征服者的人口与制度归属。以下规则是**强制同化（制度转型·类型一）**的核心成本计算依据。

### 5.1 三种制度与性别比例

> 西域五十国按社会性别结构分为三大国家类型，每种类型有独特的人口构成和制度体系：

| 制度类型 | 性别比例 | 国家数量 | 说明 |
|----------|----------|----------|------|
| **普通制度** | 男女比55:45（普通国） | 36国 | 男性主导，9种政体，征兵制仅男兵 |
| **母系制度** | 男女比30:70（母系国） | 8国 | 女性主导，6种政体，男兵:女兵=8:2~6:4 |
| **全女性制度** | 男女比0:100（全女国） | 6国 | 纯女性，4种政体，仅女兵(战力+10%) |

### 5.2 同化决策矩阵（征服者×被征服者，3×3=9种组合）

> **核心原则**：征服者决定被征服者的制度归属。同族同化免成本；跨族同化需支付成本。

| 征服者类型 | → 普通国(被征服) | → 母系国(被征服) | → 全女国(被征服) |
|-----------|-----------------|-----------------|-----------------|
| **普通国(征服者)** | ①免成本同化（同族同制度） | ②低价同化→普通制：×0.5金/人（跨族但降级） | ③高价同化→普通制：×1金/人（跨族+全女→普通最难） |
| **母系国(征服者)** | ④低价同化→母系制：×0.5金/人（跨族但升级） | ⑤免成本同化（同族同制度） | ⑥低价同化→母系制：×0.5金/人（全女→母系为降级，特权） |
| **全女国(征服者)** | ⑦高价同化→全女制：×1金/人（跨族+普通→全女最难） | ⑧低价同化→全女制：×0.5金/人（母系→全女为升级，特权） | ⑨免成本同化（同族同制度） |

> **成本说明**：
> - **免成本(×0)**：同族同制度同化，无需额外金币
> - **低价(×0.5金/人)**：跨族但制度"顺向"转化（降级或同级转化），成本较低
> - **高价(×1金/人)**：跨族且制度"逆向"转化（升级转化），成本最高
> - **"顺向"vs"逆向"**：普通→母系/全女为"升级"（逆向，高价）；母系→普通为"降级"（顺向，低价）；全女→母系为"降级"（顺向，低价，特权）；全女→普通为"大幅降级"（顺向但跨度大，仍低价）

### 5.3 双制度并行机制

> 玩家可选择**不立即同化**，保留被征服城邦的原有制度，形成"双制度并行"状态。此选择在稳定度与经济效率间取得平衡。

#### 双制度并行成本（金钱+治安双维度）

| 维度 | 两制度并行 | 三制度并行 |
|------|-----------|-----------|
| **月度管理成本（金钱）** | 每座异制度城邦+5金/月 | 每座异制度城邦+8金/月 |
| **稳定度惩罚** | 每座异制度城邦稳定度-3/月 | 每座异制度城邦稳定度-5/月 |
| **叛乱风险** | +2%/月/城邦 | +5%/月/城邦 |
| **外交影响** | 同族城邦好感+5（尊重多样性） | 无特殊加成 |
| **经济效率** | 产出-10%/城邦（制度摩擦） | 产出-20%/城邦（制度冲突） |

#### 双制度并行优势

| 优势 | 说明 |
|------|------|
| **免同化成本** | 不需支付一次性同化金币（省下500~2000金） |
| **保留特产产出** | 被征服城邦的原制度特产产出不受影响（如全女国子母河水仍正常产出） |
| **外交缓冲** | 同族城邦好感+5（尊重多样性） |
| **路线适配** | C路线（万国来朝）和E路线（纵横捭阖）推荐保留制度多样性 |

### 5.4 同化资金不足的后果

| 资金状况 | 后果 |
|----------|------|
| **资金充足** | → 正常同化，城邦纳入正常管理体系 |
| **资金不足（<50%所需）** | → **部分同化**：仅制度层面同化，人口性别比例不变（需6~12个月自然调整），稳定度-10/月持续6个月 |
| **资金严重不足（<10%所需）** | → **同化失败**：城邦进入"失控"状态——稳定度-20/月，叛乱风险+15%/月，产出归零。需3个月内补充资金完成同化，否则城邦独立 |

### 5.5 同化决策与路线适配

| 路线 | 同化策略 | 核心逻辑 |
|------|----------|----------|
| **A·君临天下** | **强制统一同化**——全部同化为自身制度 | 军事扩张不允许异制度存在 |
| **B·富甲丝路** | **保留制度为主**——附庸国保留原制度 | 经济控制大于政治控制 |
| **C·万国来朝** | **保留制度为主**——尊重多样性 | 文化融合优于制度强制 |
| **D·凤鸣昆仑** | **转为母系/全女为主**——推广母系文明圈 | 全女→母系同化成本×0.5（特权） |
| **E·纵横捭阖** | **混合策略**——不同地区不同制度 | 灵活实用，不追求统一 |
| **F·天朝藩屏** | **按宗主意愿**——宗主帝国决定同化方向 | 代主征伐，宗主说了算 |

## 六、制度转型（三种类型）

> 西域五十国的制度转型按触发机制分为**三种独立类型**。三种类型互不冲突，可在不同情境下并行发生。以下为三种类型的总览与详细规则。

### 6.1 三种转型方式对比总览

| 类型 | 名称 | 触发方式 | 人口调整 | 阶段限制 | 触发次数 | 适用对象 |
|------|------|----------|----------|----------|----------|----------|
| **类型一** | 🏛️ **强制同化** | 征服者主动选择（仅一次选择机会） | 温和调整（势力范围）+ 激进调整（被征服国） | 第四阶段后可选择；第四阶段可选双轨；第五阶段强制做出选择 | **1次**（锁定后不可重复） | 征服国自身调整 → 被征服国强制同化 |
| **类型二** | 🌸 **事件触发的主动转型** | 人口比例达标触发女权抬头 | 温和调整（遣散男公民等） | 第四阶段后满足条件均触发 | **多次**（每年至多1次） | 普通国→母系国 / 母系国→全女国 |
| **类型三** | ⚔️ **叛乱与平叛** | 性别比例失衡/外国渗透 | 平叛后严惩（叛乱者→奴隶） | 任何阶段均可触发 | **任意次数** | 母系国/全女国城邦 |

### 6.2 类型一：强制同化（🏛️ Forced Assimilation）

> 强制同化是征服国主动发起的系统性制度转型流程：征服国自身先因接触外界而调整，再对被征服国进行激进同化。**该选择只有一次机会**，且在第四阶段后才可做出。

#### 6.2.1 阶段限制与决策时间线

```
第四阶段（回合73-120）：
  ┌──────────────────────────────────────────────────────┐
  │  可选：双制度并行（保留被征服国原有制度）           │
  │  或：准备强制同化（启动转型流程）                    │
  └──────────────────────────────────────────────────────┘

第五阶段（回合121+）：
  ┌──────────────────────────────────────────────────────┐
  │  强制要求做出最终选择：                              │
  │  → 启动强制同化（执行完整转型流程）                   │
  │  → 或保持双轨制（但承受持续惩罚）                     │
  └──────────────────────────────────────────────────────┘
```

| 阶段 | 选项 | 说明 |
|------|------|------|
| **第四阶段** | ① 双制度并行 | 保留被征服国原有制度，承担月度管理成本 |
| **第四阶段** | ② 准备强制同化 | 启动强制同化流程，触发"征服者自身调整"环节 |
| **第五阶段** | 强制同化 / 双轨制 | 必须做出选择，不可回避 |

> **核心规则**：强制同化的**选择只有一次**。一旦选择"强制同化"，整个转型流程将按固定顺序推进，不可中途停止或回退。若第四阶段未选择强制同化而是选择双轨制，则**在第五阶段仍有机会选择强制同化**——但无论何时选择，都只有这一次机会。

#### 6.2.2 强制同化完整流程

```
强制同化流程（三步走）：

  第一步：征服国自身调整
  ┌───────────────────────────────────────────────────────────┐
  │  触发条件：征服国接触外界（与不同制度的国家/商路接触）   │
  │  效果：征服国自身可能发生制度转变：                      │
  │  · 全女国→母系国（凤归母巢）                            │
  │  · 母系国→普通国（父权复辟）                            │
  │  · 母系国→全女国（女儿当国）                            │
  │  · 普通国→母系国（凤鸣觉醒）                            │
  │  · 也可不变（维持原有制度）                              │
  └───────────────────────────────────────────────────────────┘
                         ↓
  第二步：针对势力范围的温和调整
  ┌───────────────────────────────────────────────────────────┐
  │  对象：征服国原有势力范围（已占领或同盟的自身领土）      │
  │  调整方式：                                              │
  │  · 遣散男公民（补偿金，自愿原则）                        │
  │  · 转业男兵（转为民生产业）                              │
  │  · 调整人口性别比例以适应新制度                          │
  │  性质：温和、有序、合法化                                  │
  └───────────────────────────────────────────────────────────┘
                         ↓
  第三步：对被征服国的激进同化
  ┌───────────────────────────────────────────────────────────┐
  │  对象：被征服国家（通过军事占领纳入版图的国家）          │
  │  调整方式：                                              │
  │  · 强制将被征服国男公民转为男奴                          │
  │  · 男奴处理：贩卖（换取金币）或繁殖（增加劳动力）        │
  │  · 强制推行征服国现行制度（即自身调整后的制度）          │
  │  性质：激进、强制、不可逆                                  │
  └───────────────────────────────────────────────────────────┘
```

> **关键原则**：征服国自身调整后的制度**决定了强制同化的最终模板**——被征服国最终被强制同化为征服国调整后的制度，而非原来的制度。

#### 6.2.3 征服国自身调整规则

> 征服国接触外界后，可能因文化交融、外交影响或内部压力而发生制度转变。此调整为**可选**——征服国可以选择转变，也可以选择不变。

**自身调整方向表：**

| 征服国当前类型 | 可转变为 | 转型事件 | 触发条件 |
|---------------|----------|----------|----------|
| **全女性国家** | 母系国家 | 💧 凤归母巢 | 与母系/普通国接触 + 稳定度>70 |
| **母系国家** | 普通国家 | 🔄 父权复辟 | 与普通国接触并受到影响 + 稳定度>70 |
| **母系国家** | 全女性国家 | 🔥 女儿当国 | 与全女国接触 + 稳定度≥70 + 公民男性比例<10%持续36回合 |
| **普通国家** | 母系国家 | 🌸 凤鸣觉醒 | 与全女/母系国接触 + 稳定度≥70 + 公民女性比例>70%持续36回合 |
| **任何类型** | 不变 | — | 无需条件（默认选项） |

> **自身调整的性质**：此为征服国"因接触外界而发生的自然演变"，属于制度转型中的**内部调整**阶段。此调整本身为温和的、有序的，不影响被征服国的当前状态。

#### 6.2.4 温和调整阶段（势力范围）

> 当征服国完成自身制度调整后，立即进入"势力范围温和调整"阶段。此调整**仅针对征服国原本的势力范围**（自身领土、核心城邦、长期同盟国），不包括后来征服的国家。

**温和调整处理表：**

| 转型方向 | 人口处理 | 军队处理 | 稳定度 |
|----------|----------|----------|--------|
| **普通→母系**（凤鸣觉醒） | 男性公民可遣散（补偿金）或保留公民身份但不可为统治者；女性获得政治权利 | 男兵可遣散（补偿金）或转业；女兵从精锐卫队中征召 | →100 |
| **母系→普通**（父权复辟） | 男性恢复政治权利；女性保留公民身份但失去统治权 | 女兵→转业成女公民；军队恢复全男兵 | →100 |
| **母系→全女**（女儿当国·事件触发） | 男性公民可遣散（补偿金）或自愿外迁 | 男兵可遣散（补偿金） | →100 |
| **全女→母系**（凤归母巢） | 恢复男性公民身份（通过收养/移民/男奴转化） | 恢复男兵征召；军队恢复8:2~6:4 | →100 |

> **温和调整的特点**：所有调整均以"补偿金+自愿/合法"方式进行，不涉及强制人身处置。稳定度恢复至100。

#### 6.2.5 激进同化阶段（被征服国）

> 当征服国完成自身制度调整和势力范围温和调整后，进入"对被征服国的激进同化"阶段。此阶段**针对所有通过军事征服纳入版图的被征服城邦**，执行强制性的制度与人口双重同化。

**激进同化执行表：**

| 征服国新制度 | 被征服国原有制度 | 人口处置 | 制度处置 |
|-------------|-----------------|----------|----------|
| **母系制度** | 普通制度 | 男公民→男奴；男奴可选贩卖（50金/人）或用于繁殖 | 转为对应母系变体，强制推行女性主导制度 |
| **母系制度** | 全女性制度 | 无男性公民，无需处置；女公民制度适应 | 转为母系变体（恢复男性公民身份） |
| **全女性制度** | 普通制度 | 男公民→男奴；男奴可选贩卖（60金/人）或用于繁殖；女公民→女公民 | 转为对应全女变体，强制推行全女性制度 |
| **全女性制度** | 母系制度 | 男公民→男奴；男奴可选贩卖（50金/人）或用于繁殖；女公民→女公民 | 转为对应全女变体，男性全部清除 |
| **普通制度** | 母系制度 | 男性恢复公民身份；女性保留公民身份但失去统治权 | 转为对应普通变体 |
| **普通制度** | 全女性制度 | 男性移民进入；女性保留公民身份 | 转为对应普通变体 |

> **成本计算**：激进同化阶段仍需按[5.2节同化决策矩阵](#52-同化决策矩阵征服者被征服者339种组合)支付同化成本（×0.5或×1金/人）。

**男奴处理选项：**

| 选项 | 收益 | 后果 |
|------|------|------|
| **💲 贩卖** | 50~60金/人（按市场价格） | 一次性金币收入；劳动力流失 |
| **🔧 繁殖** | 男奴每月产出50%纳入奴隶体系 | 持续劳动力产出；需维护成本（1粮/人/月） |

#### 6.2.6 锁定规则

> 一旦完成强制同化流程，以下内容**永久锁定**：

| 锁定内容 | 说明 |
|----------|------|
| **征服国自身制度** | 不可再通过任何方式转型（不可再触发凤鸣觉醒/女儿当国/父权复辟/凤归母巢） |
| **被征服国制度** | 被强制同化后不可再改变制度 |
| **强制同化选择** | 仅此一次，不可重复选择 |

> **例外**：如果征服国在自身调整阶段选择了"不变"，则视为完成强制同化流程但未改变自身制度——此后自身制度仍然锁定，但被征服国的激进同化仍按征服国当前制度执行。

### 6.3 类型二：事件触发的主动转型（🌸 Event-Triggered Active Transformation）

> 当普通国或母系国的人口性别比例达到或接近母系国/全女国的标准时，女权力量抬头，触发主动制度转型。此转型**与是否选择过强制同化完全无关**，可独立发生。

#### 6.3.1 触发条件

| 触发情况 | 当前类型 → 目标类型 | 人口条件 | 其他条件 |
|----------|---------------------|----------|----------|
| **女权抬头 → 母系化** | 普通国 → 母系国 | 女性公民比例 ≥ 65%（接近母系国70%标准） | 稳定度≥50 |
| **极端女权 → 全女化** | 母系国 → 全女国 | 男性公民比例 ≤ 15%（接近全女国0%标准） | 稳定度≥50，与全女国有接触 |

> **人口比例计算**：以势力**全部城邦公民人口**合并计算，不包括奴隶和士兵。

#### 6.3.2 触发效果

> 一旦触发，玩家可选择接受或拒绝转型。接受则立即执行温和人口调整，拒绝则进入冷却期（12个月内不可再次触发同一事件）。

**接受转型的效果表：**

| 转型方向 | 人口调整 | 制度调整 | 稳定度 |
|----------|----------|----------|--------|
| **普通→母系**（凤鸣觉醒） | **温和调整**：男性公民可遣散（补偿金，200金/千人）或保留公民身份但不可为统治者；女性获得全部政治权利 | 转为对应母系政治制度变体；经济制度不变 | →100 |
| **母系→全女**（女儿当国） | **温和调整**：男性公民可遣散（补偿金，200金/千人）或自愿外迁至邻国；不涉及强制转奴隶 | 转为对应全女政治制度变体；经济制度不变 | →100 |

> **与强制同化的区别**：此处的人口调整始终为温和方式（补偿金+自愿），绝不对被征服国执行激进同化。

#### 6.3.3 触发频率与独立规则

| 规则 | 说明 |
|------|------|
| **触发频率** | 每年至多触发1次（满足条件后每年判断一次） |
| **与强制同化的关系** | **完全独立**——已选择过强制同化的国家仍可因人口比例达标而触发主动转型，不受任何影响 |
| **城堡国籍限制** | 仅限当前玩家势力（征服国自身），不作用于被征服国 |
| **多次触发** | 可多次触发：如普通国→母系国（凤鸣觉醒）→人口继续变化→母系国→全女国（女儿当国） |

> **设计意图**：主动转型反映的是社会内部因人口结构变化而自然发生的制度演进，不受外部征服政策的干扰。即使选择了强硬的强制同化路线，社会内部的人口变化仍可能推动制度自我革新。

### 6.4 类型三：叛乱与平叛（⚔️ Rebellion and Suppression）

> 母系国或全女国内部因性别比例失衡，或受到外国势力渗透，可能引发叛乱。此类型为**被动触发**，玩家需及时应对。

#### 6.4.1 性别比例失衡引发叛乱

| 触发条件 | 说明 |
|----------|------|
| **母系国城邦** | 公民/奴隶男女比持续>6:4（男性比例超过正常母系国标准）+ 稳定度<30 持续3回合以上 |
| **全女国城邦** | 不允许男女比例失衡（男性始终为0），但若因特殊事件引入男性且未及时处置，稳定度-20/月直至处置完成 |

**失衡引发叛乱的后果：**

| 阶段 | 效果 |
|------|------|
| **初期（稳定度下降阶段）** | 稳定度-10/月，产出-15%，叛乱风险+10%/月 |
| **中期（骚乱阶段）** | 稳定度<20 → 城邦产出归零，士兵叛变概率20%/月 |
| **爆发（独立阶段，稳定度<0）** | 城邦以**普通国**身份独立（脱离玩家控制），不可逆 |

> **特殊机制**：若独立城邦为全女国境内叛乱，因全女国无男性公民，叛乱由外部渗透的男性奴隶/雇佣兵发动。

#### 6.4.2 外国渗透引发叛乱

> 母系国或全女国收到外国势力渗透（外交、谍报、宗教传播等），可能引发内部叛乱。

| 触发条件 | 说明 |
|----------|------|
| **母系国** | 外国传教士/商旅/间谍活动持续6回合以上 + 该城邦稳定度<50 + 与外国好感度>60（外国在该城邦影响力大） |
| **全女国** | 同上条件，但渗透难度+20%（全女国封闭性强） |

**渗透引发叛乱的后果：**

| 阶段 | 效果 |
|------|------|
| **初期（渗透阶段）** | 城邦稳定度-5/月，外国在该城邦影响力+5/月 |
| **中期（酝酿阶段）** | 稳定度<40 → 城邦产出-10%，出现亲外国派系，加入外国概率+5%/月 |
| **爆发（叛变阶段，稳定度<10）** | 城邦脱离玩家控制并加入渗透国（若渗透国为普通国，城邦制度转为普通制） |

#### 6.4.3 平叛规则

> 当叛乱发生时（无论类型），玩家可选择派兵平叛。

| 平叛行动 | 条件 | 效果 | 后果 |
|----------|------|------|------|
| **武装平叛** | 派遣军队进入叛乱城邦 | 平定叛乱需要兵力≥叛乱势力兵力×1.5 | 胜利：城邦恢复，叛乱男性→男奴，稳定度→70；失败：城邦独立 |
| **安抚谈判** | 派遣魅力≥60的武将/文官 | 谈判成功率=武将魅力×0.5% + 城邦稳定度×0.3% | 成功：城邦恢复，叛乱者遣散，稳定度→80；失败：叛乱继续，稳定度再-10 |
| **外部援助** | 请求同盟国/宗主国派兵协助 | 援助兵力=同盟国总兵力×20% | 成功：城邦恢复，同盟国好感+10；失败：同盟国好感-20 |

**平叛成功后的处置规则：**

| 叛乱类型 | 处置方式 | 稳定度恢复 |
|----------|----------|-----------|
| **性别比例失衡叛乱** | 调整人口比例（遣散多余男性/引入女性人口），叛乱首恶→奴隶 | →70 |
| **外国渗透叛乱** | 关闭该城邦对外交流3个月，驱逐渗透势力，恢复制度 | →60（额外惩罚，因已受外部影响） |

> **防止复发**：叛乱平息后，若根本原因未解决（性别比例未调整/渗透势力未清除），该城邦在12个月后可能再次触发同类型叛乱。

### 6.5 转型事件索引速查
