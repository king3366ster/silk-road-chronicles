### 1.3 月度结算顺序

```
每月结算顺序（含消耗类型标注）：

1.  【月度产出】农业/工业/纺织/商业产出 → 资源入库
2.  【月度消耗】男公民粮食(1粮/人) + 女公民粮食(0.8粮/人) + 奴隶粮食(0.6粮/人) + 士兵粮食(1.5~1.8粮/人)
3.  【月度消耗】马匹马料(2粮/匹)
4.  【月度消耗】盐消耗(人口÷100)
5.  【月度消耗】铁匠铺燃料(5木/月)
6.  【月度消耗】兵器日常损耗(0.5%)
7.  【月度消耗】军费维护(兵力÷100金)
8.  【月度消耗】武将/文官俸禄(10~50金/人)
9.  【月度消耗】建筑维护费
10. 【月度收入】税收(人口×0.5金)
11. 【月度收入】部落贡赋(各种资源)
12. 【月度收入】商路收入(金币)
13. 【月度收入】酒吧收入(+5金/酒吧)
14. 【月度消耗】冬季额外消耗(皮毛+兵器损耗)
15. 【一次性】检查是否有建设完成/训练完成/培养完成
16. 【一次性】随机事件触发（可能产生一次性消耗或收入）
```

### 1.4 关键经济指标

```
月度净收入 = 月度总收入 - 月度总消耗
  月度总收入 = 税收 + 商路收入 + 部落贡赋 + 酒吧收入 + 舞女收入
  月度总消耗 = 粮食消耗 + 盐消耗 + 军费 + 俸禄 + 建筑维护 + 燃料

一次性储备金 = 当前金库 - 预计一次性支出
  预计一次性支出 = 建设预算 + 征兵预算 + 外交预算 + 购买预算

健康经济标准：
  月度净收入 > 0（每月盈余）
  一次性储备金 > 月度总消耗×3（至少3个月缓冲）
  粮食储备 > 月度粮食消耗×6（至少6个月口粮）
```

#### 贸易市场规则

| 规则 | 说明 |
|------|------|
| **市场准入** | 需与市场所在城邦建立外交关系（中立以上）才能交易 |
| **距离浮动** | 买入价 = 基础价 × (1 + 距离浮动率0%~30%)，越远越贵 |
| **供应上限** | 每个市场每月各类商品有收购/出售上限，防止批量倾销 |
| **区域特产限制** | 区域独特特产和专属特产**仅在城邦所在区域的贸易市场出售**（见03-资源.md 三、节） |
| **季节波动** | 秋季粮食价格-10%（丰收），冬季皮毛价格+20%（需求增加） |
| **商路加成** | 建立商路连接两个市场后，两市场间贸易价格-10% |

#### 奴隶市场规则

| 规则 | 说明 |
|------|------|
| **市场准入** | 需与市场所在城邦建立外交关系（中立以上）才能购买奴隶 |

| **月供应上限** | 每个市场每月各类奴隶有供应上限（大国市场30~50人/类，中等国市场15~25人/类） |

| **全女国男奴繁殖** | 全女国无男性人口，购买男奴后**强制立即消耗**：消耗1男劳奴/男匠奴，6个月后产出10个女公民；消耗1男战奴，9个月后产出20个女公民。男奴不可用于其他用途 |
| **阿玛宗奴市特权** | 全女/母系国在阿玛宗女武奴市购买**任何女奴**价格额外-10%（与全国优惠叠加） |

| **疏勒特权** | 疏勒为西域最大奴隶市场，全品类供应且价格-5%，每月供应量+30% |
| **区域差异** | 天山北市场男奴多女奴少；南道市场女奴比例较高；北道市场最均衡 |

### 2.3 跨区域贸易与商路系统

> **核心机制**：玩家通过建立**商路**连接不同区域的市场，实现跨区域资源流通和特产运输。

| 商路类型 | 连接方式 | 建立成本 | 效果 |
|----------|----------|----------|------|
| **区域内商路** | 同区域两个市场之间 | 200金+2马匹 | 两市场间贸易价格-10%，每月自动运输标准资源 |
| **跨区域商路** | 不同区域两个市场之间 | 500金+5马匹+3骆驼 | 两市场间贸易价格-10%，可运输区域特产（运输成本+20%） |
| **丝路主干商路** | 连接≥3个区域的超级商路 | 1000金+10马匹+5骆驼 | 全部连接市场贸易价格-15%，特产运输成本降至+10% |

**商路收入公式**：
```
每月商路收入 = 基础收入(50金) × 商路等级 × 市场等级系数 × 季节修正
  · 商路等级：区域内×1.0，跨区域×1.5，丝路主干×2.0
  · 市场等级系数：★★★★×1.2，★★★×1.0
  · 季节修正：夏季+10%，冬季-20%
```

## 三、经济制度体系

> **六大经济制度** 是所有国家（全女/母系/普通）均可选择的基本经济体制。每种制度适用于特定场景，玩家可随发展阶段切换。不同国家类型在应用同一制度时，因人口结构、文化传统差异，策略重心和实际效果截然不同。

### 3.1 六大经济制度总览

| # | 制度 | 核心思想 | 核心加成 | 核心代价 | 适合国家类型 | 优先路径 |
|---|------|----------|----------|----------|-------------|----------|
| 1 | **绿洲农耕制** 🌾 | 以农为本，稳中求进 | 农业产出+15%，粮食储备上限+25% | 商业收入-10%，商路收益-15% | 普通农业国 | D-种田发展 |

### 3.2 制度1：绿洲农耕制 🌾

#### 核心机制

| 项目 | 内容 |
|------|------|
| **适用场景** | 开局稳定发展、粮食安全优先、备战扩张前积累期 |
| **核心加成** | 农业产出+15%（含农民、农场奴隶）、粮仓容量+25%、食物损耗-10% |
| **核心代价** | 商业产出-10%、商路收入-15%、市场购买力-5% |
| **最佳路径** | D-种田发展 → 后期可转 A/B/C |

#### 3.2.1 普通国家应用

> **代表城邦**：焉耆⚒️、姑墨🍇、婼羌🏹、扜弥🍷

| 维度 | 策略详解 |
|------|----------|
| **人口分配** | 农业55%（男农民为主）+ 矿工8% + 伐木8% + 手工业12% + 商业7% + 士兵10%（男兵） |
| **核心循环** | 男农民(3粮/月)→粮食盈余→养更多男兵→军事扩张或人口增长→更多农民→正循环 |
| **性别优势利用** | 男公民占55%，农业效率最大化（男农民+3 vs 女农民+2.5）；重体力职业（矿工/伐木）全由男性承担 |
| **奴隶策略** | 购买男劳奴(50金)为主分配农场，快速拉升粮食产出；少量女匠奴用于纺织辅助 |
| **出口策略** | 粮食盈余可用于酿造美酒(2粮→1美酒)出口，或直接贸易换取金币和铁矿 |
| **转型时机** | 人口>5000且粮食盈余>1000/月时，可转商路贸易制或军事征伐路径 |

**典型发展曲线**（焉耆为例）：
```
回合1-20：农业55% → 粮食盈余800/月 → 人口4000+
回合21-50：建灌溉渠+粮仓升级 → 粮食盈余2000/月 → 转产纺织/酿酒
回合51+：粮食贸易换金币 → 建设商业建筑 → 可转型商路贸易制
```

#### 3.2.2 母系国家应用

> **代表城邦**：瑶池国🌸、月氏国🌙、赤谷国🔥

| 维度 | 策略详解 |
|------|----------|
| **人口分配** | 农业50%（女农民为主）+ 矿工6%(男奴替代)+ 伐木5%(男奴替代)+ 手工业18%(女纺+采药)+ 商业10%+ 士兵11%(70%男兵+30%女兵) |
| **性别劣势弥补** | 女农民(+2.5粮)效率为男农民83%，需多配10-15%农业人口补足缺口；矿工/伐木工由男劳奴替代（男性奴隶价格+5%但可接受） |
| **核心特点** | 纺织业天然优势（女纺织+2 vs 男纺织+1），采药业优势（女采药+2 vs 男采药+1），粮食自给率略低于普通国但手工业附加值更高 |
| **奴隶策略** | 男劳奴分配矿场/伐木场弥补重体力缺口；女劳奴分配农场+纺织双用途；侍妾占比控制在5-10%以维持人口增长 |
| **出口策略** | 丝绸+药草为高附加值出口品，用出口收入进口粮食弥补缺口；粮食依赖度可容忍至70%自给 |
| **人口增长** | 月增长率0.5%（自然）+ 圣水受孕/外交通婚（最高可追至1.5%），增长速度慢于普通国，需珍惜每1个人口 |

**典型发展曲线**（瑶池国为例）：
```
回合1-20：农业50%+手工业18% → 粮食盈余500/月 + 丝绸/药草150金/月

回合51+：可用瑶池圣水出口(180金/瓶)+外交联姻 → 人口增长追平 → 转特色精工制更优
```

| **②商路贸易制** 🏪 | ✅**首选** | ⚠️可行（金换铁） | ✅最佳补充 | ❌冲突 | ✅协同 |
| **③游牧畜牧制** 🐏 | ⚠️可行（马队商路） | ✅**首选** | ❌冲突 | ❌冲突 | ❌冲突 |
| **④庄园贡赋制** 🏰 | ⚠️可行（贡赋贸易） | ✅最佳补充 | ❌冲突 | ✅**首选** | ⚠️可行 |
| **⑤国家专营制** 🏛️ | ✅协同（控制流通） | ✅可行（军工业） | ⚠️可行 | ✅协同 | ✅**首选** |
| **⑥特色精工制** 💎 | ✅协同（特产贸易） | ❌冲突 | ✅最佳补充 | ❌冲突 | ✅**首选** |

**推荐组合**：
- **普通大国**（楼兰/疏勒/龟兹）：②商路贸易制 + A-商路霸业 = 经济起飞最快
- **普通军事国**（乌孙/车师/危须）：③游牧畜牧制 + B-军事征伐 = 骑兵无敌
- **母系中等国**（瑶池国/玄女国）：⑤国家专营制 + E-特色精专 = 小即强大
- **全女小国**（弱水国/素女国）：⑥特色精工制 + E-特色精专 = 弯道超车
- **全女大国**（西梁女国/东女国）：②商路贸易制→后期转⑤国家专营制 = 经济霸权
- **母系游牧国**（月氏国）：③游牧畜牧制 + 后期转⑤国家专营制 = 先征服再垄断

### 3.9 经济制度阶段演进策略

> 同一国家在不同发展阶段可采用不同经济制度，以下为三类国家的**推荐制度演进路线**：

#### 普通国家（以疏勒为例）

```
阶段一(1-36回合)：①绿洲农耕制 → 扎根基础
  ★ 农业+15%确保粮食安全，在保护期内打好物质基础

阶段二(37-72回合)：②商路贸易制 → 经济起飞
  ★ 保护期结束前储备3000金+，切换商路制后立即开辟3条商路

阶段三(73-120回合)：④庄园贡赋制 或 ⑤国家专营制 → 规模扩张
  ★ 吞并小国后奴隶增多，庄园制放大红利；或控制疏勒市场后国家专营控价
```

#### 母系国家（以瑶池国为例）

```
阶段一(1-36回合)：①绿洲农耕制(微调版) → 保粮+手工业
  ★ 农业50%+手工业18%，女农+女纺双线，用丝绸出口补粮食缺口

阶段二(37-72回合)：⑤国家专营制 → 瑶池圣水商业化
  ★ 瑶池圣水(2单位/月/180金)专营后品质提升+售价+15%=230金/瓶
  ★ 圣水外交赠礼(+45好感)打开外交局面

阶段三(73-120回合)：②商路贸易制 → 圣水+丝绸+玉石三线贸易
  ★ 瑶池圣水专营已成壁垒，商路制进一步放大贸易收入
  ★ 最终形成：瑶池圣水(外交)+丝绸(常规贸易)+玉石(高价值)三引擎
```

#### 全女性国家（以西梁女国为例）

```
阶段一(1-36回合)：⑥特色精工制 → 特产为王
  ★ 保护期内聚焦子母河水+天衣丝绸精工生产
  ★ 子母河水70%自用(人口增长)+30%出口(启动资金)

阶段二(37-72回合)：②商路贸易制 → 贸易网络
  ★ 特产通过商路辐射西域，建立3条跨区域商路确保粮食进口
  ★ 天衣丝绸(45好感)定向赠予产粮大国换取粮食协议

阶段三(73-120回合)：⑤国家专营制 → 垄断定价
  ★ 子母河水国家专营：定价权控制+外交禁运武器
  ★ 最终西梁女国成为：人口输出国+丝绸品质标杆+外交枢纽
```

**制度演进总原则**：
| 国家类型 | 前中期(阶段一~三) | 中后期(阶段四~五) | 终局(阶段六) |
|----------|-----------------|-----------------|-------------|
| **普通国** | ①/② → 稳定发展商路 | ②/④ → 规模经济 | ②/⑤ → 经济霸权 |
| **母系国** | ①/③ → 弥补人口短板 | ⑤/② → 特色垄断+贸易 | ⑤ → 完全专营控制 |
| **全女国** | ⑥ → 精工起步 | ② → 贸易网络 | ⑤ → 终极垄断 |

## 四、发展路径系统

### 3.1 五大发展路径总览

| 路径 | 核心资源 | 核心优势 | 适合国家类型 | 难度 |
|------|----------|----------|-------------|------|
| **A-商路霸业** 🏪 | 金币+商路 | 经济崛起最快，外交影响力强 | 楼兰、疏勒、龟兹 | ★★★☆☆ |
| **B-军事征伐** ⚔️ | 铁矿+兵器+人口 | 征服效率最高，扩张速度最快 | 乌孙、车师、危须 | ★★★★☆ |
| **C-文化融合** 🕌 | 玉石+丝绸+香料 | 外交降服，不战而屈人之兵 | 于阗、龟兹、瑶池 | ★★★☆☆ |
| **D-种田发展** 🌾 | 粮食+木材+石材 | 内政稳固，厚积薄发 | 焉耆、婼羌、扜弥 | ★★☆☆☆ |
| **E-特色精专** 💎 | 专属特产+科技 | 以小博大，弯道超车 | 全女国、小国、中等国 | ★★★★★ |

### 3.2 A-商路霸业发展路径

```
核心思想：以商立国，以路通天下
```

**阶段①：商路奠基（0-20回合）**
- 优先建设：集市(Lv1)+仓库+驼厩
- 核心目标：与3个以上城邦建立外交关系
- 关键资源：金币(储备300+)、马匹(至少5匹)、骆驼(至少3匹)
- 人口分配：商业15%+农业45%+手工业20%

**阶段②：商路扩展（20-50回合）**
- 优先建设：商路驿站+市场升级(Lv2)+金库
- 核心目标：建立至少2条跨区域商路
- 关键资源：金币(持续收入>200/月)、特产商品(丝绸/玉石/美酒)
- 人口分配：商业25%+农业35%+手工业25%

**阶段③：商路霸权（50+回合）**
- 优先建设：丝路主干商路+疏勒贸易特许+金库升级(Lv3)
- 核心目标：控制≥3条丝路主干，形成贸易网络
- 关键资源：月收入>1000金，外交关系遍布西域
- 特色能力：**经济制裁**（封锁敌国商路）、**贸易垄断**（控制特定商品）

### 3.3 B-军事征伐发展路径

```
核心思想：铁与血，征服西域
```

**阶段①：军备积累（0-20回合）**
- 优先建设：民兵营+铁匠铺+兵器坊
- 核心目标：储备兵器≥50，兵力≥200
- 关键资源：铁矿(月产>20)+木材(月产>15)+石材(月产>10)
- 人口分配：农业45%+矿工10%+军事15%

**阶段②：征伐扩张（20-50回合）**
- 优先建设：精锐兵营+城墙+马场
- 核心目标：征服1-2个小国，兵力扩张至500+
- 关键资源：兵器储备>200，马匹(骑兵路线)
- 人口分配：农业35%+矿工12%+军事25%

**阶段③：军事霸权（50+回合）**
- 优先建设：要塞+军营升级(Lv3)+军械库
- 核心目标：拥有精锐军团(精兵/老兵比例>30%)，可同时对2国作战
- 关键资源：月兵器产出>30，兵力>1500
- 特色能力：**强制臣服**（征服后直接附庸）、**恐惧外交**（军威碾压）

### 3.4 C-文化融合发展路径

```
核心思想：以德服人，文化同化
```

**阶段①：文化积累（0-20回合）**
- 优先建设：寺庙+学堂+使馆
- 核心目标：与3个以上城邦建立友好关系(好感>60)
- 关键资源：丝绸(储备20+)+玉石(储备10+)+香料(储备5+)
- 人口分配：农业45%+手工业20%+商业15%

**阶段②：外交网络（20-50回合）**
- 优先建设：文化广场+外交使馆+图书馆
- 核心目标：与≥5个城邦结盟或联姻，好感>80
- 关键资源：月入选特产(子母河水/天衣丝绸等)收入稳定
- 人口分配：农业35%+手工业25%+商业20%

**阶段③：文化霸权（50+回合）**
- 优先建设：西域学宫+博物馆+大剧院
- 核心目标：文化影响力覆盖≥10个城邦，可通过外交降服小国
- 关键资源：月度文化点数>50，外交好感覆盖全域
- 特色能力：**文化同化**（降服城邦同化速度+50%）、**联姻网络**（多国联姻联动）

### 3.5 D-种田发展路径

```
核心思想：广积粮，缓称王
```

**阶段①：基础建设（0-20回合）**
- 优先建设：农田+灌溉渠+粮仓+木材场+石材场
- 核心目标：粮食月盈余>500，基础资源储备充足
- 关键资源：粮食(储备>1000)+木材(>200)+石材(>100)
- 人口分配：农业50%+伐木10%+采石8%+其他32%

**阶段②：产业升级（20-50回合）**
- 优先建设：纺织工坊+铁匠铺+酿酒坊+城市升级(Lv3)
- 核心目标：建立完整产业链，加工资源比例>40%
- 关键资源：月收入>300金，粮食盈余>1000/月
- 人口分配：农业40%+手工业25%+商业15%

**阶段③：厚积薄发（50+回合）**
- 优先建设：所有建筑Lv3+城墙升级+人口上限提升
- 核心目标：总人口>15000，储备金>10000，粮食储备>30000
- 关键资源：全方位自给自足，对外依存度<20%
- 特色能力：**灾年无虞**（粮食储备可撑3年）、**爆发式扩张**（储备充足，可快速转军事）

### 3.6 E-特色精专发展路径

```
核心思想：一招鲜，吃遍天
```

**阶段①：特色聚焦（0-20回合）**
- 优先建设：本国专属特产建筑+专属兵营+专属工坊
- 核心目标：垄断本国专属特产，月特产收入>200金
- 关键资源：本国特有资源(子母河水/瑶池圣水/天衣丝绸等)
- 人口分配：农业35%+特产相关30%+其他35%

**阶段②：特产垄断（20-50回合）**
- 优先建设：特产加工升级+专属军队+特色文化建筑
- 核心目标：成为特定商品的全域唯一供应商，掌握定价权
- 关键资源：专属特产月产>5单位，外交依存度高
- 人口分配：特产相关35%+农业30%+其他35%

**阶段③：不可替代（50+回合）**
- 优先建设：专属科技树+终极兵种+国家特色传奇建筑
- 核心目标：其他国家在特定领域必须依赖你，形成战略筹码
- 关键资源：专属特产世界级产量，专属兵种战力碾压同等级
- 特色能力：**战略垄断**（可对特定国家禁运）、**特色降服**（用特产换取附庸）

### 3.7 发展路径转换机制

| 转换 | 条件 | 代价 | 适应场景 |
|------|------|------|----------|
| **A↔B** | 经济→军事 | 3000金军备转型费+10回合 | 商路扩展遇阻，军事破局 |
| **A↔C** | 经济→文化 | 2000金文化转型费+8回合 | 贸易对手变盟友 |
| **B↔D** | 军事→种田 | 1500金建设转型费+12回合 | 战争疲劳，休养生息 |
| **C↔D** | 文化→种田 | 1000金改建费+6回合 | 外交封锁，自给自足 |
| **→E** | 任意→特色 | 5000金+本民族专属事件触发 | 发现专属特产或触发国家特色事件 |
| **E→** | 特色→任意 | 3000金转型费+15回合 | 单一战略受制，需要多元化 |

> **转换冷却**：任何发展路径转换后需等待**20回合**才可再次转换，防止玩家频繁切换导致游戏缺乏策略深度。
