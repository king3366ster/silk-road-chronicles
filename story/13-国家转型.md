## 一、概述

国家类型转换是指西域五十国在**普通国家、母系国家、全女性国家**三大类型之间的制度性转型。不同类型的转型触发条件、事件流程、影响范围各不相同。本文档集中论述各类**国家类型转换触发事件**的详细规则与剧情节点。

### 1.1 转型路径总览

```
  普通国家 ──[凤鸣觉醒]──→ 母系国家 ──[女儿当国]──→ 全女性国家
      ↑                       │                         │
      └───────[父权复辟]──────┘                         │
                                ↑                        │
                                └──────[凤归母巢]────────┘

  注意：全女性→普通 = 不可能（信仰崩溃）
```

### 1.2 三大国家类型核心参数

| 维度 | 普通国家（36国） | 母系国家（8国） | 全女性国家（6国） |
|------|-----------------|----------------|------------------|
| **人口构成** | 男女比55:45 | 男女比30:70 | 男女比0:100 |
| **统治者性别** | 男性或女性均可 | 必须为女性 | 必须为女性 |
| **政治制度** | 6种标准制 | 6种母系变体 | 6种全女变体 |
| **兵源** | 仅男兵 | 男兵:女兵=8:2~6:4 | 仅女兵（战力+10%） |
| **转型难度** | ★★★☆☆ | ★★★★☆ | ★★★★★ |

## 二、主动转型事件（类型二）

> 当人口性别比例发生变化，社会内部女权力量抬头或男权复辟时触发的主动制度转型。与强制同化（类型一）完全独立，可并行发生。

### 2.1 凤鸣觉醒（普通→母系）

> 普通国家因女性人口比例持续超过阈值，女权力量抬头，触发制度向母系方向转型。

#### 触发条件

| 条件项 | 要求 |
|--------|------|
| **人口条件** | 势力全部公民女性比例≥65%（接近母系国70%标准），连续36回合（1年） |
| **稳定度** | 稳定度≥50 |
| **外交接触** | 与至少1个母系或全女国家有外交接触/商路往来 |
| **阶段限制** | 阶段四起可触发 |
| **触发频率** | 每年至多1次 |

#### 触发事件链

| 事件节点 | 回合 | 内容 | 玩家选择 |
|----------|------|------|----------|
| **① 女权思潮** | 女性比例≥60%时首次提示 | 城邦中出现女权思潮运动，女性贵族要求在议事会中获得更多席位 | ①压制（稳定度-10，女性满意度-15）②倾听（女性满意度+10） |
| **② 女性集会** | 女性比例≥62%时 | 女性市民在广场集会，要求政治权利平等 | ①武力驱散（稳定度-15，女性满意度-20）②和平对话（女性满意度+10） |
| **③ 女权领袖出现** | 女性比例≥65%时 | 一位具有魅力的女性领袖出现，领导女权运动 | ①逮捕领袖（女性满意度-25，稳定度+5）②支持领袖（女性满意度+20，稳定度-5） |
| **④ 转型抉择** | 女性比例≥65%持续36回合后 | 女权运动达到高潮，要求正式实行母系制度 | **【接受转型】或【拒绝转型】** |

#### 接受转型效果

| 效果项 | 详情 |
|--------|------|
| **政治制度** | 转为对应母系政治制度变体（如君主专政制→母系君主专政制） |
| **经济制度** | 不变（可后续转型） |
| **人口调整** | 男性公民可遣散（补偿金200金/千人）或保留公民身份但不可为统治者 |
| **军队调整** | 男兵可遣散（补偿金）或转业；女兵可从精锐卫队中征召 |
| **稳定度** | →100 |
| **外交影响** | 与全女国/母系国外交好感自动+15；与普通国家好感-10 |
| **路线解锁** | 解锁D线·凤鸣昆仑（若之前未解锁） |

#### 拒绝转型效果

| 效果项 | 详情 |
|--------|------|
| **稳定度** | -20（女权运动镇压） |
| **女性满意度** | -30（持续6个月） |
| **叛乱风险** | +10%/月，持续12个月 |
| **再次触发** | 条件持续满足时，12个月后再次触发判定 |

> **设计意图**：凤鸣觉醒是社会人口结构变化驱动的自然转型，与军事征服路线独立。普通国女性玩家可借此事件平稳过渡到母系统治；男性玩家若想保持男权统治，需持续维持男性人口优势。

### 2.2 女儿当国（母系→全女）

> 母系国家男性人口持续减少至接近零时，社会向全女性化转型。

#### 触发条件

| 条件项 | 要求 |
|--------|------|
| **人口条件** | 势力全部公民男性比例≤15%（接近全女国0%标准），连续36回合（1年） |
| **稳定度** | 稳定度≥50 |
| **外交接触** | 与至少1个全女国家有外交接触 |
| **阶段限制** | 阶段四起可触发 |
| **触发频率** | 每年至多1次 |

#### 触发事件链

| 事件节点 | 内容 | 玩家选择 |
|----------|------|----------|
| **① 男性减少** | 城邦中适龄男性数量持续下降，女性承担更多社会职能 | ①鼓励男性生育（人口政策调整，花费500金）②顺应自然趋势（不干预） |
| **② 女神崇拜兴起** | 全女国宗教文化传入，女神崇拜在女性中流行 | ①允许传播（宗教多元化，稳定度-5）②禁止传播（宗教保守，女性满意度-10） |
| **③ 全女呼声** | 激进派女性贵族提出驱逐全部男性的动议 | ①支持动议（男性满意度-20，女性满意度+15）②否决动议（女性满意度-10） |
| **④ 转型抉择** | 男性比例≤15%持续36回合后，全女转型条件成熟 | **【接受转型】或【拒绝转型】** |

#### 接受转型效果

| 效果项 | 详情 |
|--------|------|
| **政治制度** | 转为对应全女政治制度变体（如母系君主专政制→全女君主专政制） |
| **经济制度** | 不变（可后续转型） |
| **人口调整** | 男性公民可遣散（补偿金200金/千人）或自愿外迁至邻国；**不涉及强制转奴隶** |
| **军队调整** | 男兵可遣散（补偿金）；军队转为全部女兵 |
| **稳定度** | →100 |
| **外交影响** | 与全女国外交好感自动+25；与普通国家好感-15 |
| **特殊规则** | 若存在男性玩家角色，触发「唯一合法男性」事件（详见11-玩家角色.md） |

#### 拒绝转型效果

| 效果项 | 详情 |
|--------|------|
| **稳定度** | -15 |
| **女性满意度** | -20（持续6个月） |
| **激进派独立风险** | 激进派女性贵族可能发动分裂，部分城邦以全女国身份独立 |
| **再次触发** | 条件持续满足时，12个月后再次触发判定 |

### 2.3 案例：各国转型推荐时机

| 国家类型 | 推荐转型 | 最佳阶段 | 前置准备 |
|----------|----------|----------|----------|
| 普通国（楼兰） | 凤鸣觉醒→母系 | 阶段四 | 提高女性人口比例至65%+，稳定度≥70 |
| 普通国（龟兹） | 凤鸣觉醒→母系 | 阶段五 | 先发展文化影响力，再转型 |

| 全女国（西梁） | 不可转型普通 | — | 全女→普通=不可能 |

## 三、强制同化中的自身调整（类型一·附属）

> 当征服国选择强制同化时，自身可能先因接触外界而发生制度转变。详见[04-制度.md](04-制度.md)第6.2节。此处仅列出触发条件摘要。

### 3.1 自身调整触发条件汇总

| 转型事件 | 方向 | 触发条件 | 属于强制同化第一步 |
|----------|------|----------|:------------------:|
| **🌸 凤鸣觉醒** | 普通→母系 | 与全女/母系国接触 + 稳定度≥70 + 女性公民比例>70%持续36回合 | ✅ |
| **🔄 父权复辟** | 母系→普通 | 与普通国接触并受影响 + 稳定度≥70 | ✅ |
| **🔥 女儿当国** | 母系→全女 | 与全女国接触 + 稳定度≥70 + 男性公民比例<10%持续36回合 | ✅ |
| **💧 凤归母巢** | 全女→母系 | 与母系/普通国接触 + 稳定度≥70 | ✅ |

### 3.2 与主动转型的差异

| 维度 | 强制同化（自身调整） | 主动转型（类型二） |
|------|--------------------|------------------|
| **触发前提** | 必须先选择强制同化 | 与强制同化选择无关 |
| **人口阈值** | 更严格（女性比例>70%而非≥65%） | 较宽松（女性比例≥65%） |
| **调整性质** | 作为同化流程的一部分，后续还有激进阶段 | 独立发生，仅温和调整 |
| **锁定效果** | 完成强制同化后自身制度永久锁定 | 不锁定，可再次转型 |

## 四、叛乱引发的被动转型（类型三）

> 因性别比例失衡或外国渗透引发的叛乱，可能导致城邦以另一类型的国家身份独立。

### 4.1 男权反扑叛乱

| 维度 | 说明 |
|------|------|
| **触发情景** | 母系国或全女国内部因男性比例升高，或受普通国家渗透，男性势力发动叛乱 |
| **转型方向** | 全女/母系→普通（独立） |
| **触发条件** | ①母系国：男性公民比例>40%+稳定度<30持续3回合；②全女国：引入男性未及时处置 |
| **独立后果** | 叛乱的城邦以普通国身份独立，脱离玩家控制，不可逆 |
| **平叛方式** | 详见[04-制度.md](04-制度.md)第6.4.3节 |

### 4.2 渗透引发叛变

| 维度 | 说明 |
|------|------|
| **触发情景** | 外国势力（普通国为主）向母系/全女城邦渗透，煽动内部反对派 |
| **转型方向** | 全女/母系→普通（加入渗透国） |
| **典型事件** | 「外来商旅的阴谋」「传教士的秘密使命」「敌国间谍网」 |
| **预防手段** | 维持稳定度≥50，定期清理外国间谍，限制外来人员活动范围 |

## 五、阶段相关国家转型事件索引

以下为各游戏阶段可触发的国家转型事件汇总，按阶段排序：

| 游戏阶段 | 可触发事件 | 说明 |
|----------|-----------|------|
| **阶段一** | 无 | 新手保护期，无国家转型事件 |
| **阶段二** | 无 | 新手保护期，无国家转型事件 |
| **阶段三** | 男权反扑叛乱（被动） | 保护期结束，可能因前期布局引发叛乱 |
| **阶段四** | 凤鸣觉醒、女儿当国（主动+强制同化自身调整） | 制度选择期，转型事件密集 |
| **阶段五** | 凤鸣觉醒、女儿当国、父权复辟、凤归母巢 | 强制同化最终决定期，转型达到高潮 |
| **阶段六** | 凤鸣觉醒、女儿当国（延续触发） | 势力扩张期，转型影响范围扩大至所有被占领国 |
| **阶段七** | 外部势力干预转型 | 外部势力正式参战，可能通过外交/军事手段强制改变国家类型 |

## 六、事件触发判定流程

```
每回合判定开始
    │
    ├── 是否处于新手保护期（阶段一~二）？→ 是 → 跳过所有转型判定
    │
    ├── 是否触发主动转型判定？
    │   ├── 普通国：女性比例≥65%持续36回合？→ 是 → 凤鸣觉醒事件
    │   ├── 母系国：男性比例≤15%持续36回合？→ 是 → 女儿当国事件
    │   └── 否 → 继续监测
    │
    ├── 是否触发强制同化自身调整？
    │   ├── 已选择强制同化且未完成自身调整？→ 是 → 检查转型条件
    │   └── 否 → 跳过
    │
    └── 是否触发叛乱转型？
        ├── 性别比例失衡？→ 是 → 男权反扑叛乱判定
        ├── 外国渗透持续？→ 是 → 渗透叛变判定
        └── 否 → 继续监测
```
