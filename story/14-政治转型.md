## 一、政治制度概述

西域五十国共有**6种基本政治制度**，每种制度在三大国家类型（普通/母系/全女）下各有对应的变体，总计18种政治制度变体。

### 1.1 六种基本政治制度

| 编号 | 政治制度 | 核心特征 | 权力中心 | 社会等级 | 适合路线 |
|:----:|----------|----------|----------|----------|----------|
| **P1** | 君主专政制 | 君主独裁，中央集权 | 国王/皇帝 | 严格等级制 | A·君临天下 |
| **P2** | 部落酋长制 | 部落联盟，首领推举 | 部落首领/酋长 | 部落等级制 | E·纵横捭阖 |
| **P3** | 贵族议政制 | 贵族共治，权力制衡 | 贵族议会/元老院 | 贵族—平民等级 | B·富甲丝路 |
| **P4** | 城邦市政制 | 市民自治，市政管理 | 市政议会/执政官 | 市民平等（有产者特权） | C·万国来朝 |
| **P5** | 宗教领袖制 | 政教合一，神权统治 | 大祭司/宗教领袖 | 神职—世俗等级 | C·万国来朝 |
| **P6** | 封建奴隶制 | 领主分封，奴隶劳动 | 大领主/奴隶主 | 领主—平民—奴隶三级严格等级 | F·天朝藩屏 |

### 1.2 三大类型变体示例（以君主专政制为例）

| 国家类型 | 变体名称 | 核心差异 |
|----------|----------|----------|
| **普通国** | 君主专政制 | 国王可为男性或女性；继承权男女均可 |
| **母系国** | 母系君主专政制 | 国王必须为女性；继承权限女性；男性最高为顾问/将军 |
| **全女国** | 全女君主专政制 | 国王必须为女性；全部官员为女性；男性不得担任任何官职 |

> 其他5种政治制度（P2~P6）同理，每种制度在母系国/全女国下均有对应的变体名称和规则调整。详见[04-制度.md](04-制度.md)第1~4章。

## 二、政治制度转换触发事件

> 政治制度转换（在同一国家类型内部转换政治制度）独立于国家类型转换。玩家可在满足条件时主动发起政治制度变革。

### 2.1 转换基本规则

| 规则项 | 说明 |
|--------|------|
| **转换范围** | 仅在**同一国家类型**的6种政治制度之间转换（如普通国君主专政制→普通国贵族议政制） |
| **转换前提** | 当前稳定度≥60 |
| **冷却时间** | 每次转换后需间隔24回合（约8个月）方可再次转换 |
| **转换消耗** | 视制度差异大小，需消耗500~2000金 |
| **稳定度影响** | 转换后稳定度-20（随时间恢复） |

### 2.2 转换触发方式

政治制度转换可通过以下四种方式之一触发：

| 方式 | 说明 | 典型场景 |
|------|------|----------|
| **① 改革决策** | 玩家主动发起政治改革 | 君主觉得权力不够集中→推行君主专政制；贵族势力膨胀→推行贵族议政制 |
| **② 事件触发** | 特定剧情事件推动 | 市民起义要求自治→推行城邦市政制；宗教势力崛起→推行宗教领袖制 |
| **③ 外部压力** | 被大国/帝国要求改制 | 被汉朝要求实行城邦市政制以纳入朝贡体系；被匈奴要求实行部落酋长制 |
| **④ 革命/政变** | 内部权力斗争导致制度更迭 | 贵族发动政变推翻君主→推行贵族议政制；奴隶起义推翻奴隶主→推行城邦市政制 |

#### 2.2.1 改革决策转换条件

| 目标制度 | 前置条件 | 消耗 | 稳定度变化 | 特殊要求 |
|----------|----------|:----:|:----------:|----------|
| **P1 君主专政制** | 当前兵力≥总人口×10% | 1500金 | -15 | 需获得军队支持（军忠诚度≥60） |
| **P2 部落酋长制** | 势力覆盖≥3个不同部落 | 800金 | -10 | 需获得各部落首领支持 |
| **P3 贵族议政制** | 贵族满意度≥60 | 1200金 | -10 | 需召开贵族大会 |
| **P4 城邦市政制** | 平民满意度≥60 | 1000金 | -15 | 需召开市民大会 |
| **P5 宗教领袖制** | 宗教影响力≥200 | 2000金 | -20 | 需获得宗教领袖支持 |
| **P6 封建奴隶制** | 奴隶人口≥总人口×30% | 500金 | -10 | 需获得大领主支持 |

#### 2.2.2 事件触发转换

| 事件名称 | 触发条件 | 目标制度 | 玩家选择 |
|----------|----------|----------|----------|
| **市民起义** | 平民满意度<20持续6回合 | P4 城邦市政制 | ①镇压（稳定度+10，人口-5%）②接受改制（转为城邦市政制，稳定度-10） |
| **宗教复兴** | 宗教影响力≥150+稳定度<40 | P5 宗教领袖制 | ①限制宗教势力（稳定度+5，宗教影响力-30）②接受神权统治（转为宗教领袖制） |
| **贵族政变** | 贵族满意度<20+兵力集中 | P3 贵族议政制 | ①调动军队镇压（战争）②与贵族妥协（转为贵族议政制，贵族满意度+30） |
| **奴隶大起义** | 奴隶满意度<10持续6回合 | P6 封建奴隶制强化/P4 | ①血腥镇压（人口-10%，稳定度-30）②释放奴隶改制（转为城邦市政制，稳定度+10） |
| **外邦施压** | 被大国/帝国要求，外交压力≥70 | 随要求而定 | ①接受要求（好感+20）②拒绝（外交关系-30，战争风险+15%） |

#### 2.2.3 革命/政变转换

| 类型 | 触发条件 | 流程 | 后果 |
|------|----------|------|------|
| **宫廷政变** | 某派系影响力≥80+忠诚度<30 | 政变方需成功发动→旧统治者被废黜→新势力上台 | 胜利：制度转换，稳定度-30；失败：叛乱派系被清洗 |
| **平民革命** | 平民满意度<10+人口≥5000 | 革命军需攻占首都→旧政权倒台→成立新政体 | 胜利：转为城邦市政制/贵族议政制；失败：镇压，人口-15% |
| **军事政变** | 军忠诚度<20+将军影响力≥80 | 将军率军进入首都→要求权力交接 | 胜利：转为君主专政制/部落酋长制；失败：将军被处决 |

## 三、政治制度与经济制度匹配关系

> **核心规则**：并非所有经济制度都能与任意政治制度兼容。不兼容的组合将产生惩罚效果，严重时可能导致制度崩溃。

### 3.1 配套联动收益与惩罚规则

```mermaid
graph LR
    A[政治制度] -- "决定经济制度上限" --> B[经济制度]
    B -- "影响军事实力上限" --> C[军事制度]
    C -- "反馈社会稳定性" --> A
```

### 3.2 政治—经济制度兼容性矩阵

| 政治制度 ↓ | 绿洲农耕制 | 商路贸易制 | 游牧畜牧制 | 庄园贡赋制 | 国家专营制 | 特色精工制 |
|-------------|:----------:|:----------:|:----------:|:----------:|:----------:|:----------:|
| **P1 君主专政制**     | ✅ 兼容 | ✅ 兼容 | ✅ 兼容 | ✅ 兼容 | ✅ **最佳适配** | ✅ 兼容 |
| **P2 部落酋长制**     | ✅ 兼容 | ❌ 排斥 | ✅ **最佳适配** | ✅ 兼容 | ❌ 排斥 | ✅ 兼容 |
| **P3 贵族议政制**     | ✅ 兼容 | ✅ 兼容 | ❌ 排斥 | ✅ **最佳适配** | ❌ 排斥 | ✅ 兼容 |
| **P4 城邦市政制**     | ✅ 兼容 | ✅ **最佳适配** | ❌ 排斥 | ❌ 排斥 | ✅ 兼容 | ✅ **最佳适配** |
| **P5 宗教领袖制**     | ✅ 兼容 | ✅ 兼容 | ❌ 排斥 | ✅ 兼容 | ✅ 兼容 | ✅ 兼容 |
| **P6 封建奴隶制**     | ✅ **最佳适配** | ✅ 兼容 | ❌ 排斥 | ✅ **最佳适配** | ❌ 排斥 | ✅ 兼容 |

**图例**：
- ✅ **兼容** = 可正常运行，无额外收益/惩罚
- ✅ **最佳适配** = 产生联动收益（产出+15%，稳定度+5）
- ❌ **排斥** = 产生联动惩罚（产出-20%，稳定度-10/月，叛乱风险+5%/月）

#### 3.2.1 排斥组合的详细惩罚

| 排斥组合 | 惩罚说明 |
|----------|----------|
| **部落酋长制 + 商路贸易制** | 部落自治度高，无法统一管理商路税收，商人各自为政，税收混乱 |
| **部落酋长制 + 国家专营制** | 部落首领抵制中央统制经济，私商泛滥，专营形同虚设 |
| **贵族议政制 + 游牧畜牧制** | 贵族议会制度建立在固定领地基础上，游牧迁移与议会制矛盾 |
| **贵族议政制 + 国家专营制** | 贵族议政制强调分权制衡，国家专营需高度集权，两者冲突 |
| **城邦市政制 + 游牧畜牧制** | 城邦市政制度以城市为中心，游牧畜牧无固定城市，无法适配 |
| **城邦市政制 + 庄园贡赋制** | 市民自治排斥庄园领主制度，平民不愿为领主负担贡赋 |
| **封建奴隶制 + 游牧畜牧制** | 封建等级制度需固定领土分封，游牧迁徙无法维持领地制度 |
| **封建奴隶制 + 国家专营制** | 领主拥有领地自治权，抵制国家统一专营经济 |

#### 3.2.2 最佳适配组合的详细收益

| 最佳适配组合 | 联动收益 |
|-------------|----------|
| **君主专政制 + 国家专营制** | 中央指令直达，经济效率+15%；专营收入直接归王室，收入+15%；稳定度+5 |
| **部落酋长制 + 游牧畜牧制** | 部落游牧传统得到制度保障，畜牧产出+15%；部落凝聚力+10%；稳定度+5 |
| **贵族议政制 + 庄园贡赋制** | 贵族领主有权有产，庄园管理制度成熟，税收+15%；稳定度+5 |
| **城邦市政制 + 商路贸易制** | 市民经商自由度高，商路管理高效，贸易收入+15%；城市发展+10%；稳定度+5 |
| **城邦市政制 + 特色精工制** | 市民有产者有技术，手工业发达，精工产出+15%；稳定度+5 |
| **封建奴隶制 + 绿洲农耕制** | 奴隶大田庄制度效率高，粮食产出+15%；奴隶管理成本-10%；稳定度+5 |
| **封建奴隶制 + 庄园贡赋制** | 领主庄园+奴隶劳动产出最大化，农业产出+20%；稳定度+5 |

## 四、政治制度与军事制度匹配关系

### 4.1 政治—军事制度兼容性矩阵

| 政治制度 ↓ | 全民征兵制 | 贵族私兵制 | 雇佣兵制 | 部落兵制 | 常备军制 | 民兵自卫制 |

| **P2 部落酋长制**     | ✅ 兼容 | ❌ 排斥 | ✅ 兼容 | ✅ **最佳适配** | ❌ 排斥 | ✅ 兼容 |
| **P3 贵族议政制**     | ❌ 排斥 | ✅ **最佳适配** | ✅ 兼容 | ❌ 排斥 | ❌ 排斥 | ✅ 兼容 |
| **P4 城邦市政制**     | ❌ 排斥 | ❌ 排斥 | ✅ 兼容 | ❌ 排斥 | ✅ 兼容 | ✅ **最佳适配** |
| **P5 宗教领袖制**     | ✅ 兼容 | ❌ 排斥 | ✅ 兼容 | ❌ 排斥 | ✅ **最佳适配** | ✅ 兼容 |
| **P6 封建奴隶制**     | ✅ 兼容 | ✅ **最佳适配** | ❌ 排斥 | ✅ 兼容 | ❌ 排斥 | ❌ 排斥 |

**图例**：
- ✅ **兼容** = 可正常运行
- ✅ **最佳适配** = 战力+10%，军费-10%
- ❌ **排斥** = 战力-20%，军费+20%

#### 4.1.1 排斥组合说明

| 排斥组合 | 说明 |
|----------|------|
| **君主专政制 + 贵族私兵制** | 君主不允许贵族拥有私人武装，私兵制被视为对王权的威胁 |
| **君主专政制 + 部落兵制** | 部落兵制具有部落自治属性，与中央集权矛盾 |
| **君主专政制 + 民兵自卫制** | 民兵自治组织与君主独裁体制不兼容 |
| **部落酋长制 + 贵族私兵制** | 部落内部平等，不允许贵族私兵特权 |
| **部落酋长制 + 常备军制** | 常备军需中央财政长期支持，部落制度无法维持 |
| **贵族议政制 + 全民征兵制** | 全民征兵会赋予平民军事权力，动摇贵族统治基础 |
| **贵族议政制 + 部落兵制** | 贵族议政已超越部落阶段，部落兵制与贵族议会制度不匹配 |
| **贵族议政制 + 常备军制** | 议政体制决策缓慢，无法及时指挥常备军；军费控制权在议会，常备军运作受限 |
| **城邦市政制 + 全民征兵制** | 市民不愿承担长期兵役，城邦制度倾向专业军队 |
| **城邦市政制 + 贵族私兵制** | 市民自治排斥贵族私军特权 |
| **城邦市政制 + 部落兵制** | 城邦文明已超越部落阶段 |
| **封建奴隶制 + 雇佣兵制** | 领主有领主自有农奴兵，不需要雇佣兵 |
| **封建奴隶制 + 常备军制** | 封建分权无法维持中央常备军 |
| **封建奴隶制 + 民兵自卫制** | 奴隶不被允许武装，领主不允许民兵自治 |

#### 4.1.2 最佳适配组合的详细收益

| 最佳适配组合 | 联动收益 |
|-------------|----------|
| **君主专政制 + 常备军制** | 中央直接统率职业军队，指挥效率+10%；战力+10%；军费-10% |
| **部落酋长制 + 部落兵制** | 部落全民皆兵的传统延续，战力+10%；补给自足，军费-10% |
| **贵族议政制 + 贵族私兵制** | 贵族自备武装战力高昂，战力+10%；军费由贵族承担，军费-10% |
| **城邦市政制 + 民兵自卫制** | 市民保卫家园斗志高，城防战力+15%；无需军费维持 |
| **宗教领袖制 + 常备军制** | 宗教狂热+职业军队，战力+15%；军费-10% |
| **封建奴隶制 + 贵族私兵制** | 领主武装与庄园经济结合，战力+10%；后勤自给，军费-10% |

## 五、综合制度转换事件

> 当玩家同时转换多种制度时，触发综合事件链，产生复合效果。

### 5.1 大规模改革事件

| 事件名称 | 触发条件 | 涉及制度 | 效果 |
|----------|----------|----------|------|
| **全面维新** | 同时转换政治+经济制度 | 政治+经济 | 消耗3000金，稳定度-30，持续24回合恢复 |
| **军政改革** | 同时转换政治+军事制度 | 政治+军事 | 消耗2500金，稳定度-25，战备状态持续12回合 |
| **经济军事双改** | 同时转换经济+军事制度 | 经济+军事 | 消耗2000金，稳定度-20，产出-15%持续12回合 |
| **全盘改制** | 同时转换三种制度 | 政治+经济+军事 | 消耗5000金，稳定度-40，持续36回合恢复；完成后的综合收益为三种最佳适配之和 |

### 5.2 转换顺序与连锁反应

```
制度改革流程图：

  ① 先改政治制度（需6个月过渡期）
     │
     ↓
  ② 经济制度可随后调整（政治制度稳定后再改）
     │
     ↓
  ③ 军事制度最后调整（经济基础决定军费上限）

  提示：逆序改革可能导致一段时期的不兼容惩罚
```

## 六、阶段相关政治转型事件汇总

| 游戏阶段 | 可触发事件 | 说明 |
|----------|-----------|------|
| **阶段一** | 无 | 新手保护期，无权进行制度变革 |
| **阶段二** | 城邦管理改革（选举/任命执政官） | 获得管理权后可小幅调整管理架构 |
| **阶段三** | 贵族政变、外邦施压（系列事件） | 保护期结束，制度压力开始显现 |
| **阶段四** | 全面改革决策、市民起义、宗教复兴 | **制度选择期**，可自由选择并改革制度 |
| **阶段五** | 革命/政变、强制同化决定 | 强制做出最终制度选择的关键期 |
| **阶段六** | 势力级制度改革、综合改革事件 | 以势力领袖身份推行势力范围的制度统一 |
| **阶段七** | 外部势力干预改制 | 外部势力正式参战，可能强迫改制度（详见24-外部势力.md） |
