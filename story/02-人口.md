> **奴隶买卖价格公式**：
> - **买入价格** = 基础价格 × (1 + 距离浮动率) × 1.05

> - 同一市场各品类每月有收购上限，防止单次批量倾销扰乱市价

## 三、人口增长公式

### 3.1 公民人口增长

> 公民人口增长的基础计算公式适用于所有国家，但月增长率因国家类型而异。

#### 3.1.1 计算公式

```
公民人口月增长 = 当前公民人口 × (公民月增长率 × 城市等级对应比率 × 季节修正 × 剩余粮食修正 + 额外增长率)
```

| 变量 | 说明 | 取值范围 |
|------|------|----------|
| **公民月增长率** | 基础月增长率，按国家类型固定 | 普通1% / 母系0.5% / 全女0% |
| **城市等级对应比率** | Lv1无加成，Lv2+10%，Lv3+15%，Lv4+20%，Lv5+25% | 1.0~1.25 |
| **季节修正** | 春季+10%，夏季+5%，秋季-5%，冬季-20% | 0.80~1.10 |
| **剩余粮食修正** | 粮食盈余>10%时+10%，盈余>30%时+20%，赤字时-10% | 0.90~1.20 |
| **额外增长率** | 圣水受孕/外交通婚等额外获得的月增长率（见3.4节） | 0%~1.5% |

#### 3.1.2 国家类型基础月增长率

| 国家类型 | 公民月增长率 | 奴隶月增长率 | 说明 |
|----------|-------------|-------------|------|
| **普通国家** | 1% | 1% | 标准自然增长，不可使用繁殖方式 |
| **母系国家** | 0.5% | 0.5% | 自然增长率较低，可使用繁殖方式补充 |
| **全女国家** | 0% | 0% | 无自然增长，完全依赖繁殖方式 |

#### 3.1.3 圣水受孕（基础繁殖方式）

> 母系/全女国家可通过消耗圣水类资源获得额外月增长率：

| 资源 | 消耗 | 效果 | 说明 |
|------|------|------|------|
| **子母河水** | 每消耗2单位 | 获得+1%月增长率 | 对公民和奴隶各+0.5%，总+1% |
| **瑶池圣水** | 每消耗1单位 | 获得+1%月增长率 | 对公民和奴隶各+0.5%，总+1% |
| **弱水泉水** | 每消耗1单位 | 获得+1%月增长率 | 对公民和奴隶各+0.5%，总+1% |

> **增长率上限**：母系/全女国家的总月增长率上限为**1.5%**（含圣水受孕+外交通婚叠加）。以上两种方式获得的增长率均为基础增长率，适用于公式中的"额外增长率"变量。

#### 3.1.4 上限与人口上限

| 国家类型 | 公民月增长率上限 | 年增长率上限 | 最大人口上限 |
|----------|-----------------|-------------|-------------|
| **普通国家** | 1% (不可提升) | +12%/年 | 大国100,000 / 中等国70,000 / 小国50,000 |
| **母系国家** | 1.5% (圣水+通婚叠加) | +18%/年 | 同上 |
| **全女国家** | 1.5% (圣水+通婚叠加) | +18%/年 | 同上 |

### 3.2 母系性别比例约束机制

> 母系国家的核心特征是女性占多数（初始男女比30:70）。随着游戏发展，玩家可能通过各种手段改变性别比例。为维持母系社会的本质特征，设置以下约束：

- **性别比例红线**：当男性人口占比超过40%（即男女比>40:60）时，触发**"母系失衡"**状态
- **失衡效果**：
  - 🚫 **禁止购买男性奴隶**：奴隶市场中的男性奴隶选项被锁定
  - 🚫 **禁止男性奴隶转化**：无法将男性奴隶转化为男性公民
  - 🚫 **男性公民自然增长停止**：人口增长时，新增公民全部为女性
  - ⚠️ **母系文化加成减半**：所有母系文化加成效果降低50%
  - ⚠️ **外交联姻成功率-20%**：母系国家认为你已背离传统
- **恢复条件**：当男性人口占比回落至40%以下（男女比≤40:60）时，自动恢复正常状态
- **设计目的**：防止玩家通过大量购买男性奴隶或转化手段将母系国家变为事实上的普通国家，保持母系国家的独特游戏体验

### 3.3 奴隶人口自然增长

> **核心规则**：奴隶人口增长公式与公民类似，但受侍妾占比加成影响。

#### 3.3.1 计算公式

```
奴隶人口月增长 = 当前奴隶人口 × 奴隶月增长率 × 城市等级对应比率 × (1 + 侍妾占比加成)
```

| 变量 | 说明 | 取值范围 |
|------|------|----------|
| **奴隶月增长率** | 基础月增长率，按国家类型固定 | 普通1% / 母系0.5% / 全女0% |
| **城市等级对应比率** | Lv1无加成，Lv2+10%，Lv3+15%，Lv4+20%，Lv5+25% | 1.0~1.25 |
| **侍妾占比加成** | = (侍妾数量 ÷ 奴隶总数) × 100% | 0%~100% |

- **侍妾占比加成示例**：
  - 100个奴隶中有10个侍妾 → 加成 = (10/100)×100% = 10% → 月增长系数 = 1.1
  - 100个奴隶中有50个侍妾 → 加成 = (50/100)×100% = 50% → 月增长系数 = 1.5
- **新增奴隶性别分配**：新增奴隶由玩家根据性别分配为不同分工（男劳奴、女匠奴等），性别比例根据国家制度：
  - 普通国家：55%男 / 45%女
  - 母系社会国家：30%男 / 70%女
  - 全女性国家：0%男 / 100%女
- **新增奴隶分工**：玩家确认分工后**不可改变**（见五、节奴隶分工不可变规则）

### 3.4 繁殖方式（6种）

> 以下6种繁殖方式中，前5种为全女/母系国家专属（月增长率共享上限**1.5%**），第3种"侍女/舞女租借"为所有国家通用。优先级：**优先消耗外交通婚额度，再消耗圣水额度**。

#### 方式1：圣水受孕

> 见3.1.3节详述。消耗子母河水/瑶池圣水/弱水泉水获得月增长率。

#### 方式2：外交通婚

> 仅母系/全女国家可使用。

| 项目 | 说明 |
|------|------|
| **条件** | 需与目标国外交关系≥友好（60+） |
| **消耗** | 每次消耗金币+外交点数 |
| **效果** | 一个国家的通婚+0.5%月增长率 |
| **上限** | 总月增长率上限1.5%（优先消耗外交通婚额度，再消耗圣水额度） |

> 通过多国外交联姻可提升月增长率，是长期稳定的人口增长策略。

#### 方式3：男奴繁殖

| 奴隶类型 | 消耗 | 产出 | 时间 |
|----------|------|------|------|
| **男劳奴** ♂ | 消耗1个 | 产生10个女公民 | 6个月后 |
| **男匠奴** ♂ | 消耗1个 | 产生10个女公民 | 6个月后 |
| **男战奴** ♂ | 消耗1个 | 产生20个女公民 | 9个月后 |

> 男奴繁殖是一种一次性人口爆发手段，将男性奴隶转化为女性公民，适合急需补充人口的场景。

#### 方式4：侍女/舞女租借

> 特殊的繁殖方式，所有国家均可使用（不限全女/母系）。

| 项目 | 说明 |
|------|------|
| **条件** | 在奴隶市场选择"侍女/舞女租借" |
| **租期** | 租借12个月 |
| **产出** | 12个月后产生2个女公民（租借后同时归还侍女/舞女） |
| **限制** | 租借期间不产生任何收益/加成，不可赠送/使用等 |

#### 方式5：收养孤儿

> 被动触发机制，无需玩家主动操作。

| 项目 | 说明 |
|------|------|
| **触发条件** | 毗邻国发生交战时自动收养战争孤儿 |
| **效果** | 邻国一个月内每发生过一次交战，月人口增长额外+0.5%（增加女公民，非奴隶） |
| **上限** | 上限不超过1%月增长率 |

> 此类增长适用于公式中的"额外增长率"变量。

#### 方式6：掠夺获取

> 通过军事行动获取战俘，补充奴隶人口。
> 所有国家通用。

| 项目 | 说明 |
|------|------|
| **目标** | 掠夺获取女奴隶 |
| **触发** | 对敌国领土进行掠夺/征服行动 |
| **产出** | 根据掠夺规模获得女奴隶，直接增加奴隶人口 |

> 掠夺获取的奴隶不经过自然增长计算，直接加入奴隶人口池。

> **性别差异规则**：
> - **月粮耗**：男公民1粮/月，女公民0.8粮/月（女性消耗更低）
> - **重体力产出**：男性在采矿(+33%)、伐木(+18%)、采石(+33%)方面优于女性
> - **精细产出**：女性在纺织(+100%)、采药(+100%)方面优于男性
> - **农业/商业/酿酒/畜牧**：男女产出相同，但女公民粮耗更低（性价比更高）
> - **全女性国家**：无男性，需通过更多女公民或奴隶弥补重体力产出不足，但女公民低粮耗是优势

> **征兵规则（按国家类型区分）**：
> - **普通国家**：**只征召男兵**。男公民（无论当前职业）均可被征召入伍。女公民不可被征召为士兵。
> - **母系社会国家（8个）**：征召**70%男兵+30%女兵**。女兵仅限中高级兵士(精锐卫队)，需城市等级≥Lv2。
> - **全女性国家**：**只征召女兵**。女公民（无论当前职业）均可被征召入伍。
>
> 征兵时公民脱离当前职业，成为士兵。士兵月粮耗1.5粮（精兵/老兵1.8粮），不区分男女。退役后可重新分配职业（消耗1回合）。
>
> **女兵战力加成**：同兵种女兵战斗力比男兵**高10%**。女兵纪律性更强、阵亡率略低（-5%）、兵器损耗略低（-3%/战）。

| 奴隶类型 | 可分配职业 | 月产出 | 月粮耗 | 备注 |
|----------|-----------|--------|--------|------|
| **男劳奴** | 农场/矿场/伐木 | +3粮 或 +2铁 或 +3木 | 0.6 | 重体力效率高 |

> **奴隶效率**：奴隶产出效率约为同性别公民的**0.8倍**（专业训练不足），但粮食消耗仅0.6/月（远低于男公民1/月、女公民0.8/月），是**高性价比劳动力**。奴隶不可分配为商人/士兵（战奴除外）。

> **奴隶分工不可变规则**：玩家一旦确认奴隶的分工（男劳奴、女匠奴、男战奴、舞女、侍妾等），**不可再改变**。请谨慎分配。唯一例外：奴隶可通过培养路线转化为公民（需消耗金币+时间）。

> **侍妾加成说明**：侍妾的加成作用于**奴隶人口自然增长**（见三、节3.3）。侍妾占比越高，奴隶年增长率越快。

## 七、人口-资源产出计算公式

```
月度资源总产出 = Σ(公民产出) + Σ(奴隶产出) + 部落贡赋 + 建筑加成

具体计算：
  粮食产出 = 农民(♂×3+♀×2.5) + 男劳奴×3 + 女劳奴×3 + 猎人×1 + 牧民×1
           × 城市等级倍率 × 季节修正 × 地形修正 × 文官加成

  铁矿产出 = 矿工(♂×2+♀×1.5) + 男劳奴×2
           × 城市等级倍率 × 文官加成

  木材产出 = 伐木工(♂×3+♀×2.5) + 男劳奴×3
           × 城市等级倍率

  石材产出 = 采石工(♂×2+♀×1.5) + 男劳奴×2
           × 城市等级倍率

  丝绸产出 = 纺织工(♂×1+♀×2) + 女劳奴×2 + 女匠奴×3
           × 城市等级倍率 × 国家特色加成

  金币收入 = 商人×3(本地) + 商人×5(商路) + 舞女×10 + 税收(人口×0.5)
           × 城市等级倍率 × 季节修正 × 文官加成

月度资源总消耗：
  粮食消耗 = 男公民×1 + 女公民×0.8 + 奴隶×0.6 + 士兵×1.5(精兵/老兵×1.8) + 马匹×2
  盐消耗 = 总人口÷100
  金币消耗 = 军费 + 建设费 + 奴隶购买 + 科技研发
```

## 九、人口增长对资源的影响链

```
人口增长 → 劳动力增加 → 资源产出增加 → 但粮食消耗也增加

关键平衡点：
  每新增1名男公民→分配农民→+3粮产出-1粮消耗=净+2粮/月 ✅
  每新增1名女公民→分配农民→+2.5粮产出-0.8粮消耗=净+1.7粮/月 ✅
  每新增1名男劳奴→分配农场→+2.4粮产出-0.6粮消耗=净+1.8粮/月 ✅
  每新增1名士兵→不产出→0-1.5粮消耗=净-1.5粮/月 ❌ 需要其他人养

最佳人口结构（粮食自给率最高）：
  农民45% + 纺织12% + 商业12% + 矿工8% + 伐木5% + 采石3% + 狩猎3% + 其他7% + 士兵5%
  → 粮食自给率≈110%（微盈余）
  → 但军事力量弱

最佳人口结构（军事强国）：
  农民35% + 纺织10% + 商业10% + 矿工8% + 伐木5% + 采石3% + 士兵25% + 其他4%
  → 粮食自给率≈85%（需贸易补足）
  → 军事力量强

结论：人口越多→总产出越高→但需要合理分配职业→粮食是核心瓶颈
