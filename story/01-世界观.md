## 一、时代背景

### 1.1 时代设定

> **一句话**：公元前100年左右，汉武帝凿空西域，丝路初开；西部高原葱岭与天山北麓尚未被汉朝完全控制，匈奴与汉之争在此上演，这是西域英雄争霸的黄金时代。

公元前100年左右，正值汉武帝刘彻当政的鼎盛时期。在北方，汉朝对匈战争已达三十年之久——从马邑之谋（前133年）到漠北之战（前119年），卫青、霍去病等名将大破匈奴主力，"封狼居胥"成为不朽传奇。雄主汉武帝的目光，早已越过河西走廊，投向了更远方的西域——那片传说中盛产汗血宝马、玉石美酒、香料丝绸的广阔天地。张骞的凿空之旅（前138~前126年）打开了通往西域的大门，而此刻的西域，正处于历史转折的关键节点。

### 1.2 六大外部势力

| 势力 | 位置 | 状态 | 对西域影响力 | 接壤关口 |
|------|------|------|-------------|----------|
| **汉朝** 🏯 | 东方（河西走廊以东） | 武帝鼎盛，对匈占优，积极经营西域 | ★★★★★ | 阳关、玉门关（南道/北道东端接壤） |
| **匈奴** 🐺 | 北方（大漠+天山以北草原） | 漠北之战受挫但未灭，西域北道/天山北仍受影响 | ★★★★☆ | 天山山口（车师北境）、且弥山口（天山北接壤） |
| **康居** 🐫 | 西北（咸海+锡尔河下游草原） | 游牧强国，与汉朝无直接冲突，保持中立 | ★★★☆☆ | 阗池西岸（乌孙西北接壤） |
| **大夏** 🏛️ | 西南（阿姆河中游+巴克特里亚） | 希腊化王国，与葱岭以西城邦有贸易往来 | ★★★☆☆ | 葱岭西口（无雷/休循以西接壤） |

> **大夏与贵霜的区隔**：大夏（Bactria）是亚历山大东征后建立的希腊化王国，控制阿姆河中游的肥沃平原；贵霜（Kushan）从大月氏五翕侯中的贵霜翕侯崛起，控制兴都库什山脉和犍陀罗地区。二者在文化上（希腊化vs印度-希腊混合）、军事上（希腊方阵vs游牧骑兵）、外交倾向上（亲汉vs中立）均有显著差异。游戏中大夏偏文化/贸易，贵霜偏军事/扩张。
>
> **地理限制**：南边是昆仑山脉（平均海拔5,500米，无法通行），不存在"南方帝国"。西域与南亚次大陆通过葱岭（帕米尔高原）连接，但高海拔使大规模军事行动几乎不可能。

### 1.3 六大势力交战规则

> 六大外部势力（汉朝、匈奴、康居、大夏、贵霜、羌人部落联盟）是西域50国之外的超然存在，有特殊的交战规则。

#### 1.3.1 阶段交战限制

| 阶段 | 规则 |
|------|------|
| **阶段一~五**（占领/同盟<36国） | 六大势力与玩家**可相互掠夺**，但**不可被占领**（不影响联姻/同盟关系） |
| **阶段六**（占领/同盟≥36国） | 六大势力**正式参战**，可攻占玩家城邦，触发决战剧情 |

#### 1.3.2 掠夺六大势力

> 阶段一~五期间，玩家可掠夺六大势力，获得远超普通城邦的战利品。

| 维度 | 规则 |
|------|------|
| **触发条件** | ①城市等级≥Lv2 ②与目标势力关系≤中立 ③完成目标势力≥1条关联故事线 |
| **掠夺收益** | ①金币：目标势力金库×15%~30%（普通掠夺×1.5倍）②资源：目标势力粮仓×15%~30% ③特产：高概率掠夺专属特产（概率=目标势力特产数量×15%）④俘虏：高概率掠夺人口（目标势力人口×3%~8%） |
| **掠夺代价** | ①与目标势力关系-40 ②目标势力可能派兵报复 ③威望-15 |
| **掠夺特殊效果** | 掠夺成功会影响势力间外交平衡：例如掠夺匈奴后，汉朝好感+20；掠夺汉朝后，匈奴好感+15；掠夺康居后，大夏好感+10 |

#### 1.3.3 第六阶段正式战争

> 阶段六触发后，六大势力可自由发展、触发剧情事件、干预西域事务、开始攻占城邦。

| 维度 | 规则 |
|------|------|
| **触发条件** | 玩家占领/同盟≥36国，自动进入第六阶段 |
| **六大势力行动** | ①可出兵攻占玩家城邦 ②触发剧情事件干预西域 ③可自由发展军力 ④相互之间也可能开战 |
| **玩家决战胜利条件** | ①将所有六大势力军队**赶出西域** ②攻占每个势力与西域接壤的**全部关口** |
| **各势力关口** | 汉朝：阳关、玉门关（2个）\| 匈奴：天山山口、且弥山口（2个）\| 康居：阗池西岸（1个）\| 大夏：葱岭西口（1个）\| 贵霜：葱岭南口（1个）\| 羌人部落联盟：昆仑山口（1个） |
| **关口攻防** | 每个关口为独立战场，攻占后该势力无法再通过此关口进入西域 |

#### 1.3.4 第六阶段前势力兵力

> 阶段六之前，六大势力与西域接壤的关口驻军规模固定，作为威慑存在。

| 势力 | 关口驻军规模 | 说明 |
|------|-------------|------|
| **汉朝** 🏯 | 阳关15万、玉门关20万 | 武皇帝国精锐，装备精良 |
| **匈奴** 🐺 | 天山山口15万、且弥山口10万 | 草原骑兵为主，机动性强 |
| **康居** 🐫 | 阗池西岸12万 | 游牧劲旅，中等装备 |
| **大夏** 🏛️ | 葱岭西口10万 | 希腊化方阵，防守型 |
| **贵霜** 🕌 | 葱岭南口12万 | 新兴势力，象兵特色 |
| **羌人部落联盟** 🐑 | 昆仑山口10万 | 松散部落军，装备较差 |

> 阶段六后，六大势力可自由发展军力，不再受此规模限制，可能通过剧情事件获得额外兵力加成。

### 1.5 环境影响

| 地理特征 | 对游戏的影响 |
|----------|-------------|
| **昆仑山脉** | 南边天险，无南方出口。**楼兰→婼羌→羌人领地→（即可进入青藏高原远端）**，仅可通过此路线进行极有限接触 |
| **天山山脉** | 分隔北道与天山以北草原，穿越天山需要特定山口通道（且弥、车师） |
| **葱岭（帕米尔高原）** | 西域西端天险，高海拔使冬季行军几乎不可能。丝路必经之地，控制葱岭即控制丝路西段 |
| **塔克拉玛干沙漠** | 南道与北道之间的天然屏障，需要绕行或跟随商队穿越 |
| **博斯腾湖/罗布泊** | 重要水源地，周边城邦农业产出+20% |

### 1.6 时间系统

> **核心时间单位**：1回合 = 10天，3回合 = 1个月，36回合 = 1年。

| 单位 | 对应关系 | 说明 |
|------|----------|------|
| **1回合** | 10天 | 游戏最小时间单位，每次操作消耗1回合 |
| **1月** | 3回合 | 月度结算周期（税收、粮食消耗、奴隶增长等均在每月结算时执行） |
| **1季** | 9回合（3个月） | 季节转换周期，部分季节效果（冬季皮毛消耗、秋季粮食丰收）按此生效 |
| **1年** | 36回合（12个月） | 年度宏观周期，用于阶段转换判定 |

> **所有消耗/产出类规则说明中的"月"均指3回合的月度结算周期，"回合"指单次操作单位。**

### 1.5 六大游戏阶段

> 玩家开局时选择**全部50国中的任一城邦**（包括六大全女性国家、八大母系国家、36个普通国家），每个城邦有独特的外交环境、资源禀赋和发展路径。

> **保护期**：前两个阶段（阶段一+阶段二，共回合1-36，即12个月/1年）为保护期，别国、敌国、外部势力均不会攻击或掠夺玩家。保护期让玩家有充足时间学习游戏机制、发展经济、完成故事线。

| 阶段 | 对应回合 | 时间跨度 | 核心目标 | 保护期 | 关键特征 |
|------|----------|----------|----------|:------:|----------|
| **阶段一：城邦初立** | 回合1-12 | 第1天~第4个月末（约4月） | 完成城邦专属故事线，获得管理权 | ✅ | 故事线按制度分类（全女/母系/普通三类通用），完成后解锁基础技能 |
| **阶段二：部落征途** | 回合13-36 | 第5~第12个月末（约8月） | 开发个性化城邦故事线，征服/联盟周边部落 | ✅ | 每个城邦故事线完全个性化，完成全部周边部落后结束本阶段 |
| **阶段三：城市发展** | 回合37-72 | 第13~第24个月末（约12月/1年） | 城市升级，外交扩张，跨区域竞争，攻略其他国家 | ❌ | 保护期结束，外部势力介入。需触目标故事线后方可征服/联盟 |
| **阶段四：丝路争霸** | 回合73-120 | 第3~第4年末（约2年） | 同化决策，制度选择，全面争霸 | ❌ | 占领/同盟≥2国触发，可灵活选择双制度并行或强制统一 |
| **阶段五：帝国博弈** | 回合121-180 | 第5~第6年末（约2年） | 与六大帝国正面对抗/合作，左右西域大局 | ❌ | 占领/同盟≥5国触发，帝国事件频发，多线作战成为常态 |
| **阶段六：一统西域** | 回合181+ | 第7年起 | 完成所选路线的最终胜利条件 | ❌ | 触发最终挑战：线路A灭国战、线路B经济封锁、线路C文化对决、线路D姐妹盟总攻、线路E帝国协调 |

**阶段触发/转换条件**（详见[07-剧情-故事线.md](07-剧情-故事线.md)）：
- **阶段一→二**：自动转换（完成城邦专属故事线后获得管理权）
- **阶段二→三**：满足任一条件→城市等级≥Lv3 / 主动攻击 / 外交事件 / 开辟商路
- **阶段三→四**：占领/同盟≥2个国家
- **阶段四→五**：占领/同盟≥5个国家 → 强制统一制度（一次性同化代价）
- **阶段五→六**：占领/同盟≥36个国家（触发最终决战剧情）

## 二、五十国概览

> 📋 **各国详细地理环境、核心特产、城邦特殊能力、战略地位等信息，请参阅 [08-城邦地理与特产志](09-地理描述.md)**。本节聚焦于国家分类、政治制度、推荐策略等游戏机制层面。

### 2.1 国家分类体系

西域五十国按综合实力分为三大类（12大国 + 18中等国 + 20小国 = 50国），按地理分为四大区域（南道11普通国+北道9普通国+天山北6普通国+葱岭10普通国 = 36普通国，外加6全女国+8母系国）：

| 类别 | 数量 | 特征 |
|------|------|------|
| **大国** | 12 | 初始人口8,000~12,000，发展上限50,000，周边5个部落 |
| **中等国** | 18 | 初始人口5,000~8,000，发展上限35,000，周边4个部落 |
| **小国** | 20 | 初始人口3,000~5,000，发展上限25,000，周边3个部落 |
| **合计** | **50** | 12+18+20 = 50 |

**国家类型分布**：
- **西域三十六国（普通国）**：男女比55:45，覆盖四大地理区域
- **全女性国家（6个）**：西梁女国🌸(大国)、东女国👸(大国)、苏毗女国(中等国)、阿玛宗⚔️(中等国)、弱水国(小国)、斯基泰🏹(中等国)
- **母系社会国家（8个）**：瑶池国(大国)、月氏国(大国)、玄女国(中等国)、赤谷国(中等国)、青鸾国(中等国)、素女国(小国)、灵山国(小国)、玉河国(小国)

> **部落规则**：
> - **数量**：大国周边5个部落、中等国周边4个部落、小国周边3个部落，无例外。
> - **制度一致**：每个周边部落的制度与所属城邦一致。普通城邦的周边部落为普通部落，母系城邦的周边部落为母系部落，全女性城邦的周边部落为全女性部落。
>
> **部落总计**：(12×5) + (18×4) + (20×3) = 60+72+60 = **192个部落**

### 2.2 三大国家类型与政治制度

> 西域五十国按社会性别结构分为**三大国家类型**：普通国家（36国）、母系国家（8国）、全女性国家（6国）。每种类型有独特的政治制度、人口构成和社会规则。三种类型之间可通过特定剧情事件进行**制度转型**。

#### 2.2.1 三大类型对比

| 维度 | 普通国家（36国） | 母系国家（8国） | 全女性国家（6国） |
|------|-----------------|----------------|------------------|
| **人口构成** | 初始男女比55:45，自然出生比50:50，**无性别比例约束** | 初始男女比30:70，自然出生比30:70，**性别比例红线40:60**（男女比不得超过40:60，无论公民还是奴隶） | 100%女性（0:100），公民与奴隶必须为女性 |
| **统治者性别** | 男性或女性均可 | 必须为女性 | 必须为女性 |
| **政治制度** | 9种（城邦制/贵族制/部落制/汗国制/市政制/议会制/君主制/军镇制/酋长制） | 6种（母系城邦制/母系部落制/母系君主制/母系议会制/母系贵族制/母系神女制） | 4种（全女性君主制/全女性贵族制/全女性部落制/全女性神女制） |
| **征兵规则** | 仅男兵 | 70%男兵+30%女兵 | 仅女兵（战力+10%） |
| **经济模式** | 5种（商会/行会/牧场/庄园/屯田） | 女性产出+20%，纺织/酿造+20% | 重体力产出降低，精细产出不变 |
| **外交联姻** | 男方向女方下聘 | 女方主导（效果+50%） | 姐妹联姻/外交通婚 |
| **可走路线** | A/B/C/E/F + D（凤鸣觉醒解锁） | A/B/C/D/E/F（全部六条） | A/B/C/D/E/F（全部六条） |

> **关于普通国家的女性统治者**：普通国家虽以男性主导为主，但**不排斥女性担任最高统治者**（女王/女帝/女酋长等）。历史上西域诸国确有女性执政的先例。普通国家允许女性统治者的设计意义在于：当玩家后续触发「凤鸣觉醒」剧情时，若当前统治者恰好为女性，则转型过程更加自然顺畅；若当前统治者为男性，则需通过**退位/禅让**剧情——将王位传给女性角色（如公主、女性文官/武将等），再完成制度转型。

#### 2.2.2 制度转型体系

> 三大国家类型之间存在**有向的转型路径**，构成游戏中最具戏剧性的制度变革系统。转型一旦完成，大多不可逆转。
>
> **核心原则**：所有制度转型规则均针对**主动转型**（如凤鸣觉醒）或**征服者视角**（如父权复辟、女儿当国、凤归母巢）。被征服国家**无权选择**——只能接受被同化（反抗行为视为征服者同化成本的一部分）。

```
制度转型路径图（箭头表示可转型方向）：

  普通国家 ──[凤鸣觉醒]──→ 母系国家 ──[女儿当国]──→ 全女性国家
      ↑                      │                         │
      └──────[父权复辟]───────┘                         │
                              ↑                        │
                              └─────[凤归母巢]──────────┘

  注意：全女性→普通 = 不可能（信仰崩溃）
```

| 转型事件 | 方向 | 可逆性 | 触发阶段 | 核心条件 |
|----------|------|--------|----------|----------|
| **🌸 凤鸣觉醒** | 普通→母系 | 未执行过强制同化时可逆 | 阶段四起 | 女性公民>70%连续36回合 + 与全女/母系国有接触 + 稳定度≥70（触发"母仪天下"事件） |
| **🔄 父权复辟** | 母系→普通 | 不可逆 | 阶段四起 | 占领普通国并强制同化为普通（触发"父权反扑"事件）+ 稳定度>70 |
| **🔥 女儿当国** | 母系→全女 | 不可逆 | 阶段四起 | 与全女国有接触 + 占领任何国家并强制同化为全女（触发"姐妹同心"事件）+ 稳定度>70 |
| **💧 凤归母巢** | 全女→母系 | 不可逆 | 阶段四起 | 与母系/普通国有接触 + 占领任何国家并强制同化为母系（触发"凤引鸾归"事件）+ 稳定度>70 |

#### 2.2.3 制度转型详细规则

##### 🌸 凤鸣觉醒（普通→母系）

> 普通国家通过人口结构调整和社会变革，转型为母系国家。这是普通国家解锁D线（凤鸣昆仑）的唯一途径。

**触发条件**（全部满足）：
1. 游戏已进入**第四阶段**（丝路争霸）或之后任意阶段
2. 公民女性人口比例连续**36回合（1年）**超过70%
3. 玩家在外交/军事上与全女性/母系国家有接触（如联姻、联盟、占领、掠夺、献礼等至少一项）
4. 国家稳定度≥70
5. **仅能触发一次**——无论玩家选择接受或拒绝，此后不再出现

**触发时机**：阶段四开始后，任意时间满足条件均可触发（玩家选择强制同化前后均可）。凤鸣觉醒**仅可执行一次**，根据触发时机的不同，效果如下：

- **强制同化触发前执行凤鸣觉醒**：普通国家 → 母系国家。转型后，后续可根据母系国家的规则触发强制同化（同化为普通/母系/全女，根据国家同化的制度要求）
- **强制同化触发后执行凤鸣觉醒**：母系国家转化的普通国家 或 原生普通国家 → 母系国家。此后国家**不可逆地成为母系国家**

若玩家触发强制同化制度后选择凤鸣觉醒，后续一律视为**强制同化为母系制度**（覆盖此前任何制度选择）。

**统治者处理**：
- 若当前统治者为**女性**：直接转型，无额外剧情
- 若当前统治者为**男性**：触发「退位/禅让」剧情——男性统治者将王位传给女性角色（公主、女性文官/武将等由玩家选择），再完成制度转型

**人口与奴隶处置**（转型时立即执行，按顺序分两步）：

**第一步：男性公民→男奴（优先执行）**

- 即：男性公民中超出40%比例的部分，强制转化为男奴

**第二步：男奴→玩家选择处置方式（后执行）**

- 女奴：全部保留，转型后按母系国家奴隶规则运行

##### 隐藏剧情：凤鸣觉醒（阶段四触发）

> 普通国家在特定条件下可触发隐藏剧情，转变为母系国家——这是游戏中最具戏剧性的制度转型之一。

**触发条件**：玩家进入**阶段四**时，满足以下全部条件：
1. 玩家所属普通国家的公民女性比例连续1年（36回合）超过70%
2. 触发原因包括但不限于：战争导致男性人口大量损失、引入母系/全女性国家人口政策、外交联姻/通婚结构调整等
3. 玩家已完成至少1条全女系或母系盟友的故事线

**剧情事件**：「凤鸣觉醒」

> 当女性比例持续超过70%长达一年后，国内女性势力抬头——贵族女性、商人女性、将军女性开始联合请愿，要求改革现有制度为母系制。她们提出的理由是：既然国家人口结构已经女性占绝对多数，政治权力也应随之转移，女性掌权将更符合国情、提升效率。

**玩家选择**：
- ✅ **接受转型**：将普通国转为母系国（未执行过强制同化时可逆）
- ❌ **拒绝转型**：维持现状，但女性人口持续下降（部分女性移民全女国/母系国），且女性武将忠诚-15

**接受后的效果**：
| 维度 | 效果 |
|------|------|
| **制度变更** | 国家转为母系制度，男女比从55:45变为30:70（女性掌权） |
| **同化豁免** | 已占领国家一律免除同化成本 |
| **新占领规则** | 新占领国家统一按母系规则运行 |
| **路线解锁** | 触发「母系联盟」胜利路线（线路D）：以母系国家领袖身份联合全女国/母系国完成统一 |
| **外交加成** | 与全女国/母系国外交好感自动+25，与之相反的普通国家好感-15 |
| **可逆性** | 未执行过强制同化的情况可逆 |

> 设计意图：凤鸣觉醒为普通国家玩家提供了一条独特的转型路线——无需开局选择母系国家，而是通过人口结构调整和剧情选择自然转型。这既增加了游戏的重玩性，也丰富了"母系联盟"路线的玩家来源。

| **推荐开局(全女)** | 西梁女国(友善开放)、东女国(中立平衡)——以宗主之力弥补军事短板 |
| **推荐开局(母系)** | 瑶池国(文化外交)、月氏国(草原外交)——利用宗主军事援助快速扩张 |
| **胜利条件** | ①藩屏之基(50国统一) + ②宗主之令(臣服≥60回合) + ③代主征伐(≥15国) + ④宗主之愿(满足宗主核心诉求) + ⑤终极抉择(三选一) |
| **独特结局** | 🏯 藩王建国（独立）→西域藩王 / 🐉 忠藩世家（继续称臣）→西域都护 / ⚔️ 反噬宗主→西域霸王 |
| **胜利称号** | 西域藩王/西域都护/西域霸王 |

### 3.3 凤鸣觉醒（普通国→D线解锁）

> 「凤鸣觉醒」是普通国家通往凤鸣昆仑(D线)的隐藏剧情路径——通过人口结构调整和剧情选择，普通国可转型为母系国家，从而解锁D线。详见 [07-剧情-故事线.md](07-剧情-故事线.md) 阶段四·凤鸣觉醒事件。

| 维度 | 说明 |
|------|------|
| **触发时机** | 阶段四开始时判定（需提前布局人口政策） |
| **触发条件** | 国家女性人口比例连续1年（36回合）超过70% |
| **转型效果** | 国家转为母系制度，男女比从55:45变为30:70（女性掌权） |
| **同化豁免** | 已占领国家一律免除同化成本 |
| **新占领规则** | 新占领国家统一按母系规则运行 |
| **路线解锁** | 触发「母系联盟」胜利路线(D线)：以母系领袖身份联合全女国/母系国完成统一 |
| **外交加成** | 与全女国/母系国外交好感自动+25，与普通国家好感-15 |
| **可逆性** | 未执行过强制同化的情况可逆 |

> **设计意图**：凤鸣觉醒为普通国家玩家提供了独特的转型路线——无需开局选择母系国家，而是通过人口结构调整和剧情选择自然转型。这既增加了游戏的重玩性，也丰富了D线的玩家来源。
