## 一、十六种资源商品总览

> **设计原则**：精简为16种核心资源/商品（8基础+8贸易），每种都有完整的获取→存储→消耗闭环。所有商品和奴隶均可通过市场贸易购买，不同市场有品类和价格差异。

## 三、区域独特特产与专属特产贸易规则

> **核心规则**：区域独特特产（四大区域各3种共12种）和专属特产（全女国/母系国/特殊城邦产出）**仅在城邦所在区域的贸易市场出售**，不可跨区域直接购买。跨区域获取需通过商路贸易或外交手段。

| 特产类型 | 贸易限制 | 获取方式 |
|----------|----------|----------|
| **16种标准资源** | 无限制，任何贸易市场均可购买 | 直接购买/商路贸易/外交 |
| **区域独特特产**（12种） | 仅在产出区域的贸易市场出售（详见四、贸易市场体系） | 在对应区域市场购买，或通过跨区域商路贸易获取（运输成本+20%），或通过外交赠礼/联姻/战利品获取 |
| **专属特产**（19+8种） | 仅在产出城邦所在区域的贸易市场出售 | 在对应区域市场购买，或通过外交赠礼/联姻获取 |

**跨区域获取特产的途径**：
- **商路贸易**：建立连接两个区域的商路后，可通过商路自动运输区域特产（需支付运输成本+20%）
- **外交赠礼**：其他国家赠送的特产不受区域限制
- **联姻聘礼**：联姻获得的特产不受区域限制
- **战利品**：征服城邦后获得的库存特产不受区域限制

### 4.2 跨区域贸易与商路系统

> **核心机制**：玩家通过建立**商路**连接不同区域的市场，实现跨区域资源流通和特产运输。

| 商路类型 | 连接方式 | 建立成本 | 效果 |
|----------|----------|----------|------|
| **区域内商路** | 同区域两个市场之间 | 200金+2马匹 | 两市场间贸易价格-10%，每月自动运输标准资源 |
| **跨区域商路** | 不同区域两个市场之间 | 500金+5马匹+3骆驼 | 两市场间贸易价格-10%，可运输区域特产（运输成本+20%） |

**商路收入公式**：
```
每月商路收入 = 基础收入(50金) × 商路等级 × 市场等级系数 × 季节修正
  · 商路等级：区域内×1.0，跨区域×1.5，丝路主干×2.0
  · 市场等级系数：★★★★×1.2，★★★×1.0
  · 季节修正：夏季+10%，冬季-20%
```

## 六、生产-消耗平衡发展曲线

> **设计原则**：资源生产与消耗在国家发展过程中必须形成动态平衡。以下平衡曲线展示了不同发展阶段（城邦等级、人口规模）下各项核心资源的净产出变化趋势，帮助玩家理解何时需要调整经济结构，避免因失衡导致的恶性循环。

### 6.1 核心资源人均净产出公式

```
月度资源净产出 = 总产出 - 总消耗

【粮食人均净产出】
粮食净产出 = 农民产出 + 牧民产出 + 渔猎产出 + 贡赋 + 进口
             - 男公民粮耗(1×男公民数) - 女公民粮耗(0.8×女公民数)
             - 奴隶粮耗(0.6×奴隶数) - 士兵粮耗(1.5~1.8×士兵数)
             - 马匹粮耗(2×马匹数) - 酿酒消耗(2粮/美酒)

【金币人均净产出】
金币净产出 = 税收 + 商路收入 + 贸易收入 + 舞女收入 + 部落贡赋 + 战利品
             - 军费(兵力/100金) - 俸禄(10~50×武将数) - 建筑维护 - 铁匠铺燃料
             - 一次性支出摊销(建设/征兵/外交/购买等)

【工业资源净产出】
铁矿净产出 = 矿工产出(男+2/人,女+1.5/人) + 男劳奴产出(+2/人) - 锻造消耗(2铁/兵器) - 市场出售
木材净产出 = 伐木工产出(男+3/人,女+2.5/人) + 男劳奴产出(+3/人) - 建筑消耗 - 铁匠铺燃料(5木/月)
石材净产出 = 采石工产出(男+2/人,女+1.5/人) - 建筑消耗
兵器净产出 = 铁匠铺锻造(2铁→1兵) + 市场购买 - 征兵消耗(步×2/骑×1/弓骑×1) - 月度损耗(0.5%库存)
```

### 6.4 发展失衡预警指标

> 以下预警指标在城邦**每个季度末自动计算**，触发任一红色预警时系统将弹出提示。

#### 6.4.1 粮食安全预警

```
【粮食储备安全线】
  安全: 粮食储备 ≥ 月度粮食消耗 × 6 (半年口粮)
  警戒: 粮食储备 = 月度粮食消耗 × 3~6 (3~6个月，需关注)
  危险: 粮食储备 < 月度粮食消耗 × 3 (不足3个月，红色预警)

【不同城邦等级的最低粮食储备要求】
   Lv1: ≥500粮   Lv2: ≥2000粮   Lv3: ≥5000粮   Lv4: ≥15000粮   Lv5: ≥50000粮

【粮食危机连锁反应】
   粮食赤字持续1个月 → 人口增长-10%
   粮食赤字持续3个月 → 每月流失1%人口
   粮食赤字持续6个月 → 触发"饥荒"事件(人口-5%~10%，满意度-30%，叛乱风险+15%)
```

#### 6.4.2 经济平衡预警

```
【月度净收入安全线】
  安全: 月度净收入 > 月度总消耗 × 10% (每月有盈余)
  警戒: 月度净收入 ≈ 0 (收支平衡，无法应对突发事件)
  危险: 月度净收入 < 0 (持续赤字，需调整经济结构)

【储备金安全线】
  安全: 金库 ≥ 月度总消耗 × 3 (3个月缓冲)
  危险: 金库 < 月度总消耗 (不足以支撑1个月，破产风险)

【经济危机连锁反应】
   金币赤字持续3个月 → 武将忠诚-5/月
   金币赤字持续6个月 → 武将可能叛逃
   金库归零 → 强制停止所有建设/征兵/外交行动
   金库归零持续3个月 → 触发"财政崩溃"事件(士兵大量逃亡，满意度-40%)
```

#### 6.4.3 工业资源预警

```
【兵器储备安全线】
  安全: 兵器库存 ≥ 未来3个月预期征兵消耗 × 2
  危险: 兵器库存 < 未来1个月预期征兵消耗

【木材/石材储备安全线】
  建设期: 建筑消耗 > 月产出时，储备需 ≥ 下一个建筑计划总消耗 × 1.5
  平稳期: 储备 ≥ 月产出 × 3 即可
```

### 6.5 发展阶段的平衡调整策略

#### 6.5.1 Lv1→Lv2（农业期→建设期）

> **核心矛盾**：人口少、产出低，但建筑消耗大，农业人口减少导致粮食安全风险。

**推荐调整路径**：
```
1. 开局农业占比60%+，确保粮食盈余50%以上
2. 当人口>2000且粮食盈余>500/月时，开始转5%~10%农业人口为矿工/伐木工
3. 优先建设：农田(谷)→集市→民兵营(谷)→粮仓→金库
4. 【平衡检查点】每月末检查粮食储备是否>500，不足则暂时停止建筑建设
```

#### 6.5.2 Lv2→Lv3（建设期→平衡期）

> **核心矛盾**：基础设施建设基本完成，但手工业+商业尚未成为经济支柱，粮食自给率开始下降。

**推荐调整路径**：
```
1. 农业占比逐渐从55%降至45%，转出手工业(+5%)+商业(+5%)
2. 当城市等级达到Lv3后，手工业占比目标22%，商业占比目标15%
3. 建立第1条商路（区域内商路优先，成本低见效快）
4. 【平衡检查点】粮食自给率>100%时安全转产；<90%时需先建灌溉渠或进口协议再转
```

#### 6.5.3 Lv3→Lv4（平衡期→扩张期）

> **核心矛盾**：军事扩张导致军费+兵器消耗激增，人口增长导致粮食消耗增加，经济重心从自给转向贸易依赖。

**推荐调整路径**：
```
1. 农业占比降至35%~40%，接受粮食自给率降至80%~90%（用贸易收入进口补充）
2. 商业占比提升至18%+，手工业占比20%+，形成"出口创汇→进口粮食"循环
3. 建立至少2条跨区域商路(确保货物供应)或1条丝路主干(超级经济引擎)
4. 储备兵器≥200，马匹≥100（支撑至少一次大规模扩张）
5. 【平衡检查点】月度净收入>200金后方可开启军备扩张；金库储备>3000金
```

#### 6.5.4 Lv4→Lv5（扩张期→霸权期）

> **核心矛盾**：人口规模庞大(12000+)，一城经济无法完全自给，需多城协同+贸易网络维持平衡。

**推荐调整路径**：
```
1. 农业占比稳定在25%~35%（不可低于25%），粮食依赖进口30%~45%
2. 商业占比20%~25%+，将经济重心完全转向贸易+服务业
3. 建立≥5条商路（含≥1条丝路主干），形成覆盖西域的贸易网络
4. 手工业占比根据国家类型调整：普通国15%~18%/母系国25%~30%/全女国30%~35%
5. 建立"多城分工体系"：主城专精商业+手工业，分城专精农业+矿业
6. 【平衡检查点】每季度检查粮食储备是否>50000，不足则增加农业配比或进口协议
```

### 6.7 关键平衡公式速查

```
【粮食及格线】
  粮食净产出 ≥ 0 ⇔ 农民占比 ≥ 粮食消耗 ÷ (农民人均产出 × 总人口)

【简单算法】
  每100人口需要 = 农业人口占比 × 农民人均产出 ≥ 人均粮食消耗
  普通国人均消耗 ≈ 0.6~1.0粮/人/月(含士兵+奴隶)
  全女国人需消耗 ≈ 0.5~0.8粮/人/月(女粮耗低)

【金币及格线】
  月度净收入 > 0 ⇔ 商业收入 + 手工业产出 + 税收 > 军费 + 俸禄 + 建筑维护

【征兵可持续线】
  兵器月消耗 ≤ 兵器月产出 + 兵器月进口量
  马匹月消耗 ≤ 马匹月产出 + 马匹月进口量
```
