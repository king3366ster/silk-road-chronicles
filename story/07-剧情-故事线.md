# 西域群英传 — 剧情·任务线

> **文档版本**：v2.1 | **最后更新**：2026年5月
> **所属游戏**：西域群英传 - Silk Road Chronicles（RPG + 策略战棋）

---

## 一、主线剧情结构

### 1.1 故事背景

西汉末年至东汉初年，丝绸之路刚刚打通，西域50国林立，汉匈争霸的余波未平。玩家扮演一个无名小卒，从一穷二白开始，通过政、商、军三条路线逐步崛起，最终统一西域诸国。

### 1.2 六阶段主线进程

> **保护期设计**：前两个阶段为保护期，别国、敌国、外部势力均不会参与，不会有国家攻击或掠夺玩家。保护期让玩家有充足时间学习游戏机制、发展经济、完成故事线。

#### 阶段一：城邦初立（通用阶段·保护期·回合1-12）

> **核心特征**：故事线按国家制度分为三类（全女性/母系/普通），同类国家的故事线完全相同。完成第一阶段后，玩家获得城邦管理权，解锁基础技能。

```
三类通用故事线：
  全女性国家 → 🌸"女儿建国"线：①母神启示 ②姐妹同心
  母系国家   → 💎"母系传承"线：①女祖之训 ②男女共治
  普通国家   → 🏜️"部落崛起"线：①草原/绿洲之梦 ②领主之路

完成奖励：城邦管理权 + 基础资源包 + 1名武将 + 解锁资源采集/人口管理
阶段结束：获得城邦管理权 → 自动进入第二阶段
```

#### 阶段二：部落征途（个性化阶段·保护期·回合13-36）

> **核心特征**：从第二阶段开始，每个城邦的故事线完全个性化。玩家需完成城邦专属故事线（大国3条/中等国2条/小国2条），再逐一触发周边部落故事线（每个部落1条）。

```
关键节点：
  1. 城邦专属故事线：大国3条/中等国2条/小国2条（深度个性化）
  2. 周边部落故事线：大国5条/中等国4条/小国3条
  3. 城市等级提升至Lv2

解锁内容：奴隶系统、农业升级、商业系统、马场、税收、锻造
阶段结束：所有周边部落已征服/联盟 → 保护期结束 → 进入第三阶段
```

#### 阶段三：城市发展（开放竞争阶段·回合37-72）

> **核心特征**：保护期结束，外部势力开始介入。玩家可攻略其他国家/部落，但必须先触发目标的个性化故事线（大国3条/中等国2条/小国2条/部落1条）。
>
> **同化规则**：占领/同盟其他国家时，**默认要求立即同化**（将目标转为自身制度），并付出一次性同化成本。此阶段的同化是"预选"——第四阶段若选择双制度并行，则已同化城邦恢复原有制度；若选择统一同化，则一律按最新选择执行。
>
> **第三阶段同化成本减免**：
> - **全女性国家**：免同化成本（文化同化力强，同化自然完成）
> - **母系国家**：免同化成本（外交联姻机制使同化和平顺畅）
> - **普通国家**：正常支付（跨族×0.5金/人，全女↔普通×1金/人）
> - **注意**：进入第四阶段后统一按标准成本执行，不再享受减免，但不需要补缴第三阶段已减免的费用

```
触发条件（满足任一即进入，否则仍停留在第二阶段）：
  ① 城市等级≥Lv3
  ② 主动攻击周边国家/部落
  ③ 开展外交事件（联盟/联姻/宣战）
  ④ 开辟商路

攻略规则：先触发目标故事线 → 完成后才能军事征服或外交联盟
解锁内容：外交系统、征兵系统、科技研发、宣战、商路、谍报、外部势力介入
```

#### 阶段四：丝路争霸（同化决策阶段·回合73-120）

> **核心特征**：占领/同盟≥2个国家时进入。需面对"同化决策"——如何处理新征服国家的制度差异。此阶段可灵活选择（含双制度并行），但需承担月度管理成本。

```
同化决策核心规则：
  ① 三种制度与性别比例
     制度 | 男女比 | 主导性别 | 核心特征
     ----|-------|---------|---------
     **普通** | 55:45 | 男性（官方/军事/外交领域主导） | 男女分工明确，女性可担任文官/商贾，男性主导军事和政务
     **母系** | 30:70 | 女性（全面掌权） | 女性占据军政商要职，男性从事农牧/手工业等生产劳动
     **全女性** | 0:100 | 女性（纯女社会） | 无男性存在，通过特殊的繁殖方式维持人口，需外部进口重体力劳动产品

  ② 不可逆原则与女化回路
     制度转换遵循以下规则：
     · **普通→母系**（女化之路事件）：不可逆
     · **普通→全女性**：不可能 → 必须经由母系过渡
     · **母系→普通**（母系归凡事件）：可逆 → 仍可通过(女化之路事件)再次转为母系，此后不可逆
     · **母系→全女性**（母系蜕变事件）：不可逆
     · **全女性→母系** (女儿归母事件)：不可逆
     · **全女性→普通**：不可能 → 全女性国家无法转为普通社会

  ③ 女化之路事件（普通→母系）
     · 触发条件：女性人口连续12回合（1年）>70%，且本国已占领至少1个母系/全女国，稳定度≥70
     · 转型效果：男女比从55:45→30:70，女性全面掌权，解锁D线·西域女帝
     · 不可逆：一旦通过女化之路转为母系，无法通过母系归凡事件回到普通
     · 女化后征服规则：已女化的前普通国按母系征服者规则处理，被征服国家一律强制同化为母系（不可逆）

  ④ 母系归凡事件（母系→普通）
     · 触发条件：母系国家稳定度>90 + 女性人口占比降至50%以下 + 父权派势力崛起事件
     · 转型效果：男女比从30:70→55:45，男性重新掌权，失去D线资格，转为A/B/C/E线
     · 可逆：归凡后的普通国仍可通过女化之路事件再次转为母系
     · 二次锁定：女化之路→母系→归凡→二次女化→母系后，永久锁定为不可逆（"女权觉醒不可二度压制"）
     *注意：女化之路（普通→母系）本身不可逆，本规则仅适用于未经过女化之路的原生母系国家归凡后二次女化的情形*

  ⑤ 母系→全女性路径（母系蜕变事件）
     · 触发条件：母系国家女性人口连续6回合>90% + 文化科技≥Lv3 + 占领≥1个全女国
     · 转型效果：男女比从30:70→0:100，彻底消除男性人口，成为全女性社会
     · 不可逆：一旦转为全女性，无法通过任何方式转回母系或普通
     · 全女性国家的人口维持依赖特殊机制（繁衍泉水/子母河水等）

  ⑥ 全女性→普通/母系：不可能
     · 全女性社会基于"无男纯净"的宗教信仰，转为母系/普通等同于信仰崩溃
     *全女性国家被普通国征服时：只能强制同化为全女，或双制度并行*

  ⑦ 双制度并行：各城邦独立发展，人口性别比例加权汇总，每月需付出额外管理成本

  ⑧ 强制统一：占领/同盟≥5国时，必须为每座城邦选择统一制度，游戏进入第五阶段，并基本确定ABCDE剧情路线

同化决策（征服者类型 × 被征服者类型，共3×3=9种组合）：
  【1. 普通国征服者】
    普通→普通（同族）：①同化为普通（免成本）→走A/B/C/E线
    普通→母系（跨族）：①双制度并行 ②同化为普通
    普通→全女（跨族）：①双制度并行 ②同化为普通
    普通→[已女化]母系：①一律强制同化为母系（不可逆，解锁D线·西域女帝）
    *普通国一旦触发女化之路转型，即成为母系国，此后所有征服一律按母系征服者规则处理

  【2. 母系国征服者】
    母系→母系（同族）：①同化为母系（免成本）→走D线
    母系→普通（跨族）：①同化为母系 ②双制度并行 ③同化为普通（触发"母系归凡"剧情→失去D线，转为A/B/C/E线）
    母系→全女（跨族）：①同化为母系 ②双制度并行 ③同化为全女（触发"母系蜕变"剧情→转为全女性国家·D线强化）

  【3. 全女性国征服者】
    全女→全女（同族）：①同化为全女（免成本）→走D线
    全女→母系（跨族）：①同化为全女 ②双制度并行 ③同化为母系（触发"女儿归母"剧情→转为母系国·D线强化）
    全女→普通（跨族）：①同化为全女 ②双制度并行 ③同化为母系

双制度并行成本（金钱+治安双维度，数值已根据资源产出综合平衡）：
  · 月度管理成本（金钱）：每座异制度城邦+50金/月（三制度+80金/月）
  · 治安惩罚：每座异制度城邦治安度-5/月（三制度-8/月）。治安<60触发暴动风险，<30必发叛乱
  · 治安恢复：每驻扎100兵恢复治安+5/月；每座文化建筑恢复治安+3/月（需额外投入金币和兵力）
  · 效率惩罚：贸易收入-5%/异制度城邦，征兵效率-5%/异制度城邦
  · 强制统一（≥5城邦）：每座城邦一次性同化代价 = 人口数 × 制度差异系数
    - 同族制度×0（如母系→母系）：免成本
    - 跨族制度×0.5金/人（如母系→普通、母系→全女性）：代价中等
    - 全女性↔普通×1金/人：代价最高（涉及根本性社会结构变革）
    - 一次性代价代表行政重组+文化教育+人口迁移+社会秩序重建的综合投入

同化资金不足的后果：
  · 资金充足 → 正常同化，城邦纳入正常管理体系
  · 资金不足（差额≤30%）→ 强制借贷：向粟特商会借贷补足，偿还本金+50%利息
  · 资金不足（差额>30%）→ 城邦进入"失控"状态：
    - 治安固定20（极低），每月80%暴动概率
    - 城邦产出归零（无税收/资源/贡赋）
    - 无法征兵
    - 每月+5%累积叛乱概率，超100%必叛（城邦独立→敌对势力）
    - 周边国家好感-10/月
    - 恢复条件：支付原价×1.5同化成本后恢复正常

分支路线触发与详细说明：

   所有路线最终胜利条件统一为"统一50国"，统一方式包括三种：
     · 同盟（外交协议）：目标国保持完全自治，与玩家建立军事同盟关系。**对玩家不产生任何收益**（无贡赋、无资源、无军事调遣权），仅代表外交上的统一。不需要同化，无同化成本
     · 附庸（军事/经济压力）：目标国保留内政权，**每年将自身产出的30%供奉给玩家**（金币+粮食+特产）。但**不服从军事调遣**（不可调遣附庸国军队参战）。不需要同化，无同化成本
     · 占领（军事征服）：目标国完全纳入玩家领土，玩家获得全部产出和控制权。需要同化，需支付同化成本
     · 同盟国+附庸国+占领国均计入统一进度（**联姻国不计入**），50国全部统一即胜利

  A·军事征服线：全部"强制同化" + 军事力量最强（倾向：占领为主，必须全部直接统治）
    核心机制：强制征兵(+30%征兵量)、军事威慑(10倍兵力→自动投降)、战利品系统、附庸体系
    路线加成：军事行动效率+20%，战利品+30%
    进程：铁血扩张(2-4国) → 联军对抗(5-10国) → 最终征服(11+国)
     胜利条件：占领全部50个城邦（必须直接统治，不可使用附庸）

   B·商业帝国线："保留制度"为主 + 商业收入最高（倾向：附庸/经济附庸为主，同盟为辅）
     核心机制：商路垄断(控制节点收过路税)、经济制裁(禁运→对方收入-60%)、债务控制(债务>80%GDP→经济附庸)、特产垄断(80%产量→定价权×1.5)、丝路网络(连接≥30国→全贸易+30%)
     路线加成：商业收入+30%，经济制裁效果+50%
     进程：商路开拓(2-4国) → 经济霸权(5-10国) → 商业决战(11+国)
     关键决策：垄断vs开放、制裁vs合作、债务陷阱风险
     胜利条件：占领≥30个城邦 + 控制丝路80%贸易额，剩余20国通过商业同盟统一

   C·外交文化线："保留制度"为主 + 外交关系最广（倾向：同盟/联姻为主，附庸为辅）
     核心机制：文化输出(文化建筑→+5影响力/月)、文化认同(影响力≥100→自愿加入)、联姻联盟(血盟→不可互攻+贸易+20%)、文化大使(魅力≥70武将→+3/月)、文化事件链
     路线加成：文化传播速度+50%，联姻成功率+30%
     进程：文化传播(2-4国) → 文化联盟(5-10国) → 文化统一(11+国)
     关键决策：文化融合vs纯输出、联姻对象选择、和平vs有限战争
     胜利条件：占领≥16个城邦 + ≥36国文化认同，剩余34国通过联姻/联盟统一

   D·母系联盟线：全女/母系国家选择"转为全女/母系"为主（倾向：姐妹之盟+母系同化）
     核心机制：姐妹之盟(联姻+30%/贸易+15%/军事+20%)、母系同化(纺织/酿造+20%)、文明圈效应(每统一1国→全属性+3%，14国全统一→+25%)、父权反扑事件
     路线加成：母系/全女国家间联盟效率+30%，文明圈加成
     进程：姐妹联合(2-3个母系/全女国) → 文明圈建立(4-6个) → 母系霸权(14个全部)
     胜利条件：占领≥24个城邦（含全部14个母系/全女国）+ 征服/同化其余26国
     生产力互补：附庸国保留原制度，男性工人产出弥补全女性城邦重体力不足

   E·帝国纵横线：同化决策平衡 + 五大帝国关系均衡（倾向：同盟/联合体系为主）
     核心机制：平衡术(维持五方势力均衡)、帝国好感管理(五方全部≥60)、好感平衡机制(与一方+10→其对手-5)、西域议会(≥10国外交+≥5帝国友好→召开)、联合体系(50国联合→集体安全)
     路线加成：外交效率+30%，帝国好感获取+20%
     进程：外交起步(2-3个帝国) → 平衡之术(4-5个帝国友好) → 联合体系(五方全部友好)
     胜利条件：占领≥30个城邦 + 五大帝国全部"友好" + ≥36国加入联合体系
    帝国对国家的影响力机制（确保可玩性）：
       · 军事驻扎：汉朝驻军楼兰/车师(+500守军)，匈奴驻军乌孙/车师(+300骑兵)
       · 技术输出：汉朝→粮食+20%，匈奴→骑兵+15%，大夏→文化+20%
       · 经济渗透：大夏控制疏勒关税(抽成20%)，康居控制粟特金融
      · 政治干预：帝国好感≥80可触发政变事件
       · 军事入侵：好感≤20触发入侵（匈奴每12月/汉朝每24月/大夏每18月）
       · 文化同化：大夏佛教/汉朝儒学传播改变目标国属性
    关键决策：选边vs中立、挑拨vs调停、联合vs独立
```

#### 阶段五：帝国博弈（帝国博弈阶段·回合121-180）

> **核心特征**：占领/同盟≥5个国家时进入。五大帝国开始全面介入西域博弈，玩家需在帝国之间周旋，利用帝国矛盾获取利益，同时应对帝国军事威胁。此阶段帝国事件频发，多线作战成为常态。

```
帝国博弈核心机制：
  ① 强制统一制度：占领/同盟≥5国时，必须为每座城邦选择统一制度，支付一次性同化代价（同化成本详见阶段四）
  ② 帝国好感管理：五大帝国各有独立好感度，玩家行为影响各方好感
     - 与汉朝友好（+好感）：获得步兵技术、丝路便利
     - 与匈奴友好（+好感）：获得骑兵加成、草原通行权
     - 与康居友好（+好感）：获得贸易优惠、商路保护
     - 与大夏友好（+好感）：获得文化加成、佛教传播
     - 与羌人友好（+好感）：获得山地通行、药草资源
  ③ 帝国事件链触发（每12-24个月自动触发一次）：
     - 汉朝远征军介入 / 匈奴全面南侵
     - 大夏东扩提议 / 康居贸易战
     - 羌人部落联盟请求
  ④ 好感平衡机制：与一方帝国好感+10 → 其对手帝国好感-5
  ⑤ 帝国入侵条件：好感≤20触发（匈奴每12月/汉朝每24月/大夏每18月）
  ⑥ 帝国政变事件：帝国好感≥80可触发目标国政变

帝国博弈阶段解锁内容：
  - 五方势力全面介入（各帝国开始派遣使者、驻军、商队）
  - 帝国好感管理开放（可查看/影响五大帝国好感值）
  - 帝国事件系统激活（每季度至少1次帝国相关事件）
  - 西域议会提案（≥10国外交关系+≥5帝国友好→可召开）
  - 多线作战能力（同时应对多个战场）

## 二、五大主线剧情路线

> **五大帝国贯穿设计**：每条主线都受到不同帝国势力的深层影响，帝国的博弈关系贯穿始终，玩家的每一个重大决策都会影响与五大帝国的关系。
>
> **普通国家路线灵活性**：36个普通国家是唯一可以切换至全部五条路线的国家类型，具有最高的路线自由度：
> - 全女性国家（6国）无法走A线（无男兵无法大规模军事征服）和E线（缺乏外交灵活性）
> - 母系国家（8国）无法走A线（女性主导社会不宜大规模军事化）
> - **普通国家可走A/B/C/E全部四条主线**，外加通过**"女化之路"隐藏剧情**（人口结构调整至女性>70%）解锁D线（母系联盟），实现五条路线全覆盖
>
> 开局国家选择决定了路线的初始倾向，但玩家可通过游戏中的战略决策（经济积累/外交关系/军事扩张/人口政策等）逐步切换路线。详见 [01-世界观.md 2.7.4 五大分支路线](01-世界观.md)

### 2.1 线路A：军事征服线 ⚔️

**主题**：以武力统一西域，建立军事帝国
**触发**：选择好战型国家（乌孙、车师、阿玛宗等）开局时激活
**帝国影响**：**汉朝🐉 vs 匈奴🐺** 的丝路控制权争夺贯穿全线。玩家必须在两大死敌之间选边站，或利用双方矛盾渔利。

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权
     【帝国事件】匈奴骑兵首次骚扰边境，汉朝使者首次到访
  2. 部落征途（回合13-36）：征服周边部落→建立军事基础→遭遇强敌
     【帝国事件】匈奴要求进贡或结盟；汉朝提供初级军事技术
     【关键抉择】投靠匈奴（获得骑兵加成但失去汉朝好感）vs 效忠汉朝（获得步兵技术但匈奴敌意增加）
  3. 铁血征途（回合37-72）：组建正规军→首次攻城→吞并邻国
     【帝国事件】汉朝都护府兴衰影响军事格局；匈奴南北分裂带来机遇
     【关键抉择】利用匈奴内乱北伐 vs 趁汉朝衰弱独立扩张
  4. 西域霸主（回合73-120）：多线作战→击溃联军→称霸西域
      【帝国事件】康居部落联盟从西北施压；羌人从南部骚扰
     【关键抉择】先北后南（先击匈奴再平羌人）vs 先南后北
  5. 一统天下（回合121+）：最终决战→统一西域→建立帝国
     【帝国事件】汉朝派遣远征军或承认你的霸权；匈奴最终决战
     【结局】军事征服胜利→西域霸王
```

### 2.2 线路B：商业帝国线 💰

**主题**：以商业力量控制西域，建立丝路帝国
**触发**：选择疏勒、姑墨等商贸国家开局时激活；或在其他线路中商业声望达到"传奇"后可切换
**帝国影响**：**康居🐂 vs 大夏🏛️** 的贸易主导权争夺贯穿全线。玩家在两大势力之间周旋，争夺丝路贸易控制权。

```
剧情进程：
   1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权

   2. 丝路开拓（回合13-36）：开辟新商路→建立贸易站→遭遇商路匪患

      【帝国事件】大夏试图垄断中转贸易；康居推广佛教换取商业特权

      【帝国事件】汉朝丝路需求激增；大夏提供南方商路通道
      【关键抉择】垄断东向贸易（与汉朝合作）vs 垄断西向贸易（与康居/大夏博弈）
   5. 丝路之主（回合121+）：垄断丝路贸易→经济统一西域
      【帝国事件】五大帝国都试图分一杯羹，玩家以经济力量制衡各方
      【结局】经济霸权胜利→丝路之主
```

### 2.3 线路C：外交文化线 📜

**主题**：以文化和外交手段统一西域
**触发**：选择龟兹、于阗、西梁女国等文教国家开局时激活
**帝国影响**：**大夏🏛️ + 羌人⛰️** 作为文明与山地势力在局部区域施加影响。玩家通过文化融合和外交手段化解帝国压力。

```
剧情进程：

### 2.4 线路D：母系联盟线 🌸

**主题**：以母系/全女性文明圈为核心，建立女性主导的西域秩序
**触发**（三选一）：
- **母系/全女国家**：选择阿玛宗、西梁女国、东女国、瑶池国等母系/全女开局时自动激活
- **普通国家→女化之路**：普通国玩家在**阶段四**触发隐藏剧情「女化之路」——当国家女性人口比例连续1年（36回合）超过70%后，触发女性政治势力崛起事件，可选择接受转型为母系国家（不可逆），从而解锁线路D
- **路线切换**：在线路B/C/E中完成母系/全女联盟≥3国后可切换
**帝国影响**：**羌人⛰️** 是母系联盟最重要的外部关系，昆仑山南麓的通道控制权决定母系国家的命运。同时**汉朝🐉**对母系/全女社会持好奇与警惕并存的态度。

> **女化之路关键规则**：
> - 触发时机：阶段四开始时判定（非立即触发，玩家需提前布局人口政策）
> - 转型后效果：男女比从55:45变为30:70（女性掌权）；已占领的非普通国家免除同化成本；与全女/母系国外交好感+25；普通国好感-15
> - 不可逆：一旦转型为母系，无法恢复为普通国体制
> - 设计意图：为普通国家玩家提供"性别制度转型"这一独特叙事体验，无须开局选母系国也能走D线

```
剧情进程：
  1. 无名小卒（回合1-12）：抵达城邦→通过考验→获得管理权

     【关键抉择】接受汉朝册封（获得技术援助但失去独立性）vs 保持独立（自由但发展受限）
  4. 巾帼霸业（回合73-120）：打破偏见→建立联盟→争夺霸权
      【帝国事件】大夏使者提议文化联盟；匈奴试图征服全女性国家
     【关键抉择】建立母系联盟（联合所有母系/全女性国家）vs 融入主流外交体系
  5. 女帝之路（回合121+）：统一西域→建立母系帝国
     【帝国事件】五大帝国见证女性力量的崛起
     【结局】全女性特殊胜利→女帝
```

## 四、部落系统

### 4.1 部落总览

> **核心规则**：
> - **数量**：大国周边5个部落、中等国周边4个部落、小国周边3个部落。总计(12×5)+(18×4)+(20×3) = 60+72+60 = **192个部落**。
> - **制度一致**：每个周边部落的制度与所属城邦一致。普通城邦的周边部落为普通部落，母系城邦的周边部落为母系部落，全女性城邦的周边部落为全女性部落。
>   - 例：楼兰(普通大国)的5个部落均为普通部落；西梁女国(全女性大国)的5个部落均为全女性部落；瑶池国(母系大国)的5个部落均为母系部落。
> - **任务**：每个部落有且仅有1条独特故事线，包含3个任务节点。完成部落故事线后解锁对应能力并获得该部落的资源贡赋。

| 国家类型 | 周边部落数 | 部落任务奖励 |
|----------|-----------|-------------|
| 大国（12个） | 5个 | 每个部落：资源贡赋+解锁能力+1名武将/文官 |
| 中等国（18个） | 4个 | 同上 |
| 小国（20个） | 3个 | 同上 |

### 4.2 部落任务与能力解锁

```
部落任务解锁进程：

部落任务1 → 解锁：奴隶买卖/资源采集/农业/马场
部落任务2 → 解锁：奴隶劳作/商业/税收/锻造
部落任务3 → 解锁：奴隶培训/高级能力

完成≥3个部落任务 → 进入城市发展阶段
完成目标国≥50%部落任务 → 可对该国宣战
```

### 4.3 部落类型

| 部落类型 | 互动方式 | 说明 |
|----------|----------|------|
| **独立部落** | 通过任务/外交/军事互动 | 有独立故事线，完成后提供贡赋 |
| **争议部落** | 多国共享，需竞争控制 | 完成任务可获得控制权 |
| **归附部落** | 通过外交或军事手段使其归附 | 自动成为己方资源来源，不可掠夺 |

| 结局 | 对应主线 | 触发条件 | 称号 | 核心评价维度 |
|------|----------|----------|------|-------------|

| **📜 文化同化** | 线路C | 占领≥16个城邦 + ≥36国文化认同 | 西域圣者 | 文化建筑、联姻次数 |
| **🌸 母系联盟** | 线路D | 占领≥24个城邦（含全部14个母系/全女国）+ 征服/同化其余 | 西域女帝 | 母系国家数量、女性武将 |

## 七、剧情-系统闭环验证

### 7.1 剧情-游戏系统对应关系

```
阶段一·城邦初立 ←→ 资源采集+人口管理+基础建设
  剧情：建立城邦的故事
  系统：教程阶段，学习基础操作
  闭环：剧情推动玩家学习系统，系统掌握推动剧情进展

阶段二·部落征途 ←→ 奴隶系统+农业商业+军事基础
  剧情：征服/联盟周边部落的故事
  系统：扩张阶段，学习军事和外交
  闭环：部落征服带来资源和人口，支撑进一步发展

阶段三·城市发展 ←→ 外交系统+科技+商路+谍报
  剧情：城市繁荣与外交博弈的故事
  系统：深化阶段，多系统协同运作
  闭环：外交关系影响商路收入，科技提升产出

阶段四·丝路争霸 ←→ 全面战争+贸易垄断+外交联盟+同化决策
  剧情：争夺西域霸权、处理同化议题的故事
  系统：高级阶段，所有系统全面运作
  闭环：军事力量保障商路安全，经济实力支撑军事消耗

阶段五·帝国博弈 ←→ 帝国好感管理+多线作战+帝国事件链
  剧情：在帝国夹缝中周旋博弈的故事
  系统：帝国博弈阶段，六大帝国全面介入
  闭环：帝国关系影响战略空间，多线作战考验资源调度

阶段六·一统西域 ←→ 多路径胜利条件
  剧情：统一西域的最终章
  系统：终局阶段，验证所有系统的积累
  闭环：玩家选择的发展路线决定最终结局
```

### 7.2 任务-资源闭环

```
任务奖励 → 资源获取 → 系统建设 → 新任务解锁

具体闭环：
  城邦任务 → 初始资源+管理权 → 基础建设 → 部落征途
  部落任务 → 资源贡赋+解锁能力 → 扩张发展 → 城市发展
  城邦剧情线 → 专属武将+永久加成 → 实力增强 → 扩张征伐
  随机事件 → 机遇/挑战 → 应对决策 → 影响发展路线
```
