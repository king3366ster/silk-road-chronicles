## 相较原 STORY.md 的修正

| 修正项 | 原文问题 | 修正内容 |
|--------|----------|----------|
| **国家数量** | 标题称"36国"实际50国 | 统一为"五十国"：36普通国(西域传统36国)+6全女国+8母系国=50国；原康居调整为帝国，新增多国覆盖四大地理区域 |
| **母系社会国家** | 原文仅6个母系国 | 重新设计为8个新母系国：瑶池国(大·南道)、月氏国(大·北道)、玄女国(中·南道)、赤谷国(中·北道)、青鸾国(中·葱岭)、素女国(小·南道)、灵山国(小·葱岭)、玉河国(小·葱岭) |
| **全女性国家** | 原文3个全女国 | 扩展为6个：新增苏毗女国(中·南道)、弱水国(小·南道)、斯基泰(小·葱岭)；东女国升格为大国，阿玛宗升格为中等国 |
| **外部势力扩展** | 六大帝国 | 调整为五大帝国：汉朝(东)、匈奴(北·东北)、康居(西北)、大夏(西·西南)、羌人部落联盟(东南)，南边昆仑山脉阻隔无路可通 |
| **主线剧情扩展** | 原文4条主线 | 扩展为5条主线（新增E帝国纵横线），五大帝国影响贯穿各线 |
| **全女性资源规则** | 原文"木材铁矿无法生产" | 修正为可采集但产出降低：木材83%、铁矿75%、石材75%，精细产出不受影响 |
| **资源种类** | 原文15种（7基础+8贸易） | 修正为16种（8基础+8贸易），新增"棉花"作为基础资源，确保所有城邦核心特产均为标准资源 |
| **额外特产体系** | 原文无独立体系 | 新增27种额外特产：4种区域独特特产+6全女国专属+8母系国专属+10小国城邦专属（含精绝古物），形成跨区域贸易互补闭环 |
| **贸易市场体系** | 原文无贸易市场 | 新增9大贸易市场（四大区域分布），标准资源可跨区域购买但区域独特特产和专属特产仅限本区域市场出售 |
| **奴隶市场体系** | 原文无奴隶市场 | 新增9大奴隶市场（四大区域分布），奴隶仅能在所在区域市场出售，跨区域需通过商路运输 |
| **政治制度精简** | 原文12种制度（含7种母系专属） | 精简为6普通+6母系+3全女+6经济=21种制度，普通国家覆盖城邦/贵族/部落/汗国/市政/议会六种 |
| **小国特产扩展** | 原文8种小国专属 | 扩展为10种（新增危须铁花、且弥天山玉），专属特产总计7+8+10=25种 |
| **典型国家扩展** | 原文6个典型国家 | 扩展为10个，覆盖全部6种政治制度和4大地理区域 |
| **部落规则不统一** | 小国2-3个部落，全女性国家特殊规则 | 统一为大国5/中等国4/小国3，共191部落（12×5+17×4+21×3）；部落制度与所属城邦一致 |
| **附庸军事调遣矛盾** | 01称"不服从军事调遣"，04称"服从军事调遣" | 统一为：附庸国**缴纳30%年产出**但**不服从军事调遣**，与同盟（无收益无调遣）和占领（全部控制）形成清晰梯度 |
| **胜利条件不一致** | 01与04对B/C/E线胜利条件描述不同 | 统一为：B线"占领≥30+控制80%贸易额"、C线"占领≥16+≥36国文化认同"、E线"占领≥30+五方友好+≥36国联合体系" |
| **斯基泰国家类型** | 05称"全女大国"，01/04称"全女小国" | 统一为"全女小国"（人口4,000，与01-worldview.md一致） |
| **04§3.4重复内容** | 全女性国家剧情线在§3.1-3.3已完整列出，§3.4再次重复 | 删除§3.4重复段落，保留§3.1-3.3中的完整版本 |
