### 1.1 世界背景

**西域**，位于汉朝与匈奴之间的丝绸之路咽喉地带，是一片由绿洲、沙漠、草原和雪⼭构成的⼴袤⼟地。在这片⼟地上，五十余个城邦与部落星罗棋布，它们既相互依存又彼此竞争，共同谱写着丝路上的传奇。

西域的独特之处于——在这片⼟地上并存着**三种社会形态**：
- **普通国家**（男权主导，但未极端化）：38个
- **母系社会国家**（女性主导，包容男性）：8个
- **全女性国家**（完全排斥男性）：6个

这种多元并存的社会结构，构成了西域最复杂、最引人入胜的戏剧冲突。

### 1.2 核心冲突矩阵

西域诸国的关系网络之所以复杂，根源在于三组核心冲突的交互作用：

| 冲突轴 | 冲突内容 | 典型对立 |
|--------|----------|----------|
| **性别政治** | 男权 vs 女权 vs 平权 | 普通国 vs 全女国 |
| **文明维度** | 农耕 vs 游牧 vs 商业 | 汉化城邦 vs 匈奴附庸 |
| **地缘利益** | 丝路控制权、资源争夺、领土扩张 | 大国 vs 小国 |

## 二、主线剧情发展路径

### 2.1 游戏开端（第1-2年）

<details>
<summary>展开查看</summary>

#### 汉使遇袭

> 西域烽烟再起——汉朝使者车队在丝路遭遇匈奴劫掠，大汉使者被杀，贡品被劫。这起事件引发西域诸国的连锁反应，成为游戏的开端引爆点。

公元**96年**，一队由汉朝使节**张肃**率领的使团从长安出发，沿河西走廊西行，前往西域各国传达汉朝皇帝的善意。使团携带着丝绸、瓷器、铁器等贵重礼物，意图与西域各国建立友好关系。

然而，当使团行至**楼兰**与**匈奴**交界处时，遭遇一队匈奴骑兵的伏击。张肃及其随从奋力抵抗，终因寡不敌众而全军覆没。匈奴人掠走了所有贡品，扬长而去。

消息传到长安，汉和帝震怒。朝廷中主战派和主和派激烈争论。最终，皇帝决定派遣新的使者前往西域，同时加强在西域的军事存在。

与此同时，匈奴人也在为这次劫掠感到不安——他们本意是威慑西域各国不要倒向汉朝，却不料这起事件反而加速了汉朝在西域的介入。

西域诸国面临艰难选择：是倒向日益强大的汉朝，还是屈服于匈奴的铁蹄？抑或保持中立，在两大势力之间寻求生存空间？

</details>

### 2.2 发展期（第3-5年）

<details>
<summary>展开查看</summary>

#### 丝路风云

> 西域诸国围绕丝路控制权展开竞争与博弈，各国纷纷站队、结盟或独立发展。

随着汉朝影响力扩大，丝路贸易进入繁荣期。商队络绎不绝地从长安出发，经过敦煌、楼兰、龟兹、疏勒，最终抵达大夏、安息等国。

然而，繁荣的表面下暗藏危机：

- **匈奴**不甘心失去对西域的控制，开始策反一些摇摆不定的城邦
- **汉朝**在西域设立西域都护府，加强对各国的影响
- **大国**之间为争夺丝路贸易份额展开明争暗斗
- **小国**在大国夹缝中艰难求生，不断调整外交策略

这一时期，玩家的外交策略和经济发展将决定国家的走向。

</details>

### 2.3 冲突期（第6-8年）

<details>
<summary>展开查看</summary>

#### 西域烽烟

> 全面战争爆发！汉匈两大阵营的对峙升级为军事冲突，西域各国被迫选边站队。

- 第6年初：匈奴大军南下，攻打亲近汉朝的城邦
- 第6年中：汉朝派兵增援，双方在西域展开拉锯战
- 第7年：战争蔓延至全境，丝路贸易中断
- 第8年：关键战役决定战局走向

在此期间，玩家需决定是否参战、何时参战、支持哪一方，或利用机会扩张自己的势力。

</details>

<details>

### 2.5 终局（第10年+）

<details>
<summary>展开查看</summary>

#### 新秩序

> 最终结局——收复西域、重建丝路、称霸西域或被征服。根据玩家在游戏中的选择，走向不同的结局。

**结局一：丝路守护者**
- 统一西域主要城邦，建立稳定的区域秩序
- 丝路贸易达到历史顶峰
- 各国和平共处，文化交流繁荣

**结局二：汉朝附庸**
- 成为汉朝的附属国
- 获得汉朝保护但失去独立外交权
- 保持内部自治

**结局三：匈奴霸主**
- 倒向匈奴，成为草原帝国的一部分
- 获得匈奴军事支持
- 与汉朝为敌

**结局四：独立之王**
- 既不依附汉朝也不屈服匈奴
- 依靠自身实力在西域立足
- 走钢丝般的平衡外交

**结局五：女王国度**
- 建立以母系/全女国家为核心的联盟体系
- 女性主导的政治经济文化复兴
- 性别平等的全新社会模式

</details>

## 四、特殊事件详解

> 本章对游戏中的特殊事件进行详细说明。特殊事件是在主线剧情之外触发的独立事件，涵盖政治、经济、军事、文化等多个维度。它们可能改变国家的制度、影响国家的命运，为玩家提供超出常规游戏的策略选择。

---

### 4.1 政治制度转型事件

> 政治制度转型是游戏中最具影响力的特殊事件类型。它们改变国家的根本制度——从普通到母系，从母系到全女，或反向转型。每个转型事件都有明确的触发原因、条件、机制和后果。

#### 4.1.1 🌸 凤鸣觉醒 — 女性解放运动（普通国 → 母系国）

|----------|------|:----------:|

```
月触发概率计算公式：
  基础概率 = 5%（固定值）
  加权概率 = 基础概率 × (1 + 所有满足条件概率系数之和)
  示例：满足女性官员条件(+0.20)+母系建交(+0.20)+女性教育(+0.15) → 月触发概率 = 5% × 1.55 = 7.75%
  上限：月触发概率不超过15%/月
  （注：当某加权概率系数对应条件不满足时，该系数为0）
触发冷却期：从游戏开局起36回合内不可触发（历史惯性期）
```

**剧情概要：**

> 疏勒国的纺织女工**古兰**在酒馆中第一次听说了瑶池国——那里女人可以做官、可以当兵、可以拥有自己的财产。她不敢相信，却又无比向往。
>
> 古兰开始秘密召集姐妹们，在纺织工坊的阁楼里交换信息。起初只有五个人，然后是十个、二十个——她们传抄着来自母系国的书籍，学习着那些陌生的词汇：平等、自主、自由。
>
> 在一个月圆之夜，古兰带领三百名女性走上疏勒街头，举着横幅："我们要读书"——她们没有要求参政，没有要求当兵，只要求最基本的权利：接受教育。但这个看似温和的请求，依然引发了保守派的强烈反对。

##### 一、触发原因详解

**A模式·人口危机：**

> 全女性国家完全依赖圣水受孕、外交通婚、男奴繁殖等特殊生育方式维持人口。当这些方式的边际效益递减，或圣水资源枯竭时，人口必然萎缩。这是全女性制度的先天缺陷——一个没有男性的社会，在人口学意义上是一个封闭系统，无法长期维持。

**B模式·经济困境：**

> 全女国内重体力劳动者（矿工、伐木工、建筑工等）必须由女公民或女奴担任。女性在重体力产出方面效率低于男性（约75-80%），需要更多人力资源才能达到同等产出。当人口整体萎缩时，重体力产出缺口将急剧扩大，形成恶性循环：人口↓→重体力产出↓→经济恶化→无力购买外援→人口进一步↓。

**C模式·特殊契机：**

> 西梁女国爆发天花，人口死亡15%。邻国男性医者**李时**不顾自身安危，携带草药入城救治。一个月后，疫情得到控制。西梁女王面对救命恩人，陷入了深深的矛盾——她从小被教育"男人是祸水"，但这个祸水刚刚救了她全城子民的性命。

##### 二、触发条件详解

**A模式·人口危机：**
- 人口连续下降≥12个月，且下降幅度≥15%——说明人口替代机制（圣水/通婚/繁殖）已无法弥补基础死亡率
- 粮食自给率<50%——劳动力不足导致农业产出严重下降
- 月增长率<0.3%——远低于维持人口平衡所需的0.8-1%增长率

**B模式·经济困境：**
- 重体力产出缺口>30%——矿业/伐木/建筑/运输等需要体力的行业严重产能不足
- 连续≥6个月财政赤字——进口依赖导致国库持续出血
- 女奴供不应求——试图通过购买女奴弥补劳力，但市场供应不足（特别是重体力女奴稀缺）

**C模式·特殊契机：**
- 触发灵活性最强，可由特定剧情事件引发
- 不要求国家陷入危机——只是提供一个"思想转变"的契机
- 但即使触发，玩家仍可选择拒绝开放

##### 三、触发机制详解

```
【触发判定流程】

每月判定：
  检查是否满足任一触发模式：
  
  模式A判定：
    人口下降≥12月且≥15%？
    粮食自给率<50%？
    月增长率<0.3%？
    全部满足 → 模式A激活
    
  模式B判定：
    重体力缺口>30%？
    连续财政赤字≥6月？
    女奴供不应求？
    全部满足 → 模式B激活
    
  模式C判定：
    特殊救助剧情已完成？
    已完成 → 模式C激活
  
  任一模 激活？
  ├── 否 → 不触发（下月再次判定）
  └── 是 → 计算月触发概率
              ↓
      判定是否进入事件：
      弹出事件触发提示（非弹窗）
              ↓
      经济/人口煎熬回合数累计至临界值：
      累计触发判定成功3次（即3个月均通过判定）→事件正式弹出
```

**阶段化推进：**

| 阶段 | 时间 | 描述 |
|------|------|------|
| **潜伏期**（隐藏计数） | 累计月数（触发条件满足时开始累积；条件不满足时暂停但不清零） | 国家面临持续的困难，但尚未到触发事件的程度 |
| **触发期**（事件弹出） | 潜伏期≥12个月+触发判定成功 | 事件第一次弹出——向玩家预警："您的国家正面临严峻挑战……" |
| **抉择期**（玩家选择） | 事件弹出后立即 | 弹出两个阶段：第一阶段预告（预警），第二阶段正式抉择（3个月后） |

**临界值系统：**

> 为避免玩家在警戒期忽视预警，系统设置"临界值"：
> - 触发期弹出后，进入3个月的"警戒期"（也称"抉择缓冲期"）
> - 警戒期内不强制玩家决策，但每月弹出一次提示
> - 警戒期结束后，若玩家仍未做出选择，自动弹出正式抉择窗口

**警戒期事件：**
- 第1个月：文官报告"陛下，情况不容乐观，我们需要讨论对策"
- 第2个月：民间出现讨论"是否应该引入男性"的声音
- 第3个月：外国使节私下表示"如果贵国改变政策，我们愿意提供援助"

>

>
> 张弼冷笑："将军，乌孙人来的时候，你的方阵守住什么了？"
>
> 疏勒王**安国**陷入了艰难的选择——他知道张弼说得对，但全面转向骑兵意味着：裁减半数步兵、建设马场、购买战马、训练骑手——这些都需要金子和时间。而北方的匈奴不会等他准备好了再来。

### 4.3 经济制度转型事件

> 西域经济制度的变革往往由资源危机、商业机遇或外部压力触发。这些事件可能迫使玩家改变既有经济路线，也可能带来意想不到的收益。

#### 4.3.1 🌾 绿洲危机 — 农业崩溃与改革

| 维度 | 说明 |
|------|------|
| **触发对象** | 任何国家类型（依赖农业的国家高概率触发） |
| **触发原因** | 连续干旱、蝗灾、盐碱化等生态灾难 |
| **触发条件** | ①连续≥6个月粮食赤字（产出<消耗）或②农业人口占比>50%且遭遇天灾（干旱/蝗灾/洪水） |
| **剧情概要** | 绿洲农业体系遭遇严重危机，传统耕作方式无法维持，必须进行农业改革或转行。可能的方向：引入坎儿井技术（从汉朝）、改种抗旱作物、发展畜牧业替代农业、或干脆放弃农业转向商业进口 |

| 后果 | 选项（三选一） |
|------|------|
| **选项一：水利革命** | 引入坎儿井灌溉系统，投入2,000金+12个月；完成后农业产出+30%，抵抗干旱能力+50% |
| **选项二：畜牧转型** | 放弃50%农田转为牧场，投入1,000金+6个月；完成后畜牧业产出+40%，但粮食自给率降至30-50% |
| **选项三：进口依赖** | 扩大商路贸易，以商业收入换取粮食进口；投入1,500金+建立≥2条粮食进口商路；完成后商业收入+15%，但粮食安全依赖外交关系 |


**详细剧情：**

> 于阗国连遭三年大旱，绿洲萎缩，粮食产量下降60%。老农**库尔班**跪在干涸的河床边哭泣——他的先祖在这片土地上耕种了三百年。于阗王召集群臣讨论对策：议会的年轻派主张发展贸易换粮，保守派坚持打井抗旱。最终，于阗王采纳了折中方案——引入坎儿井技术（从大夏请来水利工程师），同时与疏勒签订粮食进口协议。三年后，于阗不但恢复了粮食生产，还因灌溉系统升级成为西域农业技术最先进的国家。
>
> 然而，水利革命并非万能。坎儿井的维护需要大量人力——每个月都需要下井清理淤泥和加固井壁。五年后，第一批坎儿井开始老化塌陷。于阗人发现，水利革命只是一个开始，持续的维护才是真正的挑战。有的城邦学于阗修了坎儿井，却因维护不善而废弛；有的则因此背上了长期的财政负担——这就是绿洲农业的困境：水是生命线，但水也是无底洞。

#### 4.3.2 🛍️ 丝路新章 — 商路革命

| 维度 | 说明 |
|------|------|
| **触发对象** | 任何国家类型（商路节点国家高概率触发） |
| **触发原因** | 丝路贸易路线发生重大变化（如新商路开辟/旧商路中断） |
| **触发条件** | ①本国位于丝路主干道且②任一商路收入连续下降≥20%持续6个月或③邻国发现新商路导致绕行 |
| **剧情概要** | 丝路格局变化，传统商路可能废弃，新商路崛起。玩家需决定如何应对——是固守旧路还是开拓新线 |

| 后果 | 选项（二选一） |
|------|------|
| **选项一：开拓新路** | 投入3,000金建立新商路（需外交协调+军事保护）；完成后商路收入+25%；但旧商路收入归零 |
| **选项二：强化旧路** | 投入2,000金升级旧商路（修建驿站+增加护卫）；完成后旧商路收入+15%；但若新路持续分流，长期仍面临衰退风险 |

**详细剧情：**

#### 4.3.3 🏰 庄园起义 — 奴隶制危机

>
> 更致命的是，其他国家的庄园主们听说了姑墨的奴隶起义，开始更加残酷地镇压自己的奴隶——这导致更多的怨恨在黑暗中酝酿。西域诸国就像坐在一堆干柴上，只等下一个火星。

#### 4.3.4 💎 专营权争 — 特产垄断战争

**详细剧情：**

>
> 尉迟信冷笑："不买？那你们就别想要和田玉了。于阗的玉石全西域独此一家——你们有本事自己去挖。"
>
> 谈判破裂。疏勒、龟兹、莎车三国宣布联合抵制于阗玉石——同时开始从葱岭以西大量进口青金石和玛瑙。于阗的玉石出口量在半年内下降了四成。
>
> 尉迟信这才慌了。他召集大臣商议对策——是降价竞争（打破专营定价体系），还是武力威慑（用军力逼三国就范），抑或寻找新的买家？
>
> 最终，于阗王选择了第三条路——派出使者远赴汉朝长安，向大汉朝廷推销于阗玉石。汉朝皇帝被精致的和田玉所吸引，签订了大额采购合同。于阗成功地将贸易重心从西域本地转向了汉朝市场，弥补了疏勒三国的缺口。但这也意味着于阗越来越依赖汉朝——当汉朝的需求波动时，于阗的经济也随之起伏。

| 规则 | 说明 |

| **事件触发优先级** | ① 叛乱/战争类（即时威胁）> ② 经济/人口危机（生存压力）> ③ 政治转型（战略选择）> ④ 文化/社会事件（长期影响） |
| **冲突事件解决** | 若同时满足多个事件的触发条件，优先触发高优先级事件。低优先级事件在冷却期后重新判定 |
| **事件冷却期** | 同一事件触发后，进入12回合冷却期（不可再次触发同一事件）。不同事件之间不共享冷却期 |
| **连锁事件** | 某些事件的完成可能触发后续事件链（如：父权复辟→男性压迫→女性不满→凤鸣觉醒→女儿当国）。连锁事件不受冷却期限制 |
| **事件存档** | 每个触发过的事件及其结果将被记录在"国家大事记"中，供玩家查阅历史决策 |
| **多线并发** | 玩家可同时面对多个事件的抉择——此时事件将以"队列"形式依次呈现，玩家需按顺序决策 |

### 4.9 🌍 自然环境与灾害事件

> 西域的自然环境严酷多变。干旱、洪水、地震、蝗灾等自然灾害可能摧毁一个国家的经济基础，也可能成为改革的契机。

#### 4.9.1 🌊 洪水之灾 — 绿洲水患

| 维度 | 说明 |
|------|------|
| **触发对象** | 位于河流沿岸的绿洲城邦 |
| **触发原因** | 春季雪山融水过多，河道决堤淹没农田和城镇 |
| **触发条件** | ①城市位于河流沿岸（塔里木河/叶尔羌河/伊犁河等）且②该年冬季降雪量>平均值的150%（游戏内通过"天象事件"判定） |

| 后果 | 效果 |
|------|------|
| **洪水爆发** | 农田被毁50%；基础设施受损（道路/建筑耐久-30%）；人口损失5-10%；稳定度-15 |
| **赈灾选项** | ①拨款赈灾（2,000金）→灾后6个月恢复，稳定度+10；②开仓放粮（消耗5,000粮）→人口损失减少50%，稳定度+5；③强制劳役重建（免费但稳定度-5）→恢复时间延长至12个月 |
| **防灾选项（洪水后长期）** | 修建堤坝（3,000金+18个月）；完成后洪水灾害概率-70%；农业产出+10%（灌溉改善） |

**详细剧情：**

> 这一年春天，天山雪水异常充沛。龟兹城外的塔里木河水位暴涨，老渔民**阿洪**看着河水中的漩涡，心中不安——他活了六十年，从未见过这么大的水。
>
> 三天后，洪水冲垮了河堤。龟兹城西部的农田变成了一片汪洋，数百座房屋被冲毁，一百多人丧生。龟兹王**白霸**在城头望着被淹没的家园，咬紧牙关："重建——我们不仅要重建，还要建得比原来更好。"
>
> 但国库里的钱只够赈灾或修堤——不能同时做两件事。他必须选择……

#### 4.11.2 ⚡ 连锁事件触发规则

| 规则 | 说明 |
|------|------|
| **连锁方向** | 事件链中的事件按照预设方向线性推进，但每个环节玩家都有选择权 |
| **跳级限制** | 不能跳级触发（如未经凤鸣觉醒直接触发女儿当国）；必须完成前置事件 |
| **分支可能** | 事件链并非唯一路径——玩家可在任一环节选择"偏离"（如凤鸣觉醒后选择温和改革而非激进转型，事件链中止） |
| **反向触发** | 某些事件链可反向进行（如父权复辟→凤鸣觉醒→母系国恢复）；但部分事件不可逆（女儿当国→全女国不可逆） |
| **并发冲突** | 当两条事件链冲突时（如同时满足凤鸣觉醒和父权复辟的条件→按优先级：父权复辟优先触发，凤鸣觉醒进入冷却期） |
