### 1.3 月度结算顺序

```
每月结算顺序（含消耗类型标注）：

1.  【月度产出】农业/工业/纺织/商业产出 → 资源入库
2.  【月度消耗】男公民粮食(1粮/人) + 女公民粮食(0.8粮/人) + 奴隶粮食(0.6粮/人) + 士兵粮食(1.5~1.8粮/人)
3.  【月度消耗】马匹马料(2粮/匹)
4.  【月度消耗】盐消耗(人口÷100)
5.  【月度消耗】铁匠铺燃料(5木/月)
6.  【月度消耗】兵器日常损耗(0.5%)
7.  【月度消耗】军费维护(兵力÷100金)
8.  【月度消耗】武将/文官俸禄(10~50金/人)
9.  【月度消耗】建筑维护费
10. 【月度收入】税收(人口×0.5金)
11. 【月度收入】部落贡赋(各种资源)
12. 【月度收入】商路收入(金币)
13. 【月度收入】酒吧收入(+5金/酒吧)
14. 【月度消耗】冬季额外消耗(皮毛+兵器损耗)
15. 【一次性】检查是否有建设完成/训练完成/培养完成
16. 【一次性】随机事件触发（可能产生一次性消耗或收入）
```

### 1.4 关键经济指标

```
月度净收入 = 月度总收入 - 月度总消耗
  月度总收入 = 税收 + 商路收入 + 部落贡赋 + 酒吧收入 + 舞女收入
  月度总消耗 = 粮食消耗 + 盐消耗 + 军费 + 俸禄 + 建筑维护 + 燃料

一次性储备金 = 当前金库 - 预计一次性支出
  预计一次性支出 = 建设预算 + 征兵预算 + 外交预算 + 购买预算

健康经济标准：
  月度净收入 > 0（每月盈余）
  一次性储备金 > 月度总消耗×3（至少3个月缓冲）
  粮食储备 > 月度粮食消耗×6（至少6个月口粮）
```

#### 贸易市场规则

| 规则 | 说明 |
|------|------|
| **市场准入** | 需与市场所在城邦建立外交关系（中立以上）才能交易 |
| **距离浮动** | 买入价 = 基础价 × (1 + 距离浮动率0%~30%)，越远越贵 |
| **供应上限** | 每个市场每月各类商品有收购/出售上限，防止批量倾销 |
| **区域特产限制** | 区域独特特产和专属特产**仅在城邦所在区域的贸易市场出售**（见03-资源.md 三、节） |
| **季节波动** | 秋季粮食价格-10%（丰收），冬季皮毛价格+20%（需求增加） |
| **商路加成** | 建立商路连接两个市场后，两市场间贸易价格-10% |

#### 奴隶市场规则

| 规则 | 说明 |
|------|------|
| **市场准入** | 需与市场所在城邦建立外交关系（中立以上）才能购买奴隶 |

| **月供应上限** | 每个市场每月各类奴隶有供应上限（大国市场30~50人/类，中等国市场15~25人/类） |

| **全女国男奴繁殖** | 全女国无男性人口，购买男奴后**强制立即消耗**：消耗1男劳奴/男匠奴，6个月后产出10个女公民；消耗1男战奴，9个月后产出20个女公民。男奴不可用于其他用途 |
| **阿玛宗奴市特权** | 全女/母系国在阿玛宗女武奴市购买**任何女奴**价格额外-10%（与全国优惠叠加） |

| **疏勒特权** | 疏勒为西域最大奴隶市场，全品类供应且价格-5%，每月供应量+30% |
| **区域差异** | 天山北市场男奴多女奴少；南道市场女奴比例较高；北道市场最均衡 |

### 2.3 跨区域贸易与商路系统

> **核心机制**：玩家通过建立**商路**连接不同区域的市场，实现跨区域资源流通和特产运输。

| 商路类型 | 连接方式 | 建立成本 | 效果 |
|----------|----------|----------|------|
| **区域内商路** | 同区域两个市场之间 | 200金+2马匹 | 两市场间贸易价格-10%，每月自动运输标准资源 |
| **跨区域商路** | 不同区域两个市场之间 | 500金+5马匹+3骆驼 | 两市场间贸易价格-10%，可运输区域特产（运输成本+20%） |
| **丝路主干商路** | 连接≥3个区域的超级商路 | 1000金+10马匹+5骆驼 | 全部连接市场贸易价格-15%，特产运输成本降至+10% |

**商路收入公式**：
```
每月商路收入 = 基础收入(50金) × 商路等级 × 市场等级系数 × 季节修正
  · 商路等级：区域内×1.0，跨区域×1.5，丝路主干×2.0
  · 市场等级系数：★★★★×1.2，★★★×1.0
  · 季节修正：夏季+10%，冬季-20%
```

### 3.2 制度1：绿洲农耕制 🌾

#### 核心机制

| 项目 | 内容 |

| **核心加成** | 农业产出+15%（含农民、农场奴隶）、粮仓容量+25%、食物损耗-10% |
| **核心代价** | 商业产出-10%、商路收入-15%、市场购买力-5% |
| **最佳路径** | D-种田发展 → 后期可转 A/B/C |

#### 3.2.1 普通国家应用

> **代表城邦**：焉耆⚒️、姑墨🍇、婼羌🏹、扜弥🍷

|------|----------|
| **人口分配** | 农业55%（男农民为主）+ 矿工8% + 伐木8% + 手工业12% + 商业7% + 士兵10%（男兵） |
| **核心循环** | 男农民(3粮/月)→粮食盈余→养更多男兵→军事扩张或人口增长→更多农民→正循环 |
| **性别优势利用** | 男公民占55%，农业效率最大化（男农民+3 vs 女农民+2.5）；重体力职业（矿工/伐木）全由男性承担 |
| **奴隶策略** | 购买男劳奴(50金)为主分配农场，快速拉升粮食产出；少量女匠奴用于纺织辅助 |
| **出口策略** | 粮食盈余可用于酿造美酒(2粮→1美酒)出口，或直接贸易换取金币和铁矿 |
| **转型时机** | 人口>5000且粮食盈余>1000/月时，可转商路贸易制或军事征伐路径 |

**典型发展曲线**（焉耆为例）：
```
回合1-20：农业55% → 粮食盈余800/月 → 人口4000+

```

### 3.5 制度4：庄园贡赋制 🏰

#### 核心机制

| 项目 | 内容 |
|------|------|
| **适用场景** | 拥有大量奴隶、附属国/部落多、需要快速集中资源 |
| **核心加成** | 奴隶产出+15%、贡赋收入+20%（含部落/附庸国）、奴隶转化公民速度+15% |
| **核心代价** | 公民满意度-10%（奴隶主压迫感）、叛乱风险+5%、自由民迁移率+5% |
| **最佳路径** | D-种田发展（奴隶密集型）或 B-军事征伐（掠夺奴隶续航） |

#### 3.5.1 普通国家应用

> **代表城邦**：疏勒🕌、焉耆⚒️、大宛🏇

| 维度 | 策略详解 |
|------|----------|
| **人口分配** | 公民农业35%+奴隶劳动35%（男劳奴农场/矿场/伐木）+手工业15%+商业10%+士兵5%（贡赋养兵，压缩军费） |
| **奴隶占比目标** | 奴隶占总人口30%-40%为最佳比例（产出最大化同时不满情绪可控），超过40%需增加治安投入 |
| **贡赋网络** | 通过武力威慑或外交手段使周边部落/小国进贡，贡赋收入占月收入20-30% |
| **转型方向** | 庄园贡赋制积累资源速度快，适合中期转型军事征伐（用积累资源暴兵） |

#### 3.5.2 母系国家应用

> **代表城邦**：瑶池国🌸、青鸾国🦚、赤谷国🔥

| 维度 | 策略详解 |
|------|----------|
| **人口分配** | 公民农业30%（女农）+奴隶劳动40%（女劳奴/女匠奴为主）+手工业15%（纺织+采药）+商业8%+士兵7%（女兵为主） |
| **特色优势** | 母系国的庄园贡赋制偏"柔性"——女性奴隶为主(70%)，理念相近，满意度惩罚减半(-5%→-2.5%)；舞女/侍妾在庄园制中价值倍增（舞女+10金/月更适配，侍妾奴隶增长加成+人口基数） |
| **侍妾策略** | 侍妾占比提至15-20%（比普通国10%高），利用侍妾+庄园加成叠加：奴隶月增长率×1.5(侍妾占比加成)×1.15(庄园加成) |
| **贡赋特色** | 母系国贡赋可不限于资源：通过外交通婚约束附属国（联姻纽带替代武力镇压），贡赋稳定性+15% |

#### 3.5.3 全女性国家应用

> **代表城邦**：东女国👸、苏毗女国👸

| 维度 | 策略详解 |
|------|----------|
| **人口分配** | 公民农业25%（女农最低）+奴隶劳动**50%**（全游戏最高奴隶比例）+手工业20%（最大化女性优势）+商业0%+士兵5%（女兵保底）——减去女兵后剩余公民全部分配手工业 |
| **全女国特色** | 全女国的庄园制是所有国家中**奴隶比例最高、依赖最深**的。原因：①无男性从事重体力，必须依赖女劳奴弥补效率差 ②女奴价格-10%，购买成本低 ③全女国对女奴的统治更稳固（理念认同+满意度惩罚减半） |

### 3.6 制度5：国家专营制 🏛️

#### 核心机制

| 项目 | 内容 |
|------|------|
| **适用场景** | 拥有稀缺资源/加工技术、需要集中力量办大事、计划经济思维 |
| **核心加成** | 资源加工效率+20%（如2铁矿→1兵器效率提升至1.67铁矿→1兵器）、资源售价+15%（国家定价权） |
| **核心代价** | 自由商业-15%（私营商业受压制）、外交好感-5（贸易伙伴不满垄断）、科技创新速度-10% |
| **最佳路径** | E-特色精专（专营特色产业）、D-种田（控制粮食流通） |

#### 3.6.1 普通国家应用

> **代表城邦**：于阗💎（玉石专营）、温宿🏔️（石材专营）、赤谷国🔥（精铁专营）

| 维度 | 策略详解 |
|------|----------|
| **核心策略** | 国家垄断1-2种关键资源的加工和销售，控制价格和出口配额 |
| **加工链专营** | 以于阗为例：玉石(矿场采集)→国家控制→玉器加工(5玉→1玉器，售价×3)→专营出口，国家赚取3倍附加值 |
| **人口分配** | 资源开采30%（国家矿场）+加工25%（国家工坊）+农业25%+商业10%+士兵8% |
| **价格控制** | 国家可定时调整售价（+0%~+30%），价格越高销量越低，需平衡短期利润与长期市场占有率 |

#### 3.6.2 母系国家应用

> **代表城邦**：赤谷国🔥（赤谷精铁专营）、玄女国👸（兵书专营）、青鸾国🦚（玉雕专营）

| 维度 | 策略详解 |
|------|----------|
| **核心策略** | 母系国国家专营制聚焦女工匠优势+专属特产：赤谷精铁(120金/单位)武器锻造加成+30%；玄女兵书(200金/卷)武将统率+10；青鸾玉雕(250金/件)武将智力+8——这些高附加值专属产品天然适合国家专营 |
| **人口分配** | 女工匠35%（国家工坊集中生产）+特色采集20%（赤谷铁矿/青鸾玉石）+农业25%+商业8%+士兵10% |
| **女性工匠红利** | 国家工坊中女匠奴(月+3丝绸/+5建设值)、女纺织工(月+2丝绸)效率远超男性，专营制下集中管理进一步放大优势（20%加成叠加） |
| **外交工具化** | 母系国可将专营品作为外交武器：限定出口配额、对敌对国禁运、对友好国优惠价——专营权本身就是谈判筹码 |

#### 3.6.3 全女性国家应用

> **代表城邦**：西梁女国🌸（子母河水/天衣丝绸专营）、弱水国🌸（弱水泉水专营）、素女国🌸（素女琴专营）

### 4.1 路线A：君临天下 ⚔️

> **胜利条件**：①铁血统一(50国全占领) + ②六关尽破(8关口全攻占) + ③霸王之证(军事传奇) + ④无人敢犯(全势力恐惧)

> **推荐国家**：乌孙🐎、车师🏹、危须🐟、阿玛宗⚔️、斯基泰🏹

#### 经济制度配置

| 阶段 | 推荐经济制度 | 配置理由 |
|------|-------------|----------|
| **①军备期**（回合1-20） | **③游牧畜牧制** 🐏 | 马匹+25%+骑兵维护-20%，快速组建骑兵军团，为后续征服积累兵力 |
| **②扩张期**（回合21-50） | **③游牧畜牧制** 🐏 持续 | 保持骑兵优势，掠夺邻国获取资源和奴隶，以战养战扩大规模 |
| **③霸权期**（回合51+） | **④庄园贡赋制** 🏰 | 征服大量城邦后，附庸贡赋+奴隶劳动维持战争机器运转，贡赋+20%+奴隶+15% |

**关键指标**：兵器月产>30，马匹储备>200匹，粮食自给率>80%，每征服一国后需3个月"战后恢复期"（产出减半）

#### 军事制度配置

| 维度 | 配置策略 |
|------|----------|

| **特殊规则** | 10倍兵力可让目标国自动投降；同化成本需大量金币支撑 |

#### 政治制度配置

| 维度 | 配置策略 |

### 4.2 路线B：富甲丝路 💰

| **经济制裁能力** | B路线核心能力——封锁敌国商路使对方收入-60%，迫使其屈服为附庸。军事不是为了打仗，而是制造经济不平衡 |

#### 政治制度配置

| 维度 | 配置策略 |
|------|----------|
| **自身制度** | **普通(父权)制**🏛️ 首选 | **母系制**💎 次选 |
| **同化策略** | **保留制度为主**——附庸国保留原制度，不花冤枉钱同化。"经济控制大于政治控制" |

| **外交管理** | 西域议会（≥10国外交+≥5帝国友好→召开）；联合体系（50国联合→集体安全）。军事行动以自卫为主，不主动宣战 |

### 4.6 路线F：天朝藩屏 🐉

> **核心机制**：臣服一方帝国，以藩属身份借宗主之力统一西域。选择一方帝国（汉/匈/大宛/康居/大夏/羌人）称臣，获得全面军事/经济援助，以宗主名义代主征伐，统一西域后面临终极抉择。

| 维度 | 内容 |

| **可走国家** | **全部50国**——普通36国+母系8国+全女6国均可。全女国可臣服任一帝国弥补军事短板；母系国可利用宗主军事援助快速扩张 |

| **设计意图** | 为偏好"依附强权+借势扩张"的玩家设计——以小博大，在帝国夹缝中求生存，最终面临忠诚与独立的终极抉择 |
