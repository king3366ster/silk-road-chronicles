## 一、武将系统

### 1.1 武将属性

每位武将/文官拥有五项核心属性：

| 属性 | 范围 | 影响 |
|------|------|------|
| **统率(leadership)** | 30-100 | 带兵数量、军队士气 |
| **武力(force)** | 20-100 | 单挑胜负、近战伤害 |
| **智力(strategy)** | 20-100 | 计策成功率、防御策略 |
| **政治(politics)** | 10-100 | 内政效率、外交效果 |
| **魅力(charisma)** | 10-100 | 招募成功率、部下忠诚 |

### 1.2 武将等级与军衔

| 军衔 | 所需等级 | 最大带兵 | 加成 |
|------|----------|----------|------|
| 伍长 | Lv1 | 500 | +0% |
| 什长 | Lv3 | 1,000 | +5% |
| 百夫长 | Lv5 | 2,000 | +10% |
| 千夫长 | Lv8 | 5,000 | +15% |
| 万夫长 | Lv12 | 10,000 | +20% |
| 将军 | Lv15 | 20,000 | +30% |
| 大将军 | Lv20 | 50,000 | +40% |

### 1.3 武将/文官获取途径（四种）

> **核心规则**：武将和文官的来源有四种途径——奴隶培养、任务奖励、攻掠国家招降、随机招聘。不存在流浪投奔等其他来源。**仅舞女和侍妾可直接培养为武将/文官，其他奴隶必须先转化为公民或士兵，再通过军功或科举事件晋升。**

#### 途径一：奴隶培养（仅限舞女/侍妾直接晋升）

| 奴隶类型 | 培养目标 | 消耗 | 培养期 | 产出属性侧重 |
|----------|----------|------|--------|-------------|
| **侍妾** ♀ | 女文官 | 400金+80粮 | 10回合 | 政治+魅力高，统率+武力低 |
| **侍妾** ♀ | 女武将 | 500金+100粮 | 15回合 | 统率+魅力高，政治+智力中 |
| **舞女** ♀ | 女文官(外交型) | 350金+60粮 | 10回合 | 魅力+政治高，武力+统率低 |

> **其他奴隶（劳奴/匠奴/战奴）** 不可直接培养为武将/文官，必须先转化为公民或士兵，再通过**军功提拔**或**科举选拔**事件晋升。

#### 途径二：任务奖励（主要来源）

**城邦任务奖励武将**：
- 完成城邦故事线后，奖励1-2名武将/文官
- **男女比例**：普通国家5:5，母系社会国家3:7，全女性国家100%女性
- 城邦任务武将/文官属性较高（平均70+），有独特技能

**部落任务奖励武将**：
- 完成部落故事线后，奖励1名武将/文官（随机决定类型）
- **男女比例**：普通国家5:5，母系社会国家3:7，全女性国家周边部落100%女性
- 部落武将/文官属性中等（平均50-70），有部落特色能力

#### 途径三：攻掠国家招降（批量来源）

- **征服一国后**：该国武将/文官有30%概率投降归顺
- **招降条件**：赠礼（100-500金）+ 说服（外交值判定）
- **忠诚度**：初始40，需提升至60+才能完全信任
- **属性保留**：招降武将保留原属性的80%，可通过后续培养恢复

#### 途径四：随机招聘（补充来源）

> **核心机制**：随机招聘不是主动操作，而是通过**公民养成事件触发**。

- **触发方式**：
  - **军功提拔**：士兵参与战斗≥5场后，有概率触发「民间英才」事件
  - **科举选拔**：城市等级≥Lv3后，每年春季可举行「科举」
  - **民间推荐**：每回合有基础概率，由武将/文官推荐民间人才
- **触发频率**：大国每年约3人，中等国约2人，小国约2人
- **招聘条件**：城市等级≥Lv2，金库≥200金
- **招聘消耗**：每人100-300金
- **男女比例**：普通国家6:4，母系社会国家3:7，全女性国家100%女性
- **属性范围**：平均45-65（低于任务奖励，高于奴隶培养初期）
- **忠诚度**：初始50（低于其他来源）

### 1.4 城邦初始武将/文官

| 国家类型 | 初始武将/文官数 | 武将:文官比例 | 说明 |
|----------|----------------|-------------|------|
| **大国**（12个） | **9人** | 根据国家属性调整 | 好战国武将多，文教国文官多 |
| **中等国**（18个） | **7人** | 根据国家属性调整 | 好战国武将多，文教国文官多 |

> **初始武将/文官属性**：平均属性50-70，忠诚度初始80（高于其他来源）。

### 1.5 武将与文官分工系统

> **核心规则**：武将和文官有明确的职责分工。武将专注于军事领域（3种任务），文官专注于内政领域（5种任务）。**武将不可执行文官任务，文官不可执行武将任务。**

#### 武将可执行任务（3种）

| 任务 | 关键属性 | 效果 | 属性要求 |
|------|----------|------|----------|
| **训练士兵** | 统率+智力 | 士兵经验+10%~30%/月，新兵→精兵速度×2 | 统率≥40 |
| **巡逻护卫** | 武力+智力 | 商路安全+20%~40%，城邦治安+15 | 武力≥40 |
| **侦察谍报** | 智力+魅力 | 获取敌方情报，提前2回合预警入侵 | 智力≥50 |

#### 文官可执行任务（5种）

| 任务 | 关键属性 | 效果 | 属性要求 |
|------|----------|------|----------|
| **内政管理** | 政治+智力 | 粮食产出+10%~30%，金币收入+10%~20%，建设速度+10%~20% | 政治≥40 |
| **外交谈判** | 魅力+政治 | 外交好感+15~30/次，贸易价格优惠5%~15%，联盟成功率+20% | 魅力≥50 |
| **贸易管理** | 魅力+智力 | 商路收入+15%~30%，市场购买价格-5%~15%，出售价格+5%~15% | 魅力≥40 |
| **科技研发** | 智力+政治 | 科技点+5~15/月，解锁新建筑/新兵种/新能力 | 智力≥50 |
| **人口管理** | 政治+魅力 | 人口增长+5%~15%，公民满意度+10%，奴隶转化速度+20% | 政治≥40 |

#### 武将/文官属性侧重分类

| 类型 | 属性侧重 | 适合任务 | 来源 |
|------|----------|----------|------|
| **猛将型武将** | 武力>统率>智力 | 巡逻护卫 | 战奴培养、军事任务 |
| **统帅型武将** | 统率>智力>武力 | 训练士兵 | 战奴培养、城邦任务 |
| **智将型武将** | 智力>统率>武力 | 侦察谍报 | 匠奴培养、探索任务 |
| **内政型文官** | 政治>智力>魅力 | 内政管理、人口管理 | 劳奴/侍妾培养 |
| **外交型文官** | 魅力>政治>智力 | 外交谈判、贸易管理 | 舞女/侍妾培养 |
| **技术型文官** | 智力>政治>魅力 | 科技研发、内政管理 | 匠奴培养 |

#### 武将文官搭配策略

```
早期（人口<500）：1文官（内政管理）+ 1武将（巡逻护卫）
  → 文官提升粮食/金币产出，武将保护商路安全

中期（人口500-2000）：2文官（内政+贸易）+ 2武将（训练+巡逻）
  → 内政文官管粮食，贸易文官管金币
  → 训练武将提升军队质量，巡逻武将保护扩张中的商路

后期（人口2000+）：3~5文官（内政+外交+贸易+科技+人口）+ 2~3武将（训练+巡逻+侦察）
  → 全方位管理，每个文官专注一个领域
  → 武将负责军事后勤保障（训练/护卫/侦察）
```

### 1.7 奴隶-公民-武将/文官完整转化机制

```
【完整转化路径图】

奴隶 ──→ 公民 ──→ 士兵(军功) → 武将
  │         │         │
  │         │         └→ 文官(科举) 
  │         └→ 分配职业（农民/矿工/商人/士兵等）
  ├──→ 战奴整编 → 普通兵(Lv1) → 精兵/老兵 → 军功 → 武将

            └─→ 文官(直接培养)

【转化路径一：奴隶 → 公民】
条件：奴隶服役≥10回合 + 城市等级≥Lv2

效果：奴隶变为公民，可分配所有职业

【转化路径二：战奴 → 士兵（整编路线）】
条件：男战奴/女战奴 + 城市等级≥Lv1 + 兵器库有兵器
消耗：1兵器 + 10金/人
效果：战奴直接编入军队成为普通兵(Lv1)
产出：战斗力×0.7，忠诚度初始40

【转化路径三：舞女/侍妾 → 武将（直接培养）】
条件：舞女或侍妾 + 城市等级≥Lv3
消耗：450-500金 + 90-100粮 + 25-30回合
效果：变为见习武将(Lv1)

【转化路径四：舞女/侍妾 → 文官（直接培养）】
条件：侍妾/舞女 + 城市等级≥Lv2
```

## 二、兵种系统

### 2.1 兵种完整培养链路

```
【步兵培养链路】
公民 → 征召 → 新兵(Lv0) → 普通步兵(Lv1) → 精锐步兵(Lv2) → 老兵步兵(Lv3)
战奴 → 整编 ────────↗               │
                                    ├→ 长矛兵专精 → 反骑大师（克制骑兵+15%）
                                    ├→ 盾兵专精 → 铁壁守卫（守城防御+25%）
                                    └→ 重步兵专精 → 破阵先锋（攻城+20%）

【骑兵培养链路】
公民 → 征召 → 新兵(Lv0) → 普通骑兵(Lv1) → 精锐骑兵(Lv2) → 老兵骑兵(Lv3)
战奴 → 整编 ────────↗               │
                                    ├→ 轻骑兵专精 → 游击大师（速度+30%，骚扰+20%）
                                    ├→ 重骑兵专精 → 突击铁骑（冲锋伤害+25%）
                                    └→ 弓骑兵专精 → 射骑高手（远程+20%，克制+15%）

【士兵升级条件】
新兵→普通兵：武将训练3回合 + 10金/人
普通兵→精兵：武将训练6回合 + 20金/人 + 战斗经验≥2场
精兵→老兵：武将训练10回合 + 30金/人 + 战斗经验≥5场 + 武将统率≥60

【战奴整编规则】
战奴 → 编入军队 → 普通兵(Lv1)，战斗力与普通新兵相同
  消耗：1战奴 + 1兵器 + 10金 + **3回合训练期**
  初始属性：战斗力与普通新兵相同，忠诚度初始40
  与普通征兵对比：普通公民征召→普通新兵（**无需回合数和金币**，即时完成）；战奴整编需额外3回合+10金
  与雇佣兵对比：雇佣兵更贵（40~150金/人）但战斗力更强（×0.9~×1.2）且即时可用；战奴更便宜但需训练等待
  忠诚提升：每参与1场胜仗，忠诚+2；每参与1场败仗，忠诚-5；忠诚<30时可能叛逃

【专精解锁条件】
  长矛兵：民兵营Lv2 + 武将统率≥50
  盾兵：城墙+民兵营Lv2 + 武将武力≥50
  重步兵：兵器坊 + 武将统率≥60
  轻骑兵：马场Lv2 + 武将统率≥50
  重骑兵：马场Lv3 + 兵器坊 + 武将统率≥65
  弓骑兵：马场Lv2 + 武将智力≥50
```

### 2.2 征兵系统

```
征兵流程：
  1. 确认征兵容量（受民兵营/精锐兵营限制）
  2. 从公民中征召（减少可用劳动力）→ **即时完成，无需回合数和金币**
  3. 消耗兵器（步兵×2兵器/骑兵×1兵器/弓骑兵×1兵器）
  4. 普通公民征召后直接成为**普通新兵(Lv0)**

> **三种兵源对比**：
> | 兵源 | 成本 | 等待时间 | 初始等级 | 战斗力 | 忠诚度 |
> |------|------|----------|----------|--------|--------|
> | **普通公民征召** | 仅消耗兵器 | **即时**（0回合） | 新兵(Lv0) | 基础值 | 初始60 |
> | **战奴整编** | 兵器+10金/人 | **3回合**训练期 | 普通兵(Lv1) | 与新兵相同 | 初始40 |
> | **雇佣兵** | 40~150金/人 | **即时** | 即战力 | ×0.9~×1.2 | 固定50 |

征兵规则（按国家类型区分）：
  普通国家：只征召男兵
  母系社会国家（8个）：70%男兵+30%女兵（需城市Lv2）
  全女性国家：只征召女兵

女兵战力加成：
  同兵种女兵战斗力比男兵高5%
  女兵纪律性更强、阵亡率略低（-3%）、兵器损耗略低（-0.5%/战）
```

### 2.3 雇佣兵系统

> **核心机制**：疏勒是西域雇佣兵中心，玩家可通过支付金币临时雇佣兵力。雇佣兵不消耗公民人口，但维护费用较高。

| 雇佣兵类型 | 雇佣费用 | 月维护费 | 战斗力 | 特殊说明 |
|-----------|----------|----------|--------|----------|
| **疏勒护卫兵** | 50金/人 | 2金/人/月 | ×0.9 | 防御型，适合守城 |
| **草原战奴兵** | 40金/人 | 1.5金/人/月 | ×0.7 | 成本低，可快速暴兵 |
| **大宛天马骑** | 150金/人 | 5金/人/月 | ×1.2 | 精锐骑兵，需天马资源 |
| **草原弓骑手** | 80金/人 | 3金/人/月 | ×1.0 | 远程机动，适合骚扰 |

> **雇佣兵限制**：
> - 雇佣兵总数不超过正规军的30%
> - 雇佣兵忠诚度固定50，不可提升
> - 雇佣兵不可升级/转专精
> - 解约时雇佣兵直接消失（不返还费用）
